Commentary

Isolated Loop Pro Series 24-track 24-channel multi-recorder.

It uses an isolated loop system that pursues the ideal tape travel.

The capstan uses a quartz-lock direct drive system.

Pitch control of up to ± 9.9% is possible in 0.1 step by quartz synthesizer system.

The reel unit uses a direct drive system.

Digital tension control by microprocessor is adopted.
In addition, a microprocessor based driving control system enables direct change.

The head block uses an integrated HPF head with good in-line performance.

1-24 channels can be switched independently (Rec, Safe, In, Outsync)

Equipped with auto-locator function of time display.

Equipped with a tape timer for real-time display.

Microprocessors are used for tape transport control, tape tension, tape speed control during FF/REW, tape timer control, auto-locator control, and channel select control.
