Commentary

This is a cassette deck equipped with a 5-point multipeak indicator and enhanced basic performance.

It is equipped with a 5-point multi-peak indicator using LED like the high-grade model.

It is designed with an emphasis on the linearity of all circuits by using a 2-stage direct-coupled microphone amplifier with a low-noise transistor and a line amplifier using the same IC as the KD-31/KD-11.
In addition, an emitter-follower circuit is also used for the headphone circuit so that high-impedance headphones up to 10k Ω can be used.

Uses a rotary wide cassette door that allows the entire cassette half to be seen.
When the cassette door is opened, the entire cassette is stored in the upper part of the panel. Moreover, when the cassette door is closed, it has a special shape with less protrusion from the panel.

The cassette holder part of the tape transport mechanism adopts a new structure that firmly holds the cassette half with steel balls from the left and right sides, making it possible to run the tape stably.
In addition, the driving system reduces wow flutter by holding the capstan and flywheel shaft with a spring from the rear.

Equipped with a timer standby mechanism that combines an electromagnetic solenoid for auto-stop and a pause mechanism. By connecting an optional timer, it is possible to record absence and play alarm clock.

Uses full auto-stop mechanism by reed switch and electromagnetic solenoid.

Running operation which can be switched from FF and REW to direct play is possible.
