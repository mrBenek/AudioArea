Commentary

This is a 3U combination model with a CD-RW recorder and a cassette deck.

It is easy to copy between a CD and a cassette, and it is also possible to copy from a record to a cassette or CD-R/RW by supporting phono input.
It also supports digital input / output for Coaxial and Optical devices.

The CD-RW recorder is equipped with a sampling converter and supports 32/44.1/48kHz.
It is also equipped with a fade-in / out recording function (1 to 24 seconds, 1 second unit, recording time and load when Auto Track is on), an auto track function that automatically increments the track when a CD track or sound starts, nine auto track levels (-24dB to -72dB, 6 dB unit), manual track increment, Rec Mute function and synchronous recording function.
In addition, it is equipped with a program playback function of one track, all tracks, random tracks and up to 25 tracks, a function of one track, all tracks and A-B repeat playback function, an intro check function that plays the beginning of a track every 10 seconds, and a function of timer playback function.

For CD-RW only, you can use the Erase function to erase after the specified track and the Unfinalize function to make the finalized disc recordable.

The cassette recorder is equipped with Oatley Verse mechanism, Dolby B noise reduction, pitch control of approximately ± 12%, RTZ function, Rec Mute function which can create 4-second unrecorded portion between songs, and CPS function which can search for songs by detecting 4-second unrecorded portion.

Wireless remote control is included.
