Commentary

A belt drive player system developed as a successor to the PS 2400.

A low-speed DC servo motor is used as the power source to reduce vibration and noise due to the low-speed rotation of this motor. In addition, uneven rotation is also reduced because the rotation is controlled by servo.

Since it uses a servo motor, speed switching and speed adjustment are electronic and do not damage the belt. In addition, a dedicated IC is used for the servo circuit to improve reliability.

A 31 cm aluminum die-cast turntable is used for the turntable.
Oilless metal is used for the bearing of this turntable and lubrication is not required.

Strobe with large neon tubes is adopted.

The tone arm adopts a universal type static balance type arm, and is equipped with functions such as lateral balancer, inside force canceler, arm lifter, 0 balance fine adjustment, and arm height adjustment device.
In addition, a gimbal support system using a high-precision pivot bearing is adopted, and vibration from the boat is cut by using a rubber damper for the support part.

An auxiliary weight is mounted for heavy cartridges.

The cartridge uses a 0.5 mil diamond needle IM type with tapered cantilever for improved sound quality. In addition, the shell can be replaced due to international standards.

Head shell angle can be adjusted.

The output code is a low-capacity code with less high-frequency loss.

The cabinet is thick and uses a newly developed special insulator to absorb external vibration.
In addition, a replacement shell stand is mounted.

Uses a removable acrylic dust cover. This dust cover is a fleece top design.
