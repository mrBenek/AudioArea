  • Pioneer PD-F51

  • ¥ 60,000 (released in 1994)

Commentary

50 + 1 file-type CD player.

You can set 50 CDs in 2 Lorin Dreyfuss G.R.C. Mio (25 sheets storage), and you can reset the CDs in the other Lorin Dreyfuss G.R.C. Mio while playing.
In addition, it uses a plus 1 slot in mechanism, so you can add one sheet quickly.

You can choose from three performance variations : a custom mode in which 50 CDs can be registered and played in three groups, a single mode in which one CD is picked up and played, and an all mode in which 50 CDs are played continuously.
When the play is restarted after stopping, the play starts from the last disc played using the last disc memory function.

The CD title (up to 8 characters) can be registered in the remote control in advance. The CD title button allows you to call up the CD you want to listen to and play it from confirmation.
