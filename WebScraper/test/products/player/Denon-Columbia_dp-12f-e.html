  • DENON DP-12F

  • ¥ 49,800 (around 1981)

Commentary

This is a full auto player with a free-setting design that can be set to fit perfectly on walls.

By devising the shape and hinge of the dust cover, it can adhere to the wall with a gap of approximately 5 mm.

The rotary system adopts a direct drive system using a linear drive motor and DENON quartz.

The tone arm uses an easily adjustable dynamic balance type light weight straight arm.
It is electronic needle pressure application.

Fully automatic mechanism controlled by microprocessor is adopted.

Quick repeat mechanism is adopted.

Equipped with an automatic record size detection mechanism.

The MC type cartridge DL-32 is installed in the cartridge.

Self-illuminated soft touch push button is adopted.

Equipped with an interlocking function with the cassette deck DR-11.
