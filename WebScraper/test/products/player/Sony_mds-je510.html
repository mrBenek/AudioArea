  • SONY MDS-JE510

  • ¥ 55,000 (released in 1997)

Commentary

MD deck with wide bit stream technology.

In order to take advantage of the wide dynamic range of the ATRAC format, we have developed a 20-bit input / output format and expanded the operational vocabulary, enabling recording and playback with 20-bit equivalent accuracy exceeding 16 bits, which is the MD format.

It is equipped with an 18-bit high quality processing sampling rate converter that automatically converts sampling frequencies such as 48 kHz and 32 kHz to MD 44.1 kHz.

By using an eccentric insulator with an offset mounting hole, vibrations transmitted from the outside cancel each other, making it difficult for the vibrations to be transmitted to the chassis. This is effective in attenuating the vibration components in the middle and high range, which affect the audibility in particular.

It is equipped with a time machine recording function. By recording a sound stored in the internal memory to a disk, recording can be started retroactively to the previous sound. With the MDS-JE510, recording can be started from a sound up to 6 seconds before.

Equipped with a music sync function.
Simply press the Music Sync button on the remote control and the connected device starts playing. It detects signal input and starts recording automatically.

It is equipped with a monaural recording and playback function that can be used to double the disc length.

The TOC EDIT function allows you to join, split, and move songs. The UNDO function allows you to undo the most recent editing operation. In addition, the A-B ERASE function allows you to easily erase between two specified points in a single song.
It is also equipped with a rehearsal function that lets you check edit points in COMBINE and DIVIDE. You can move edit points finely in up to -128 steps to + 127 steps (one step is approximately 0.06 seconds).

It is equipped with a title function that allows you to enter up to approximately 1,700 characters for each disc.
In addition to alphabets and symbols, Katakana is also available. You can input Katakana characters with the FOMA terminal or remote controller. You can also input Katakana characters using the Remote Commander RM-D10P (sold separately).

The SCMS (Serial Copy Management System) allows digital recording only for one generation and does not allow digital recording for two or more generations.

Wireless remote control is included.
