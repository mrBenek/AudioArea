  • TEAC f-500

  • ¥ 69,800 (around 1978)

Commentary

Stereo cassette deck with micro switch of logic control.

Improved operability is achieved by adopting logic control and micro switch.
The logic control allows direct switching from one mode to another.

The input volume is an L/R-independent type that makes it easy to set recording levels on the left and right sides. In addition, the input volume is a differential type that moves one side to move the other side.

A FG servo DC motor is used in the mechanism section. This motor has a built-in frequency generator (FG) to eliminate changes in motor rotation caused by voltage and frequency fluctuations of the power supply.
In addition, high-precision capstan shafts, large flywheels, and precision polishing flat belt are used to improve the transmission system.

HD ferrite head is used for the head.

It is equipped with a line / microphone input change-over switch, and by sending the line input directly to the recording amplifier, distortion and noise amplifier.

In order to take advantage of the performance of the tape, it has a 3-stage switching mechanism with independent bias and equalizer.

Equipped with a Dolby system.

Equipped with a Rec Mute mechanism.

It is equipped with a memory counter that can start a song.

Equipped with a timer mechanism.

Equipped with a peak level meter.

There was a rack handle sold separately.

By connecting a separately sold remote control unit, it can be operated from a remote location.
