Commentary

Century Series CD/CD recorder which integrates CD player and CD recorder

By integrating a CD player with a CD-R/RW recorder, it is possible to make digital copies of CDs from one unit.
The CD recorder is compatible with CD-R and CD-RW music recording discs.

For the mechanism of the recorder part, a vibration control structure mechanism using a die cast is adopted as a countermeasure against vibration which affects recording and reproduction.

Equipped with the MAKE CD function, which is convenient for copying a whole CD, it can perform auto and double speed processing from the start of dubbing to the end and finalization.
In addition to CD dubbing, Auto Track Increment also works with analog sources, detecting 2.7 seconds of silence during analog source recording.

Double speed recording from a CD player to CD-R/CD-RW is realized.

When you copy from an internal CD to a CD-R/CD-RW, you can copy text information along with music data if the CD contains text. You can also enter and edit album titles and artist names.

If the source is prohibited from digital copying, it is automatically switched to analog copying.

The CD player also has a dedicated output terminal so that it can be used as two independent CD players.

Using the Easy JOG dial, you can easily set menus, select tracks, enter text, and adjust recording levels.

SCMS is supported.

Wireless remote control is included.
