Commentary

Liberty Series cassette deck with Dolby NR / C type and linear electronic counter.

By using the TSS (Tuner Signal Sync) terminal of the TA-YX7F or TA-YX5F pre-main amplifier of the Liberty Series, you can air-check the target broadcast program at an appropriate level.

In addition to Dolby NR / B type, Dolby NR / C type is newly installed.

Equipped with a linear electronic counter that displays the amount of tape travel in minutes and seconds.

Equipped with an auto tape selector, the type of tape is automatically determined by the detection hole of the cassette half and the tape position is automatically switched.

Uses a 2-motor mechanism that uses dedicated motors for the capstan drive and reel drive.

Equipped with LED peak program meter.

Equipped with a pitch control that can change the tape speed by ± 20% during playback.

Equipped with a timer standby switch.

Remote control via wired / wireless / foot remote control (sold separately).
In addition, synchronized play is possible with the synchronized remote control.
