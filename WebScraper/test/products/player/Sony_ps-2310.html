Commentary

This belt and drive player system emphasizes basic performance.

Uses 4-pole hysteresis synchronous motor with less vibration developed exclusively for the turntable.
In addition, it uses a double-side polished urethane belt.

A 31 cm aluminum die-cast turntable is used for the turntable.
Oilless metal is used for the bearing of this turntable and lubrication is not required.

It employs a newly developed universal type static balanced tone arm and is equipped with an inside force canceler, height adjustment mechanism and head shell adjustment mechanism.
In addition, arm lifter is mounted.

The cartridge is equipped with an IM type cartridge.
Since the shell is a standard type, the cartridge can be replaced.

The speed switching uses a lever system with a protective mechanism in the drive part to reduce vibration.

A newly developed special insulator is used to prevent howling and needle flying.

The output code uses a low-capacity type with less high-frequency loss.

The acrylic dust cover is removable and has a fleece top design.
