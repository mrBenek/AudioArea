  • ¥ 69,800 (around 1998)

Commentary

This 3-head stereo cassette deck focuses on basic performance and functions.

It uses a 3-head system that uses special heads for erasing, recording, and playback to achieve high sound quality and simultaneous monitoring during recording.

Two pairs of pinch rollers and capstan are installed on both sides of the head, and a closed dual capstan mechanism is used to block vibration of the tape.

It uses an auto-accu-bias that automatically adjusts the optimum recording bias current according to the tape to be used.

It uses an auto monitor that automatically switches to tape when playback starts, and to source when recording pause state is set.

It includes a real-time counter that displays the actual tape running time, allowing you to view elapsed time and remaining tape capacity.
