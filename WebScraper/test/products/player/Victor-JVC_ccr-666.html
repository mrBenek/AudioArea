Commentary

The basic structure of this stereo cassette deck uses the same mechanism as the high-end model CCR 667.

In order to reduce hissing, which is a problem inherent to tapes, a noise suppressor switch is installed to attenuate only the hissing band.

In order to show the characteristics of normal tape and chrome tape, it is equipped with a tape selector. For chrome tape, bias current and equalizer circuit are switched simultaneously.

The read / write head uses a Cronios head with excellent abrasion resistance and magnetic efficiency.
This head uses a special permalloy for the core and has a structure of 6 layers laminated.

Features are improved by applying precision processing to each part of the mechanism such as adoption of capstan made by Schmerg Germany for the drive part and improvement of roundness.

Equipped with photoelectric auto-eject mechanism by Cbs.
It also features a manual eject that works immediately from any mode.

In consideration of maintenance such as head cleaning, a removable Head cover and holder are adopted.

Not only the input level but also the output level can be changed.

It features a large level cover and a manual eject function that can be used from any mode.
