  • Technics RS-640U

  • ¥ 69,800 (around 1976)

Commentary

Stereo cassette deck equipped with HPF head and Dolby NR circuit.

An HPF head has been adopted for the head.

Equipped with an end eye that lights up approximately 3 minutes before the end of the tape (when using C-60 tape).

By combining with a timer (sold separately), it is possible to record absence.

Equipped with a meter capable of peak checking by switching.

Equipped with Dolby NR into which MPX filter can be inserted.

Microphone input and line input can be mixed.

Equipped with a memory rewind function that automatically stops at the position of counter 999.

Equipped with a tape selector.

It has an output volume.

There was a dust cover sold separately.
