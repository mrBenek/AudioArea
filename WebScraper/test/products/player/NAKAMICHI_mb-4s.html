  • Nakamichi MB-4s

  • ¥ 39,800 (around 1993)

Commentary

A CD changer equipped with MusicBank System in pursuit of cost performance.

While adopting the same mechanism as MB-3s, cost performance is improved by adjusting D/A conversion part and function.

The mechanism part uses a music bank mechanism, and 7 disc multi-play is possible with the same feeling as a single disc type CD player.
The MB-4s employs a second generation mechanism, which greatly reduces the number of mechanical parts to improve reliability and reduce costs. In addition, the second generation mechanism achieves a high speed of 2.1 seconds while the former mechanism takes as short as 3.7 seconds to change the disc. In addition, the MB-4s achieves a compact size of approximately 1/2 in volume.
The user interface has also been greatly improved, with seven direct disk access keys and a disk calendar, making it easy to call up, store, and check the CDs you want to play.

50-program disk / track memory, random play and repeat play are available.

Only disk number 1 supports 8 cm CD.

It is equipped with the Nakamichi system remote function, and the remote control unit attached to the Nakamichi stereo receiver or integrated amplifier enables system remote control of the connected MB-4s.

Equipped with a time display function capable of displaying track time and fixed total time.

The D/A converter is a 16-bit dual D/A converter and the digital filter is equipped with a 4 x oversampling digital filter.

Wireless remote control is included.
