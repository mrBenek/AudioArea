Audio Database

  • YAMAHA P-100X

  • ¥ 49,800 (released in 1985)

Commentary

Full-auto DD player with linear tracking system.

A linear tracking arm that does not cause any tracing distortion is used for both the inner and outer periphery of the phonograph record.

Uses a front operation method that allows operation with the dust cover closed.

T4P plug-in P plug-in type MC cartridge.

It uses a DC4-phase 8-pole coreless FG servo motor.
