  • SONY PS-5170

  • ¥ 33,800 (around 1974)

Commentary

Belt drive player system equipped with semi-automatic mechanism.

Uses a 4-pole hysteresis synchronous motor developed exclusively for the turntable.
In addition, it uses a double-side polished urethane belt.

The turntable uses an aluminum die cast 30 cm type.

The tone arm employs a universal tone arm, equipped with an arm lifter and an inside force canceler.

Equipped with semi-auto mechanism capable of auto return and auto cut.

An IM type cartridge is used for the cartridge. Since the shell is a standard type, the cartridge can be replaced.

The output code uses a low-capacity type with less high-frequency loss.

It uses a removable acrylic dust cover.
