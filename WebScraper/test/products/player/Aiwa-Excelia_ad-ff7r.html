Commentary

Quick Reverse Deck with Dolby C type.

Aiwa's proprietary infrared tape sensor quickly detects the last part of the magnetic material on the tape, enabling a 0.4 second quick reverse.

The head section employs a system in which the read / write head rotates by 180 °. This eliminates the azimuth deviation that causes the degradation of the high-frequency characteristics and ensures uniform read / write characteristics in both directions.
In addition, we have achieved excellent wow flutter by using capstan with micrograin processing.

Equipped with a Dolby C type noise reduction system.

The Tuner Rec Preset Level (TRPL) allows you to preset the recording level of the tuner independent of the main recording volume.

Equipped with LH tape bias fine adjustment mechanism in order to draw out the ability of various LH tapes.

Uses IC logic control.

Equipped with 3-color 12-point peak program meter.

Can be operated from a remote location by combining with separately sold remote control unit.
