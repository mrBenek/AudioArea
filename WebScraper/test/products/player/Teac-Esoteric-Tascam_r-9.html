Commentary

This digital audio tape deck is equipped with shuttle dial and inherits the technology of TASCAM for professionals.

Equipped with shuttle dial that enables quick and accurate search.
The shuttle dial allows you to search while listening to sounds at any speed from 1 x to 16 x.
In addition to capturing the current position of the tape, you can also use the DATA dial to enter numerical values.
Direct search is also available by specifying the program number.

A 1-bit AD/DA converter is mounted independently on the L/R to achieve high linearity.

It supports sampling frequencies of 32 kHz, 44.1 kHz and 48 kHz, and is equipped with a long play mode of 32 kHz, so it can record and play continuously for up to 4 hours using 120 minute tape.

The automatic recording detection level of the start ID can be selected from five levels (-48dB/-54dB/-60dB/-66dB/-72dB) and can be set according to the state of the source.
It is also equipped with a position function that can finely adjust the position of the start ID and skip ID in 5-frame units.

Equipped with L/R-independent analog input level control for precise input level adjustment.
In addition, it is equipped with a manual / reference switching switch. In the reference position, the input is at the specified level regardless of the position of the level control pot on the front panel. It is very useful for use in fixed systems such as studios.

Equipped with a large 17-segment peak meter with a 1-second peak hold function for enhanced visibility.

Three counter displays are available : ABS time, PGM time and REMAIN time.

Equipped with a Rec Mute button that automatically records 4 seconds of silence with one touch.

Equipped with a standby function that waits for pause after channel selection.

Equipped with headphone port with level control.

It features a wireless remote control with random programming of up to 50 songs.
