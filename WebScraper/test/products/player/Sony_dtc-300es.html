  • SONY DTC-300ES

  • ¥ 110,000 (released in 1989)

Commentary

As a new standard model, this DAT deck was developed with particular attention to sound quality.

The A/D section combines a 2 fs A/D converter and a 2 fs digital filter to reduce the load on the low-pass filter by doubling the sampling frequency. The converter itself uses a newly developed chip with improved acoustic distortion. In addition, the phase is corrected by time axis control.
The D/A converter block uses a 16-bit D/A converter equipped with a 4 fs digital filter for independent L/R operation.
In addition, the A/D and D/A sections are completely separated on a dedicated circuit board to reduce mutual interference.

2D. D. + 1 + 1. D. + 1 BSL motor mechanism.
DTC-300ES has a mechanism with high reliability and excellent operation feeling by installing 2 motors exclusively for tape loading.

It is equipped with a subcode function that records absolute time from the tape top, time search using absolute time, recording start ID and end ID in addition to skip ID and start ID, rehearsal function that allows fine adjustment when writing ID, program number function, renumber number from the head of the tape, erase function that individually erases start ID, skip ID and program number, and shift number function that resets all start IDs by 0.3 seconds in advance.

Two types of input / output terminals, one for each of optical and coaxial digital, are mounted.
The optical input uses a servo-type digital signal normalizer circuit that absorbs jitter caused by cables.

Equipped with a linear counter that can be switched to 4 modes : counter display indicating running time, reproducing display indicating remaining time of the tape, program time indicating elapsed time of the music being played, and absolute time indicating absolute time from the tape top.

When recording with digital input from a BS tuner or other DAT deck, the FOMA terminal waits for up to 10 minutes in the recording pause even if the signal is interrupted. This allows you to set the recording start time with time, such as when recording with a timer.

Mechanism, digital, analog and display sections are clearly separated, and internal construction reinforced by sub-chassis is adopted.

Equipped with wide range digital peak meter of -60dB to 0 dB with each ch24 segment.
The data on which the digital peak meter display is based refers to the data of the digital processing circuit.

It is equipped with a display off function in consideration of minute noise generated by the FL tube.
In addition, it is equipped with a dimmer function that can change the display brightness by 6 levels depending on the brightness of the room or preference.

It is equipped with a cue & review function at two speeds, approximately 2 x speed and approximately 16 x speed.

Equipped with a blank search function that automatically searches for unrecorded parts that have never been recorded.

By combining the timer sold separately, it is possible to record your absence and play the alarm clock.

In addition to most of the operation of the main body, it comes with a wireless remote control which can be operated such as selection of music and pose of Sony's remote control compatible CD player.
In addition, it is possible to perform wireless CD-sync recording in which the recording pause of the main unit is released and the CD performance is started by operating both the main unit and the CD player.
