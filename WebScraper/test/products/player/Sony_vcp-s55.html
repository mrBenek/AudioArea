  • SONY VCP-S55

  • ¥ 43,000 (released in 1995)

Commentary

A video CD player developed for personal use.

Equipped with a playback control function, you can enjoy simple interactive software using the menu screen, the player itself or the number key of the remote control.
* This function is only available for software with playback control function.

It supports high-precision still images and can play high-definition still images with four times the number of pixels of moving images.

It has an on-screen display function that allows you to check the operation status on the TV screen.
In addition, it is equipped with a help guide function that guides the operation procedure in Japanese.

It is equipped with functions such as bookmark function, continuation function which can see continuation with one touch, scene playback and slow playback.

Equipped with S-video output.

Equipped with an optical digital output.
The optical digital output can only be used when playing music CDs.

Wireless remote control is included.
