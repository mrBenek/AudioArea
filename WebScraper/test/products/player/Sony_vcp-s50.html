  • SONY VCP-S50

  • ¥ 33,000 (released in 1996)

Commentary

Video CD player with compact design.

Equipped with slow playback function, speed can be adjusted in 3 stages.

It is equipped with a continuous function so that you can watch from the scene you stopped.
* If you open the tray, the continuous function will be canceled.

Equipped with two types of digest function, you can check the disk contents.
It is equipped with a disk digest function that displays the image at the beginning of each track on a 9-division screen and a track digest function that displays the development of one track on a 9-division screen.

Equipped with a scene playback function, you can directly jump to the scene you want to watch simply by specifying the scene number recorded on the video CD software with playback control function with the remote control.

Equipped with a help guide function, the operation procedure and operation error are displayed in Japanese on the TV screen.

Both the main body and remote controller are displayed in Japanese to improve operability.

Up to 24 songs can be programmed and shuffled, and one song / all songs / A-B (partial) repeat function is installed.

Equipped with optical digital output. Digital output is only available when playing music CDs.

Wireless remote control is included.
