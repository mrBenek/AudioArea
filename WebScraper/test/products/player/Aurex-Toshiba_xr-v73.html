Audio Database

  • Aurex XR-V73

  • ¥ 55,000 (released in 1986)

Commentary

Compact CD player of mini component size.

It is equipped with 20-track random memory function and direct selection function.

It is compatible with ACCR/DDRS, and you can enjoy system play by combining it with Aurex component AV700CD.

Repeat playback of all songs / memory songs is possible.

A centralized display with FL tubes is used for the display section, which can display information such as disk operation status, track number, time for each song, and total time.

It uses a soft touch loading mechanism and can operate all operations with a light touch.
