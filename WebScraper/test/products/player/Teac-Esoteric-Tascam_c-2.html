Commentary

C-series 2-motor cassette deck.

The dual capstan system blocks modulation noise.

Metal tape position is mounted on the tape selector.

The 僑居 head uses a newly developed high-efficiency erasing head that is compatible with metal tapes.

The oscillator is strengthened.

Plug-in BIAS/EQ cards (normal, cobalt, metal) are used.

Test input dedicated to tape adjustment is installed.

Equipped with a timer mechanism.

The Dolby System is installed as a noise reduction system.

Equipped with auto-rewind function.

Equipped with Rec Mute.

Equipped with peak level meter from -20dB to 5 dB.

A differential type left / right interlocking input volume is adopted.

Selected capacitors are used in the amplifier circuit.

There was a rack handle as an option sold separately.

The C-2 has a system concept and is designed to be upgradeable with peripheral devices like the other C-Series.

It can be used for dbx system (type II) by using optional RX-8.
In the dbx system, the dynamic range of the input signal is compressed to 1/2 in decibels for recording, and the dynamic range is expanded twice during playback, greatly improving the dynamic range.

The optional Audio Mixer MX-8 can be connected.
The MX-8 consists of four inputs and two outputs. Each input channel is equipped with a microphone (with a 20 dB attenuator) / line switching mechanism, a low-cut filter (100/200Hz), and a panpot function that determines sound image localization.

It is compatible with the CS-8 portable case with handle and the VX-4 stationary system rack. You can mount C-3, MX-8 and RX-8 simultaneously. The front and rear lids of the CS-8 can be removed and it can be used as a stationary system rack.
In addition, there was a blank panel sold separately in case the mixer or dbx is not used for CS-8 or VX-4.

Compatible with Test Tone Oscillator TO-8.
The TO-8 is a battery-operated oscillator with three selectable frequencies of 400 Hz, 6.3 kHz and 12.5 kHz, and -10dB and -30dB output.

Compatible with optional Remote Control Unit RC-90.
