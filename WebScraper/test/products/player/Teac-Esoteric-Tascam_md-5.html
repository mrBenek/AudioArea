Commentary

This MD recorder can record, play, edit, and enter text with a PC keyboard.

106/109 Japanese keyboard for DOS-V PC can be connected.
Each key on the keyboard is assigned an edit mode or navigation mode so that you can operate the keyboard like a multi-function remote control (see table below).
You can also use the keyboard to enter disc titles, song titles and artist names using katakana, alphabets, numbers and symbols.

The A/D and D/A conversion sections employ 64 x oversampling A/D and 128 x oversampling DD/A converters with 20-bit processing capability.

Ver4.5 is used for ATRAC which compresses and expands signals.

A built-in sampling rate converter automatically converts sources with different sampling frequencies, such as CS/BS and DAT, to 44.1 kHz.

It is equipped with a sound-synchronized recording function that senses input from the connected player or cassette deck and starts and stops automatically.

It can monitor the input signal from the input device even when no disc is inserted, and can also be used as a D/A converter for digital input.

Once loaded, the disc's playback mode and UTOC information will be saved even if the power is turned off, unless it is ejected by the backup function.

Editing functions include Divide, Erase, Move, Combine, and Undo.
You can also use the Title Ainc (Title All Increment) function to enter titles from the disc to all tracks without displaying the edit menu.

It has a タイトルコピーー function that copies the entered title and pastes it to the specified track.
