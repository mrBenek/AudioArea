  • SONY TC-3700

  • ¥ 69,800 (1 set, released in 1972)

Commentary

Cassette tape stereo with integrated FM tuner and small amplifier.

Amplifier section

The amplifier has a variable monitor function so that the recording level remains constant even if the playback volume is changed during recording.

A record player using MM type cartridge can be connected.

Tuner section

FET is used for the FM front end.

Equipped with AFC mechanism, it can accurately catch FM broadcasting.

Cassette deck section

It uses the ソニオマチック recording method and can automatically record at the optimum level.

Equipped with a tape selector for chrome tapes.

Equipped with a 3-digit tape counter that returns to 000 with one touch.

Equipped with a tape pilot lamp, tape running condition can be checked.

Equipped with a dynamic microphone (F-25) as standard.

Equipped with input / output terminals, it can be used as a deck unit.

Speaker part

A 16 cm cone type full-range unit is used for the unit.

Equipped with speaker ON/OFF switch.
