Commentary

This is a cassette deck with Dolby C.

It is equipped with FL (fluorescent display tube) 4-digit electronic counter by exclusive LSI.
In addition to the normal counter function, this counter is equipped with a TRT (Tape Run Time) mode to display the recording and playback time. At this time, the counter function is stored in the IC circuit, so when the mode is set to the counter, the counter display continues.
In addition, the Memory Stop is a multi-counter the TRT mode.

As a noise reduction system, Dolby C type is installed in addition to Dolby B.

It is equipped with newly developed 2-color FL bar graph meter driven by LSI.
This meter is equipped with a function as a peak meter.

A 10 cm long stroke slide-type volume is used to adjust the recording level.

The head uses a Permalloy recording / reproducing head.

Equipped with a Rec Mute mechanism.

Equipped with a timer standby mechanism.

There was a remote controller sold separately.
