Audio Database

  • VICTOR/JVC AL-S90

  • ¥ 50,000 (around 1986)

Commentary

A full-auto player system that employs a linear tracking system.

It uses a T4P plug-in type MC cartridge.

It uses a full-auto mechanism that can automatically detect the disc size and can play only by front operation.
It is also equipped with a repeat function and a manual music selection function by pressing the feed button.

Uses a direct drive system.
