  • Marantz Model5220

  • ¥ 79,800 (around 1977)

Commentary

Stereo cassette deck with front load design.

It is equipped with functions such as professional mixing function and master level control.

Equipped with a one touch tape selector to select bias / equalization.

It has a built-in memory counter that can be used for cue and repeated performance.

The design is unified with Marantz amplifiers and tuners.

There was a walnut cabinet as an option sold separately.
