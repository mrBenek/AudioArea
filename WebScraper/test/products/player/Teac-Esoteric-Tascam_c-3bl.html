Commentary

Black model sold as a limited version of the cassette deck C-3.

The drive mechanism is equipped with a tension servo mechanism so that uniform head touch can be obtained from the beginning to the end of winding. This ensures stable tape travel and minimizes pitch deviation.

A 3-head system has been adopted for the head section. The 3-head system, which has independent erasing, recording and playback functions, enables simultaneous recording and playback monitoring and further improves basic performance such as increasing the dynamic range.
We have developed and adopted an erasing head that has a large demagnetization effect in accordance with the characteristics of the metal tape. We also adopted the same recording / reproducing combination head as the C-1.

To bring out the performance of metal tape, it is equipped with a special tape select position.

It is equipped with a fourth select position. When you set the metal, normal or chrome position and press the ADJUST button, the LED lights up red. You can fine-tune this position using the optional Test Tone Oscillator (TO-8).

Since dbx system can be connected, the hissing specific to tape can be reduced by connecting RX-8 (sold separately).

The amplifier circuit uses carefully selected capacitors for coupling.

The input volume is of a differential type with left / right linkage, and the other is linked to the rotation of the other.
It is also possible to move only one side by itself.

Equipped with 3-step switching mechanism independent of bias and equalizer.

Equipped with a test input function dedicated to tape adjustment.

Equipped with a memory counter.

Equipped with a timer mechanism.

Equipped with Rec Mute function.
