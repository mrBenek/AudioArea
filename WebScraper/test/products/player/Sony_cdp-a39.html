  • SONY CDP-A39

Commentary

A CD player that supports the MD-PC link on the MD deck.

Equipped with control A1 terminal corresponding to the MD-PC link function of Sony MD deck.
The MDS-PC1 and MDS-S39 MD decks equipped with the MD-PC link function can be combined with a PC using the MD-PC link kit PCLK-MD1. The MD-PC link kit PCLK-MD1 is used to connect the MD deck and CDP-A39 via the control A1. This allows you to play CD programs and record programs from CD to MD programs by using the basic CD player on the PC screen, drag and drop operations, or CD-to-MD.

It is equipped with d ケオーダー for CD-text compatible software containing text information such as song names and artist names. When connected to an MD-PC link, it can be displayed in the software MD Editor on a PC.

The D/A converter is equipped with a hybrid pulse D/A converter.

The servo circuit uses a high-precision digital servo.

The Custom Edit function allows you to control peak search, time edit, program edit, time fade, manual fader and other functions.

Equipped with shuffle play and timer play (auto start) function.

Equipped with an optical digital output terminal.

Wireless remote control is included.
