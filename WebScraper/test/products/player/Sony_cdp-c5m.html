  • SONY CDP-C5M

  • ¥ 54,800 (released in 1987)

Commentary

A multi-CD player that can hold 5 CDs.

The disc tray is a roulette type rotating tray that allows you to play 5 CDs continuously, select music directly from 5 discs, and play up to 32 programs on 5 discs.

All discs can be played randomly.

By selecting single mode with the play mode switch, you can use it just like a normal CD player.

The digital filter uses a 2 x oversampling digital filter.

The digital circuit uses the same third generation LSI as the high-end model, and employs a unilinear converter system that enhances sound purity by synchronizing the main digital circuit with the master clock of the first child.

The power supply section is equipped with a Line 5 stabilizing power supply with independent digital, audio and other components.

High-quality parts such as non-magnetic oxygen-free copper heat sink are used.

It is equipped with a check key that can check and modify the program contents.

It is equipped with a memory backup function that enables you to wake up the music you want to listen to.

Equipped with a disk select key that allows direct disk selection.

Equipped with a repeat key that allows you to select one song / all songs in single mode, and one song / all songs on one disc / all discs in multi mode.

Equipped with auto pause function.

Equipped with a headphone port with volume.

Wireless remote control with disk select key is included.
