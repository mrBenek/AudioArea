  • National RS-805U

  • ¥ 35,600 (around 1974)

Commentary

An 8-track deck capable of recording and playing 8-track tapes.

Equipped with a microphone jack for stereo recording.

Equipped with a recording button that can be preset.

Smooth rotation is achieved by adopting a large flywheel.

The head uses a vertical system with less movement error.

It uses a full-scale mechanical pause switch.

Equipped with an FF button that can be used to start a song.

It uses a 1-meter, 1-volume system that can easily adjust the recording level.

Comes with a remote control box for program selection and ejection from a remote location.
