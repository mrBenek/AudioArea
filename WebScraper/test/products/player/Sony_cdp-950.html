Commentary

A CD player with custom file function.

A custom file function is included as a new function.
This function allows the CD player to store up to 226 titles of music information and messages input by the listener for each disc. Each time a disc is replaced, the CD player automatically identifies which disc it is, recalls the memory, displays messages for each disc, and plays the disc according to the performance information.
Custom files have three functions : disk memos, custom indexes, and program banks. You can file disk notes and either custom indexes or program banks.
Disk Memo allows you to enter your own message up to 10 characters per disk.
The custom index can be set up to a maximum of 6 points per disk. To set the index, press the FILE button during playback to specify the point.
The program bank can store up to 20 program performance functions.
By memory backup, stored contents are retained even when power is turned off.

The digital filter uses a 4 x oversampling digital filter.
In the D/A converter section, an independent dual D/A converter is used.

Sound quality is improved by the use of a digital / analog separated Route 6 independent stabilized power supply, a highly rigid and integrally molded monocoque chassis, and a large insulator.

Equipped with coaxial digital output.

With a built-in program editing function, you can edit each tape A and B side at once in your favorite order of songs while searching for songs that fit perfectly to the recording time.

Up to 20 songs can be played by program (RMS), and 6 mode repeat function of 1 song / whole song / part (remote control only) / program / shuffle / custom index is equipped.

It is equipped with a time keeper function with the next display to know the next program performance music.

Equipped with 2-speed music search. Slow mode is only from remote control.

Equipped with auto-space function.

Equipped with two line out outputs, variable and fixed.

Equipped with headphone port with lineout interlocking volume.

Wireless remote control is included.
