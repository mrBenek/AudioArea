  • Nakamichi MB-3s

  • ¥ 49,800 (around 1993)

Commentary

CD changer that uses MusicBank System.

The mechanism part uses a music bank mechanism, and 7 disc multi-play is possible with the same feeling as a single disc type CD player.
The MB-3s employs a second generation mechanism, which greatly reduces the number of mechanical parts to improve reliability and reduce costs. In addition, the second generation mechanism achieves a high speed of 2.1 seconds while the former mechanism takes 3.7 seconds to change the disc. In addition, the MB-3s achieves a compact size of approximately 1/2 in volume.
The user interface has also been greatly improved, with seven direct disk access keys and a disk calendar, making it easy to call up, store, and check the CDs you want to play.

50-program disk / track memory, random play and repeat play are available.

Only disk number 1 supports 8 cm CD.

The Nakamichi system remote function is installed, and the remote control unit attached to the stereo receiver and integrated amplifier of Nakamichi can operate the connected MB-3s system remotely.

Equipped with a time display function capable of displaying track time, リメイニングタイム and fixed total time.

The D/A converter uses an 18-bit dual D/A converter, and the digital filter uses an 8 x oversampling digital filter.

Wireless remote control is included.
