  • VICTOR/JVC KD-656

  • ¥ 43,800 (around 1974)

Commentary

Stereo cassette deck equipped with ID system mechanism and ANRS circuit.

Equipped with a noise reduction system ANRS that is compatible with Dolby B tapes.

Uses an ID mechanism to stabilize tape travel.

A new time constant equalizer has been adopted to support chrome tape.

A hard permalloy Cronios head is used for the recording / reproducing head.

Uses linear slide type volume for both input and output.
