Commentary

This is a cassette deck that is easy to use as well as polished for basic performance.

The DT 4300 has obtained a wow flutter of 0.07% by thoroughly improving the accuracy of the basic performance of the Dedge itself, especially the wow flutter.

It is equipped with a memory play mechanism, and automatic heading is possible.

You can mix Line + Line or Line + Microphone.
Two line inputs are provided, and the four volumes can be varied independently, making it possible to perform various recording and replay.

Equipped with peak level indicator.

Equipped with an end stop mechanism to prevent the tape from hurting.

Equipped with a tail run indicator, the running state of the tape can be checked at a glance by flashing red light.

The Dolby System is installed as a noise reduction system.
