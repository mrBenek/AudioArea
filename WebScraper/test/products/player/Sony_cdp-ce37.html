Commentary

Compact CD player with 3-disc CD roulette type changer.

A roulette type changer is used in the disk tray part, and 3 disks can be set.
It also has a disk exchange function so that you can pull out the tray without interrupting the performance and exchange other disks.

The D/A converter is equipped with a 1-bit D/A converter.

The servo circuit is equipped with a high-precision digital servo.
In this method, the servo amount of the optical pickup is adjusted for each disc to be played, and servocontrol reflecting the disc condition more accurately is realized.

It has a loop function that repeatedly plays a part of the music and can be operated from a remote control.
The length of the loop can be selected in five stages. In addition, you can choose between a normal rhythm in which the performance restarts from the looped part after the end of the loop, and a rhythm in which the performance starts from a position that is advanced by the looped time.

It is equipped with a multi-disc program function that allows you to select your favorite music from multiple discs and a shuffle play function that allows you to play the music in random order.

Equipped with 6-mode repeat function.

Equipped with an optical digital output terminal.

Equipped with a headphone port with volume.

Wireless remote control is included.
