  • AIWA/EXCELIA XC-005

  • ¥ 54,800 (around 1988)

Commentary

CD player with 18-bit linear L/R dual D/A converter.

It uses an eight times oversampling method that samples at 352.8 kHz, which is eight times the sampling frequency of 44.1 kHz, eliminating the influence of noise. This makes it possible to use a low-pass filter with excellent phase characteristics and improves phase characteristics in all bands.

The D/A converter is 18-bit linear with independent L/R dual configuration to improve channel separation.

There are two methods for peak value search : quick peak search and accurate peak search. These methods can be used according to usage conditions. Quick search can detect peak values in approximately 30 seconds per disc. Accurate search can detect accurately in normal disc playing time.
Both methods display peak values of -4dB and -2dB, making it easy to set the recording level on the deck side.

It is equipped with two digital output terminals which are coaxial with the optical system and two analog terminals which are fixed and variable.

Equipped with a disk file function, up to 10 characters such as alphabets, numbers and symbols can be recorded on the set disk, and original classification display is possible.
In addition, program performance and peak values can be recorded.

Equipped with digital Rec calibration.
The digital sine wave is stored in the ROM and can be used as a basic signal for adjustment during dubbing.

Top lock structure screws are used on the top and side surfaces to reduce vibration.

Equipped with auto / manual program edit function.

Equipped with 20-key direct music search.

8 cm single disk is supported.

Equipped with a large insulator equipped with special rubber.

Wireless remote control is included.
