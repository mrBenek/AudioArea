  • Marantz CDR630

  • ¥ 165,000 (around 1998)

Commentary

Commercial CD recorder compatible with CD-R/CD-RW.

CDR630 is fully compliant with Red Book, Orange Book PartII, and Orange Book PartIII. All CD media, CD (CD-DA), CD-R, and CD-RW media, can be finalized and played on ordinary CD players. CD-RW discs can be finalized and played on any CD-RW CD player.

Equipped with XLR analog input, RCA analog input / output and digital (coaxial / optical) input / output.

Equipped with a CD-Sync recording function that starts recording simultaneously with the playback of the source.

It is equipped with a sampling rate converter and supports sources with sampling frequencies of 32 kHz and 48 kHz.
When a digital signal is input at 44.1 kHz, it is possible to select full digital recording at 1 : 1, bypassing the sampling rate converter, making it possible to produce higher-quality discs. Full digital copying is only available when the digital source is 44.1 kHz ± 100 ppm.

Equipped with a track increment function that allows you to choose between two modes, auto and manual.
In auto mode, the input level is detected, so the track number can be automatically written even for analog sources.

When digital media is used as the source, it automatically recognizes the track number or start ID.

EIA 19-inch x2U rack-mount design

It is equipped with a volume dedicated to wireless remote control and headphones.
