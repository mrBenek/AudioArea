Commentary

An MD deck with an MD-PC Link Kit that can be connected to a PC for disk editing and character input.

Basic operation of MD deck and disc editing are possible using keyboard and mouse of connected PC.
Basic operations such as playback / stop of MD, start of music (AMS), fast forward / fast backward, and editing functions such as MOVE, DIVIDE, COMBINE, and ERASE can be performed on the PC screen.

By using the included operation software "MD Editor", you can enter Kanji / Hiragana characters from the PC keyboard, as well as Katakana characters, letters of the alphabet and numbers.
In addition, you can use your personal computer printer to print labels of the entered disc names and songs.

By connecting to a CD player compatible with MD Editor with Sony's Control A1 terminal, basic CD operations and CD-MD recording programs can be performed on a PC.
You can create a recording program by simply dragging the music you want to record from the CD basic screen with the mouse and dropping it on the MD basic screen. You can also use drag & drop to play the program so that you can listen to CDs in your preferred order and to operate the NAME function so that you can assign disc and song names to CDs from your PC.

By increasing the number of bits from the conventional 16 bits for arithmetic processing in the digital circuit from the input system to the output system, and greatly improving the processing system at the minute level, recording and playback with a wider dynamic range are realized.

It is equipped with a sampling rate converter that automatically converts 48 kHz or 32 kHz sampling frequency to 44.1 kHz.

With the built-in music sync function, you can detect signal input and start recording just by starting performance of the connected device.

It is equipped with a time machine recording function that can record up to 6 seconds before.

It is equipped with a TOC EDIT function that enables MD editing, an UNDO function that allows you to undo the last edit operation, and a rehearsal function that allows you to smoothly determine edit points.
In addition, it is equipped with the A-B ERASE function that allows you to easily erase between two specified points in a song.

Equipped with a timer standby function that allows recording and playback in combination with an optional timer.

Serial Copy Management System (SCMS) allows digital recording only for one generation, not for two or more generations.
