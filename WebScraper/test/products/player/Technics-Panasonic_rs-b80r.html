Commentary

The reverse deck was developed aiming at the reference deck of the digital generation.

A classAA amplifier is used as the recording equalizer amplifier for high-quality sound sources such as CDs.
The classAA method has a unique circuit configuration in which voltage control and current drive, which used to be carried out in one amplifier, are carried out in separate amplifiers. By having the voltage control amplifier receive the determination of the output waveform, the load on the output stage of the amplifier is reduced to 0, and excellent waveform response is achieved.

The Dolby HX-PRO system is installed to improve the recording quality of high-frequency sources such as CDs.
The Dolby HX-PRO detects the high-frequency component of a music source that constantly changes, and controls the bias current according to the change to set the optimum bias value for each moment. This prevents over-bias for the high-frequency range and provides a bias amount equivalent to the ideal setting value for the low-frequency range. This greatly improves the high-frequency range without deteriorating the low-frequency range. This is an excellent effect.
Unlike a noise reduction system, tapes recorded with the Dolby HX-PRO are compatible with all decks, so you can get better sound quality when playing back.

An AX amorphous head is used for the read / write head.
This head uses an amorphous alloy with high magnetic properties for the core material, and LC-OFC, which boasts an excellent SN ratio and high-frequency characteristics, for the head winding to achieve excellent characteristics.

The travel system uses a two motor full logic system with electronically controlled DC motors for the capstan and for the reel.

The 3 nr design includes dbx NR, Dolby B and C NR, and can be used according to the application.
In addition, the combination of dbx NR and Dolby HX Pro System has the effect of doubling the improvement in the high-frequency characteristics of the HX Pro System, and an excellent dynamic range can be obtained.

It is equipped with dual operation function which can utilize the second deck.
By connecting the external deck terminal of the RS-B80R to another deck, series play, editing and recording operation, and simultaneous recording are possible.

The input terminal is mounted on the front, and can be easily connected to portable devices.

It is equipped with a blank skip, which automatically detects an unrecorded portion of more than 10 seconds and starts playback by fast-forwarding to the next recorded portion.

It is equipped with an intro search that plays only 10 seconds of the beginning of a song sequentially.

A music repeat function that allows you to repeatedly play only specified songs.

Equipped with 3-color wide range FL meter.

You can change the headphone level.

Equipped with a running direction tape run indicator.
