  • Marantz CDR640

  • ¥ 250,000 (around 1999)

Commentary

This professional CD recorder was developed aiming at the highest level of CD recorder.

It conforms to IEC60958 Type 1, an industrial standard, and has an AEX/EBU terminal.

Equipped with ATL (Auto Track Level) function, it can be adjusted in 13 steps of 5 db in the range of -70dB to -10dB.
This feature can be turned on and off as needed and can be set to match the state of the source to help prevent false recognition during track increments.

A memory buffer of up to 4 seconds (1 mb) is reserved for Q subcode processing.
The ADT (Audio Delay Time) function adjusts the recording start timing in 16 steps from 0 to up to 4 seconds to prevent audio data from being truncated.

Equipped with a copy protection function, you can choose from 3 types : OFF (unlimited copy possible), ON (SCMS compatible, unlimited copy prohibited), and HCO (complete copy prohibited).

With built-in preset keys, it is possible to reach the desired setting change mode by simply pressing both the main body and the remote control one by one to improve operability.

Control I/O terminal (D-sub9-pin, parallel) is mounted.

Equipped with a CASC ON function that allows digital input to pass out to other devices. Up to three devices can be cascade-connected.

Equipped with analog input sensitivity switching function, switching of + 4dBu/-10dBu is possible.

Analog balanced input / output terminals are mounted.
This connector is a European type with #1 ground, #2 hot (+), and #3 cold (-).

Equipped with a TOC erase function, you can erase only the TOC of a finalized CD-RW disc and record additional music or edit the final music.

Equipped with a through monitor function, it can be used as a stand-alone AD/DA and sampling rate converter.

It is equipped with an emphasis setting mode function. In the case of digital source, the preemphasis status is copied to the disc as it is. In the case of analog source, ON/OFF can be set depending on whether preemphasis is applied or not.

Equipped with a sampling rate converter that supports 12 to 56 kHz.
Two modes are installed : SRC OFF, which lights the indicator when a source other than 44.1 kHz is detected and automatically switches to SRC mode, and SRC ON, which always activates the SRC function regardless of the source type.

If the source is 44.1 kHz (± 100 ppm), a full copy can be made by turning OFF the SRC, so that data such as HDCD or DTS can be recorded as is.

It is equipped with STP (Recording Stop Condition) function which stops recording automatically in Q subcode base or ATL base. If CD or MD is the source, it stops after about 3 seconds from the lead-out.

Equipped with track increment function.
For digital sources such as CD/MD/DAT, tracks are incremented based on the Q subcode ; for digital and analog sources without category codes, tracks are incremented after 2.7 seconds of silence (ATL) is detected.
You can change the ATL configuration.

Equipped with a CD sync function, synchronous recording is possible not only for digital sources with track ID but also for digital and analog sources without Q subcode.

A wired remote control with a cord length of approximately 3m is included.
With this remote control, various operations of the main body and numeric keypad operation are possible.

EIA19-inch x2U rack mount size.
