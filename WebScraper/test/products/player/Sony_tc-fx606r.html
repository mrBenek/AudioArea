Commentary

Stereo cassette deck using linear skating system.

If you press the OPEN button on the front panel with one touch, the drawer (drawer) will come out. Set the tape here and press the button again to store it automatically.
In addition, it can be opened and closed freely while the tape is running so that the running state of the tape can be directly confirmed.

Equipped with a regenerative reverse mechanism that adopts a rotobilateral head system.
The recording / reproducing head can be rotated 180 ° and the same head gap can be used in both directions. Moreover, since the head is die-cast mounted, there is no change in sound quality even when it is reversed.
In addition, it is possible to repeat up to 5 times.

It is equipped with AMS (automatic music sensor) that can start up to 9 songs in front and back. You can specify the number of songs you want to skip with the AMS button and select songs by pressing the fast-forward or rewind button.
It also has an auto blank skip function that moves to the regenerating condition by fast-forwarding to the beginning of the next song if the no-signal part continues for approximately 10 seconds during playback.

Laser amorphous head is used for the head.
Amorphous alloys an amorphous alloy with no crystal structure, and has high magnetic flux density and magnetic linearity.

It is equipped with a linear electronic counter that displays minutes and seconds so that the recording and playback progress of the tape can be grasped in time.
This counter is equipped with a subtraction function which displays minus by rewind from 0 as a starting point.

- ∞～+ It is equipped with a peak program meter by FL tube covering a wide range of 8 dB. This meter is divided into two colors based on 0 dB so that over-level can be easily monitored.
Level sets are also useful because the best recording level for each tape type is displayed along with the meter scale.

As a noise reduction system, Dolby B type and C type are mounted.

Equipped with an auto tape selector that automatically identifies the tape to be used. By simply setting the tape, the bias / equalizer is automatically set to the optimum position.
A forced changeover switch is provided so that it can be used for tapes with no detection hole for discrimination in the cassette half.

By using the separately sold remote control unit, it is possible to control while sitting in the listening position.
By using the Synchro Remote Control Unit, you can synchronize the start / stop timing of Sony's player system with the start / stop timing of TC-FX606R recording.

Equipped with a tape running indicator to indicate the running state of the tape.

Equipped with a timer standby switch capable of continuous recording / alarm clock playback.

Equipped with auto-play / memory play functions, Rec Mute with auto-space, headphone port and long-stroke recording level volume.
