  • VICTOR/JVC SRP-B33M

  • ¥ 32,500 (around 1972?)

Commentary

A stereo record player that supports CD-4 playback only by changing the needle.

It uses a cartridge developed for 4-ch playback.
Therefore, by replacing the needle tip with a SHIBATA needle (4DT-1X), 4-ch playback is supported.

Uses TH type high-sensitivity tone arm.

It uses a low capacity cord.

Equipped with push button type speed switching, overhang indication indicator, gel stand, special rubber isolator, accessory space, etc.
