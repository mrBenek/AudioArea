  • TOSHIBA PT-124

  • ¥ 32,800 (around 1973)

Commentary

A 4-channel 8-track player that can play 4-channel and 2-channel 8-track cartridges.

A hysteresis synchronous motor is used as the drive motor.

Equipped with a program indicator that displays the number of the program being played.

A dust-proof cover is provided at the tape insertion port to prevent dust and dust from entering the mechanism.

Equipped with a program switch capable of switching between programs.

Equipped with 3 types of eject switches, you can select manual eject, イーチイジェクト eject at the end of each program, and 4 eject eject at the end of each program.

It is equipped with a repeat mechanism so that only the desired program can be played repeatedly.
