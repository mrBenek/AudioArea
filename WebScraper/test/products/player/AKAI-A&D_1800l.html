Audio Database

  • AKAI 1800L

  • ¥ 82,000 (around the late 1960 s)

Commentary

This is a multi-purpose tape recorder that supports recording / playback of open-reel and 8-track cartridge which is an improved version of 1800.

The 1800L can record stereo data back to an 8-track cartridge, stereo / mono data back to an open reel tape, and dubbing from an open reel tape to an 8-track cartridge.
To dub from a open-reel to a cartridge, load both the recorded open reel tape and the cartridge and press the transfer button.

A shield type head has been adopted for the head part to improve the S/N ratio.

Equipped with tone control.

It can be used either horizontally or vertically.

The 1800L uses a leather cabinet.

Open reel deck section
The open reel deck uses a 2-lever system that is reliable in operation and robust.
You can also switch between stereo and monaural using the track selector.

Equipped with a 3-digit tape counter with reset button.

Equipped with a tape cleaner and a tape shifter.

Cartridge part
The 8-track stereo system divides the width of the tape by 1/8 and uses two tracks for each stereo recording and playback. Therefore, you can record and play stereo four times on one tape. The channel is automatically switched at the end of the tape, but you can switch freely by pressing the program selector button.

Equipped with a cartridge program selector button that can easily select any program.

Equipped with a program indication lamp that also indicates whether the cartridge set is complete or not.

90m (300 ft) cartridge tape delivers 15 minutes per channel, one hour in a row.
