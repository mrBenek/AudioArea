  • YAMAHA CDX-580

  • ¥ 39,800 (around 1995)

Commentary

A CD player with I-PDM DAC and intelligent digital servo.

The D/A converter uses Yamaha's original DSP I-PDM DAC.
In the DSP method, 20-bit recorded data is read from the disk with the 20-bit recorded data, and converts the data into 20-bit 8-fs data. This improves micro-level reproducibility. In addition, the I-PDM method DAC, which reduces distortion in principle by making output pulses independent, is used in conjunction with the I-PDM method DAC, which realizes the beautiful reproduction of music.

It uses an intelligent digital servo system that makes use of Yamaha's digital and LSI technologies to quickly read the state of the disc loaded in the tray and realize tracing in the optimum state at any time.

It is equipped with a synchronized REC function that is system-friendly with Yamaha KX-580/KX-W382 cassette decks, and recording to cassette tapes can be done with simple operation.

Features include 3-position dimmer control, peak search, 10-key direct access, 3-mode repeat play, random access, 25-track program play, autospace, timer play, and digital volume.

Equipped with coaxial digital output.

Wireless remote control is included.
This remote control can adjust the volume and so on.
