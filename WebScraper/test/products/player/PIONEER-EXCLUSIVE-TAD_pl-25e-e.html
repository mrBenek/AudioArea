  • Pioneer PL-25E

  • ¥ 30,000 (around 1972)

Commentary

A record player with an S-shaped tone arm and plenty of accessories.

The tone arm uses a highly sensitive S-shaped static balanced tone arm with a unique anti-skating mechanism, and is compatible with various cartridges.
The needle pressure can be reliably adjusted with the direct-reading scale.

The drive part consists of a 4-pole synchronous motor, a dynamically balanced 30 cm aluminum alloy die-cast turntable, and a polyurethane belt that does not deteriorate over time.

A low-capacity code is used for the output code to suppress line attenuation.

To prevent howling, we have adopted a large insulator developed by Pioneer.

The speed selector lever adopts a lever lock system that does not operate when the power switch is off.

The oil dump type arm lifter is linked with the start lever.

The cartridge comes with PC-15 which is a MM type cartridge.

Comes with a checker that can easily adjust the overhang when replacing the cartridge.
