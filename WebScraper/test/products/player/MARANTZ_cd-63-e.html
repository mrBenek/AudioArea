Commentary

The first Marantz CD player. The CD-63B is the black model of the CD player.

The tray uses a top-loading type.
By using a disk clamp system with an ultra-precision machining spindle and a unique magnet disk clamper, the disc can be set to an accurate disc position even when the disc is tilted.

A unique balanced swing mechanism is adopted for the mechanism part.

The servocontrol uses two 8-bit microcomputers to control focus servo, radial servo, and turntable servo.

It is equipped with a specially developed error correction circuit to correct detection errors caused by dirt and dust attached to the disk.

The low-power semiconductor laser realizes energy saving, and the unique optical system configuration realizes a compact pickup.

Equipped with a digital filter LSI originally developed by Marantz, the sampling frequency is converted to 176.4 kHz by 4 times, and a Bessel type low-pass filter with excellent phase characteristics is used to cut it.

I for display, key search, communication between microcomputers, etc.2CBus system is adopted.
This improves the efficiency of microcomputer functions.

It is equipped with playback functions such as all-playback, random access program and delete playback program.
It is also equipped with a repeat function so that it can be played repeatedly when each program is played.

* When I compare CD-63 and CD-63B catalogs, I can see the difference in specifications.
