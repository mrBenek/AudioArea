Commentary

Double cassette deck for playback only + recording / playback.

It is equipped with A deck for playback only and B deck for recording and playback.
Both of the heads are equipped with super hard metal alloy heads. In addition, in order to achieve uniform running performance necessary for double deck, A deck is equipped with dummy head instead of erasing head of B deck. This realizes uniform running performance and tape tension.

Equipped with a synchronized start function for dubbing with simple operation.
With this function, the recording of B deck starts in sync with the playback start of A deck, so it is possible to dub a failure with one touch operation.

Equipped with double-speed dubbing function, switching can be done with a single switch.

It is equipped with a direct level recording function. When dubbing, the output of the A deck automatically jumps to the recording control of the B deck for direct input. Since recording is tape-to-tape, there is basically no need to set the recording level. The sound quality degradation is prevented by recording in the shortest path.

Equipped with the A-B continuous playback function, the playback of one deck starts at the same time as the playback of the other deck. This allows continuous continuous playback for long periods of time.

Equipped with Dolby B/C NR as a noise reduction system.

Equipped with an auto tape selector, both A and B decks automatically detect and set the tape position (metal, chrome, normal) just by attaching the tape.
The set position is indicated by LED.

Equipped with a timer standby function, it is possible to record absence by using a timer together.
