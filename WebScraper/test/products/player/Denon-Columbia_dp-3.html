Commentary

Full auto player without microcomputer function from DP-5.

Equipped with a full auto mechanism for automatic record size detection.

A quick repeat method is adopted for repeat playback, and the arm does not return to the rest and replays from the first track.

A linear tracking arm is used for the tone arm.
In this method, the arm and servo circuit move the arm in a straight line perpendicular to the record groove, so that there is no tracking error distortion and clear sound is achieved.
The arm uses a long cartridge integrated design arm that is resistant to record warpage.

The arm lifter mechanism uses a linear motor for the first time and a unique new mechanism, V-lock.

The rotary system adopts a direct drive system using a linear drive servo motor and DENON quartz.

The cartridge is equipped with a wide-band MM type cartridge.

Equipped with a centralized display that shows the operation status.
The operation switch is a soft touch type.

Equipped with a locate button that allows you to set the start position of the performance freely.

The safety design is such that it does not operate when there is no record.

Equipped with dynamic damping mechanism to suppress arm squeal and howling.
