  • SONY CDP-M55

  • ¥ 32,800 (released in 1987)

Commentary

A personal-sized CD player that can be combined with a mini component or radio cassette recorder.

The digital filter uses a 4 x oversampling digital filter.
The D/A converter is equipped with a left / right independent dual D/A converter.

16 songs music calendar is installed.

Equipped with a program editing function, you can create recording programs for both sides of the tape A and B at the same time. Also equipped with a delete shuffle function, which allows you to enjoy shuffle play with only the music you want to listen to.
In addition, it is equipped with a program performance function of up to 20 songs and a repeat playback function of one song / whole song / part / program / shuffle.

Equipped with a time keeper function that displays various time information in an easy-to-see manner.

It is equipped with AMS function that can quickly start a song.

Equipped with Music Search.

Equipped with an autospace function that adjusts the interval between songs to 3 seconds.

It has a sync terminal and can be connected to a Sony deck.

It has a headphone port.

It comes with a wireless remote controller that can perform operations such as one touch direct music selection with 16 keys.
This remote control has an index search function.
