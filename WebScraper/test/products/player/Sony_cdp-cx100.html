  • SONY CDP-CX100

  • ¥ 100,000 (released in 1993)

Commentary

A CD player equipped with a newly developed changer mechanism that can hold 100 CDs.

Equipped with a newly developed CD changer mechanism, the disc is set upright on a rotary table rotating in both left and right directions, making it possible to store 100 discs.
This mechanism also uses a vertical drive system, in which the disk is rotated as it is in the lifted position and signals are read from the lateral direction.

You can select the disc you want to listen to with the jog dial equipped on both the body and the remote control.
Turn the jog dial to change the disc number on the FL display and complete the disc selection. Then press the Enter button in the center of the jog dial to start playing the disc.
Also, if you turn the jog dial during a performance, the next disc number will be displayed and you can reserve the disc to be played next. If you press the Enter button, the disc will be played immediately.

It is equipped with a function to shuffle through 100 discs. You can select a disc from 100 discs at random, and then play a piece of music on the disc at random. This function allows you to shuffle one disc and then select the next disc at random.

Equipped with an UNLOAD button that moves the playing disc to the center of the front of the main body with one touch.

The D/A converter uses an advanced pulse D/A converter, which is combined with a 45-bit noise-shaping digital filter to improve sound quality.

The custom file function allows you to set up group files, disk notes & group notes, level files, and delete banks. On the CDP-CX100, these functions are numbered for each CD stocker, so you need to reset them when you replace or replace a disk.

In a group file, you can divide up to 10 groups of 100 discs, each of which can be played continuously or shuffled.

In Disc Memo, an original name of up to 13 characters can be attached to each of 100 discs. Letters (upper and lower case), katakana, numbers and symbols can be used. In addition, it is equipped with a navigation function, and when you are stuck in setting operation, the FL display informs you of the next operation procedure every 5 seconds.
A name can be set for each group in the same way as a disk memo.

The level file allows you to store the desired level in memory for each disc, so that you can automatically enjoy the sound at an easy-to-listen volume.

With the Delete Bank, you can specify which songs you don't want to listen to on the disc and then save only the songs you want to listen to. You can store the deleted music as it is.

Equipped with two gold-plated line-out terminals, fixed and variable.
It also has an optical digital out terminal.

Equipped with multi-mode timer function, timer play using external timer is possible.
With the multi-mode timer, you can select your favorite performance mode, such as continuous performance, shuffle performance, program performance, or group performance, and play the timer.

Equipped with a remote control mode switching function, Remote Control Mode 1 allows you to operate your Sony CD player. In addition, you can set three remote control modes so that you can operate up to three CDP-CX1s independently with one remote control.

Equipped with a remote control light receiving unit terminal. By connecting a separately sold remote control light receiving unit, remote control operation can be performed while the body is placed in an invisible place.

A jacket album of 100 discs is included.

You need a special adapter for 8 centimeters CD.

Wireless remote control is included.
