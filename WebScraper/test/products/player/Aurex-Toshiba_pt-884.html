  • TOSHIBA PT-884

  • ¥ 99,800 (around 1973)

Commentary

This is a 4-channel tape deck corresponding to the 4-channel model of PT 862 d.

4-channel / 2-channel recording / playback is available.

It uses a 3-head system that enables playback monitoring while recording.

It uses a unit-type printed circuit board that can maintain uniform performance and high reliability over a long period of time.

Uses bias erase oscillation circuit with less distortion.

In order to demonstrate the performance of highly sensitive tapes (Hi-Fi tape / SH-7), a tape selector switch is mounted.
In addition, the bias current can be adjusted while watching the meter.

Uses a rotation start type lever.

Equipped with a mechanical auto-shutoff mechanism that automatically turns off the mechanism when the tape is finished.

Equipped with an easy-loading mechanism for easy tape attachment and detachment.

Equipped with a tension servo mechanism for smooth tape travel.

It has a headphone port.

Equipped with a 4-digit tape counter.

Equipped with a large level meter.

If you are recording from a stereo set that is not a discrete 4-on the PT 884, you will need a CD-4 Demodulator SY 504.
