Commentary

A CD player with K2 interface circuit.

A K2 interface circuit is mounted to eliminate digital distortion caused by jitter (time difference distortion) and ripple (waveform distortion) added to digital waveform.
In this circuit, a detection circuit controlled by a precise master clock reads only the code information from the digital signal containing the non-code component and creates the same digital waveform on another block based on that information, thereby electrically suppressing the non-code component contained before passing through the K2 interface.

The D/A converter uses an 8-times oversampling digital filter and a full-time 18-bit L/R independent combination 4 dac configuration. The 18-bit is always fully operated from the time of Chikara Oire to the time of large input, realizing high precision and resolution.

In order to prevent interference between circuits, digital / analog / K2 interface circuits are used as separate boards. The K2 interface circuit also uses D/A separation pattern on the board.

A large transformer with excellent regulation is used for the power supply, and interference is eliminated by supplying power to each of digital, servo and analog circuits independently. The rectifier circuit and constant voltage circuit are also separated to provide a stable power supply.

Newly developed lightweight pickup with improved signal reading accuracy is adopted.
In addition, we have adopted a stabilizing wheel that equalizes the moment of inertia of the different 8 cm and 12 cm discs. By realizing stable rotation, the change of servo current is reduced and high sound quality is achieved.

The chassis structure is a triple chassis with high-rigidity material between the bottom base and chassis base to disperse the resonance mode of the body. In addition, an insulator is installed between the pickup / mechanical base / chassis base, and in addition, a three layer suspension structure with a large insulation foot directly attached to the bottom base is adopted.

The Triple A Ditto system is installed as a function used for tape editing, etc.
In Mode 1 "Auto Edit", the A/B side of the tape is automatically programmed according to the order of songs on the disk. If there is a song that does not fit within the specified time, you can select a song that fits within the remaining songs.
In Mode 2 "Program Edit", you can program the A/B side of the tape in the desired song order within a specified time. You can program without calculating the time by displaying the remaining time or flashing the number of songs that do not fit within the time.
In Mode 3 "Multi Edit", you can program from multiple discs within a specified time. With this function, even if you replace the disc, the remaining time display for each selected song and the flashing display for the number of songs that do not fit within the time display operate.

By adopting a remote bar for the power switch, the wiring route is minimized.

Equipped with a display ON/OFF switch to eliminate digital noise.

It is equipped with two digital outputs which are coaxial with the optical system.

Remote control electric volume is adopted.

Gold-plated output terminal is mounted.

It is equipped with the function of selecting up to 32 random program songs, index search of 99 songs, repeat all songs / one song / program / between A and B.

It is equipped with an auto space function that allows 4 seconds between songs.

Equipped with a Compulink terminal for synchronized operation.

Wireless remote control is included.
