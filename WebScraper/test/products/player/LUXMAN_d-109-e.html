Commentary

A CD player with digital output.

A linear motor is used to drive the pickup.
The linear motor consists of a drive coil, a sensor coil, and a magnetic circuit. Since it is non-contact, it realizes smooth and responsive access and improved durability.
In addition to the adoption of aluminum die-cast for each part, a ceramic material with excellent vibration resistance is used for the mechanism base, and spindle motors, pickups, linear motors for driving, etc. are mounted accurately.

Equipped with coaxial digital output terminal.

In order to reduce the mutual influence between the analog and digital sections, an independent winding structure is adopted for the secondary side of the power transformer, and the board of each block is made independent to form the power supply circuit corresponding to the connection circuit. In particular, a 21,000 μ F block capacitor, which is equivalent to a power amplifier, is adopted for the power supply of the analog section, a fast recovery diode is adopted for the rectifier diode, and the stabilization power supply circuit is a differential amplifier type to ensure low output impedance from low to wide range.
The power supply for the D/A converter is stabilized by using a double regulator so that noise does not occur in the analog filter or output circuit.

The digital filter uses a 2 x oversampling 80th linear phase FIR type digital filter.

The D/A converter uses a 16-bit current addition type based on serial input resistance ladder method. In this method, wire ring of analog and digital sections can be simplified compared with parallel input type, and noise inflow into analog section can be suppressed.
The current addition type resistor ladder system has no effect on sound quality or charge / discharge time due to the external capacitor, and enables high-speed response.

The ALPINE/LUXMAN components can be operated by the F-105 remote control by connecting with the separately sold AV surround controller (F-105).

Features include random memory music selection, repeat performance, index search, music scan play, timer play, single play, etc.

Wireless remote control is included.
