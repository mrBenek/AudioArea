Commentary

This is a direct drive auto-changer that can play up to 6 LPs continuously.

To select single multi-play, you only need to replace the center spindle, and it is designed so that you can enjoy manual play, 1 ~ 6 times and R (repeat) memo gram, and multi-play up to 6 records.

By using the provided auto / manual spindle, you can enjoy manual play.
This is a combination type that does not require automatic or manual switching. If you use an oil dump type queuing lever, you can cue up a song in the middle. Even in manual play, the Oatley turn mechanism works and the arm automatically returns to rest.

In auto-play mode, if you set the number of revolutions and record size, you only need to operate the start / stop lever. The auto-start, auto-stop, and Oatley turn operate. It also comes with a memo gram that allows you to set 1-6 times and R (repeat) continuous performances.

If you use the spindle for auto-changer, you can enjoy multi-play of up to 6 records. If you set the memogram according to the number of records, you can play up to 6 records continuously. If you set the memogram to the R position, you can enjoy repeated playing of the last record.
The center spindle used for the auto-changer is an umbrella type spindle developed by Technics, and the dust cover can be closed even when the auto-changer spindle is used.

B. FG method is used for motor speed detection.
In addition, a unique integrated D. D motor has been adopted.

The cabinet and base have a double vibration isolation structure using TNRC, an acoustic material unique to Technics.

The tone arm uses a gimbal suspension system to achieve a high initial motion sensitivity of 7 mg.

All front operation layout is adopted.
