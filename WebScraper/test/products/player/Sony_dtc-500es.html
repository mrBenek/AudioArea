Commentary

The DAT deck was developed to bring out the high sound quality of the DAT deck and various tape operations.

The A/D converter employs a combination of a 2 fs A/D converter and a 2 fs digital filter that reduces the load on the low-pass filter by doubling the sampling frequency. This makes it possible to use a low-pass filter with moderate attenuation characteristics, thereby improving the group delay and phase characteristics and enabling A/D conversion that makes more use of the original waveform.

The D/A converter block uses a 16-bit D/A converter with a 4 fs digital filter, which is exactly the same as the DTC-1000ES reference unit. Moreover, the D/A converter itself is L/R-independent, eliminating the phase shift between channels.

2D. D. + 1 BSL motor mechanism is adopted for the mechanism part.
This product is equipped with two direct drive motors for capstan and head drum rotation on a base unit made of high hard aluminum alloy with excellent processing accuracy and little aging, which is also adopted in DTC-1000ES. One motor is used rationally for take-up and supply reel while suppressing the point of tape running accuracy.

It is equipped with a digital peak margin display function that digitally displays how much more recording level is available in dB units when recording.

It is equipped with a subcode function that records absolute time from the tape top, time search using absolute time, start ID and skip ID recording, rehearsal function that can be finely adjusted when writing ID, program number function, renumber number from the head of the tape, erase function that can individually erase start ID, skip ID and program number, repeat function, and shift number function that resets all start IDs in 0.3 second increments.
You can also play RMS by remote control.

Equipped with two types of digital input / output terminals : optical type and coaxial type.

The digital / analog / display section is separated, and the internal construction reinforced by sub-chassis is adopted.

Equipped with a linear counter that can be switched to 4 modes : counter display showing the tape running time, remaking display showing the remaining time, program time showing the elapsed time of the music being played, and absolute time showing the absolute time from the tape top.

Equipped with wide range digital peak meter of -60dB to 0 dB with each ch24 segment.

It has a display OFF function in consideration of the minute noise generated by the FL tube.

It is equipped with a cue & review function at two speeds, approximately 2 x speed and approximately 16 x speed.

Equipped with a blank search function that automatically searches for unrecorded parts that have never been recorded.

Equipped with an autoplay function that starts playback immediately after rewinding the tape.

Equipped with a memory play function to rewind and play back to the position of counter 0 : 00.

By combining with the separately sold timer, it is possible to record the absence and play the alarm clock.

Equipped with an output variable headphone jack.

Wireless remote control is included.
