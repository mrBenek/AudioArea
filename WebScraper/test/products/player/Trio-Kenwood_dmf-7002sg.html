Commentary

MD deck of K' sEsule series.

Digital technology is used for both recording and playback in order to maximize the recording capability of MD.
On the recording side, input data is D. R. I. V. E. processed by 20-bit Rec D. R. I. V. E., converted into 20-bit data to reproduce minute signals buried in quantization distortion, and then ATRAC processed and recorded on disk to improve reproducibility of minute signals.
On the output side, a 24-bit D/A converter is used to reproduce signals recorded with a 20-bit Rec D. R. I. V. E. at a high S/N ratio.

High-efficiency coding technology (audio compression technology) ATRAC employs the latest ATRAC ver4.5, which performs 24-bit arithmetic processing. This improves micro-level reproducibility and expands dynamic range compared to conventional types.

The Digital Rec Volume allows you to create an easy-to-listen MD disc that matches the recording level of the source, and the Fade In / Fade Out operation allows you to set from 1 second to 10 seconds.

It is equipped with a sampling rate converter that automatically converts sampling frequencies from 32 kHz, 44.1 kHz and 48 kHz to 44.1 kHz. SCMS (serial copy management system) enables digital recording of only one generation.

Equipped with functions for moving, erasing, splitting, and connecting tracks, such as quick move which can replace the order of songs together, track move which can replace only one song, quick erase which erases multiple songs, track erase which erases only one song, all erase which erases all songs, track combine which can connect two songs, and track combine which can split one song.

It has a title input function that allows you to enter up to 1792 characters for the entire disc and up to 80 characters for each song.
By using the separately sold Cyber Tightler CT-G90, more speedy title input is possible.
It is also possible to print labels using Cyber Tiller.

The A/D converter uses 4th order ⊿ Σ + D. R. I. V. E. conversion.

Also supports monaural recording, which is convenient for long-term recording.
