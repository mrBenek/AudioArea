  • VICTOR/JVC JL-B11H

  • ¥ 23,500 (around 1973)

Commentary

A low-cost record player focusing on basic performance.

The drive section uses a belt drive system using a 4-pole synchronous motor.
In addition, the turntable part is a 31 cm aluminum alloy die-cast turntable, and the belt is made of polyurethane with carefully checked flatness.

A universal tone arm using TH system is adopted for the tone arm part.
In the TH method, the tracing ability of the arm is improved by lowering the center of gravity of the arm moving part. In addition, the force that tries to return to the correct position acts, so that even warped records can be traced stably.
In addition, it is equipped with a thread suspension type inside four scan cellar and an oil dump type arm lifter.

MM type cartridges are used.
This cartridge can be replaced with a CD-4-compatible needle for CD-4 playback.

Low-capacity code is used for output code.

A push-button type is used for changing the rotational speed.

The strong cabinet and isolator suppress howling.

The acrylic dust cover is a free hinge type and can be fixed at any angle.
Also, it can be put on and taken off.

An overhang indicating point has been installed to confirm the correct needle tip position.

There is an accessory space.
