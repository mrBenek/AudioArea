Commentary

Video CD player with 5-disc roulette changer mechanism.

Equipped with a 5-disc roulette changer mechanism, it can continuously play software across multiple discs, and the disc exchange function allows you to exchange other discs while playing.

Equipped with a scene playback function, you can call up the scene you want to watch by using the scene number.

Equipped with a bookmark function, you can register a scene you want to watch repeatedly.

It is equipped with a digest play function, and the contents of the disc can be displayed on a split screen and confirmed by image and sound.

With a built-in playback control function, you can enjoy simple interactive software using the menu screen, the player itself or the number key of the remote control, and gain operability that was not available in conventional AV media.

Equipped with a high-definition still image function, you can enjoy high-definition still images with 4 times the number of pixels in addition to standard still images with the same image quality as movies.

It is equipped with functions such as key control function which changes in 11 stages and karaoke mode in which you can reserve up to 8 songs.

Wireless remote control is included.
