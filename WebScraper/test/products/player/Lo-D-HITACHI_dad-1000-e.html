  • Lo-D DAD-1000

  • ¥ 189,000 (released in 1982)

Commentary

The first CD player announced by Hitachi Lo-D.

It uses an upright see-through front loading system that can see the rotation of the disk.

In addition to functions such as random memory music selection, one touch music selection, scanner play, memory stop, auto repeat, it is equipped with a location indicator which can tell the pickup position at a glance.
