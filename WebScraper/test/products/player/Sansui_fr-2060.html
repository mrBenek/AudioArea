  • SANSUI FR-2060

  • ¥ 33,600 (around 1972)

Commentary

A record player with an elaborate hi-fi design.

With a 16-pole synchronous motor dedicated for automatic operation and a unique return position detection mechanism, smooth operation is possible without applying lateral pressure to the needle tip.

Uses 30 cm aluminum alloy die-cast turntable.
The flywheel effect is large and stable rotation is possible.

The tone arm uses a static balance type tone arm for accurate and easy needle pressure adjustment.
The balance weight is a needle pressure direct reading type.

A MM type cartridge is used.

Both shell and arm terminals are gold-plated.

It uses an insulator made of special synthetic rubber, a low-capacity output cord, and a sturdy player case.

A fleece top type dust cover, a head shell stand and a remote controller can be connected.

A separately sold remote controller can be connected.
