Audio Database

  • YAMAHA P-701

  • ¥ 44,800 (around 1985)

Commentary

Full auto player equipped with MC cartridge.

Uses a front operation system that can be operated with the dust cover closed.

The cartridge is a dual MC type.

Uses ARS (anti-resonance straight) arm.

The motor uses a DC4-phase 8-pole coreless Hall type motor that does not generate cogging in principle, and achieves stable rotation with the quartz PLL servo system.
