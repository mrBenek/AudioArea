Commentary

A CD changer that uses a magazine system.

It uses a magazine system up to 10 disks can be loaded at one time.
This allows you to play one at a time, play ten copies in succession, or play a program over ten copies.

With the All Disc Shuffle function, you can shuffle through 10 discs of the same song without twice playing it.

It is equipped with a multi-disk program function that enables tape editing across all disks loaded in the magazine.
Insert a disc and press the multi-program button to display Disc-1. Program the song you want to insert. Then switch the disc to display Disc-2. Repeat this operation to program the song.
The program time is accumulated continuously from the first image.

Equipped with 6-mode repeat function of one disc for all tracks, all discs for all tracks, one disc shuffle, all disc program tracks, all discs shuffle, and select disc shuffle.

The digital filter is equipped with an 8 x over-sampling 18-bit digital filter with noise shaping technology.

The S-Servo III is used in the servo circuit to provide a smooth servo current by removing noise components with a low-pass filter. This stabilizes the power supply from the power supply and reduces the influence on the audio circuit.

The display section has a 10-disc 20-track music calendar.

It is equipped with AMS function that can quickly start a song.

The manual fader function allows you to fade in and out wherever you want.

Equipped with an optical digital output.

Equipped with a headphone port with volume.

Wireless remote control with 10-disk 20-key is included.
You can perform music search from this remote control.
