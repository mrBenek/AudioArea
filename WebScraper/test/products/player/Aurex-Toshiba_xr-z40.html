  • Aurex XR-Z40

  • ¥ 69,800 (around 1986)

Commentary

A CD player with skip music selection and manual search.

It is equipped with a skip selection function that allows you to call up a song you want to listen to just by specifying a song number, and a manual search function that allows you to quickly find detailed parts of a song.

It has an 8-track random memory function.

Equipped with a repeat function for all songs / memory songs.
