Commentary

This is a double deck that combines a 10-track 16-memory program playback function and recording / re-quick reverse.

One shot dubbing start button allows program selection dubbing (constant speed / double speed).

Equipped with a program index scan function that allows you to check the head of a song you want to listen to.

It is equipped with a blank skip function to skip the blank part and start playback.

We have invested in the rewind auto play function which starts playing from the head of the tape after rewinding.

It is equipped with a blank search so that you can find the blank portion of the tape.

Full logic and silent mechanism with 2 + 1 motor is adopted for both deck I and II.

Equipped with a Dolby B/C NR system as a noise reduction system.

Equipped with an auto tape select function.

It is equipped with a continuous play function that continuously plays Deck I and Deck II.

Auto / Synchro / Rec Mute function is installed in Deck II.

Recording at the same level during dubbing is possible.

Equipped with a timer start mechanism.

Equipped with a music scan function for 10 songs before and after.

Equipped with a joy-full menu function, you can enjoy music roulette and introquiz.
