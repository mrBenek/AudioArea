Commentary

The first DENON CD player.

A 3-spot type laser pickup is used for the pickup part.
This method consists of a main laser spot for code detection and two sub laser spots for tracking error detection. The reflected light from these two laser spots for error detection is converted into an electrical signal before entering an error signal amplifier. If tracking is accurate, the output of the error signal amplifier is 0. If there is even a slight difference, the error signal amplifier produces an output as an input difference. This error output goes into a servo circuit and moves the objective lens to correct the position of the main laser spot.
In addition, a focus servo system is adopted so that the focus of the focus lens always matches with the accuracy of ± 1 μ m against the vertical fluctuation of the disk.

The signal read by the laser pickup is first amplified by a preamplifier to a level that is easy to process. Then, 0 or 1 is determined by a data straw circuit. Errors are detected and corrected. The signal is rearranged in the order before interleaving. After that, the signal is converted into an analog signal by a D/A converter.

The D/A converter is equipped with a newly developed D/A converter. It generates a highly accurate signal internally in advance, uses it as a reference to configure the operation of the converter itself accurately, stores the value in the internal RAM (Random Access Memory), and automatically corrects errors caused by element variations.

The drive system uses a direct drive system equipped with a DC servo motor developed for CD players.
