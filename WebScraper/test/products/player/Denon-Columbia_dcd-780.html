Commentary

A CD player with Lambda Super Linear Converter.

The D/A converter is equipped with a LAMBDA (Ladder-form Multiple Bias D/A) super linear converter.
The lambda method uses a newly developed LSI to divide digital data into two systems, each of which is shifted near the 0 cross point by applying digital bias in the + and - directions to eliminate 0 cross distortion in principle.
When the reproduced digital data is smaller than the shift amount, there is no value crossing the original 0 cross point, so no 0 cross distortion occurs. Then, by adding the two shifted data after D/A conversion, the DC component is eliminated and the two shifted data are combined, so that an analog output with no 0 cross distortion can be obtained. There is no MSB inversion in these two data, and no 0 cross distortion occurs in the combined waveform. In addition, even if there is an error in any bit, the distortion is reduced to 1/2 because it is the average of the two signals.
In this method, when the reproduced digital data exceeds the shift amount, distortion may occur because the reproduced digital data crosses the original 0 cross point. In order to solve this problem, the MSB correction with the super linear converter is used. In addition, when there is no or only a few peak signals in a single song (more than about 99.6% of the maximum level), the lambda eliminates the shift operation and uses the converter's dynamic range to ensure a wide dynamic range.

The digital filter uses a 20-bit 8 x oversampling digital filter.
In addition, DCD 780 introduces a noise-shaping technique that reduces audible requantization noise by feeding back only the round-off error component of the digital filter output and subtracting it from the incoming data.

Equipped with a digital attenuator, the output level can be adjusted in 12 digital stages using a digital filter.

Equipped with an auto-edit function that shows the appropriate tape length for the disc you want to record.
This function automatically determines the interval between songs closest to 1/2 of the total recording time of the disc and displays the recording time in the first half and the second half.

Equipped with a display ON/OFF switch.

Equipped with a digital output terminal.

Equipped with functions such as direct selection, automatic search, manual search, program of up to 20 songs, program call, repeat of all songs / program / A-B, pause standby, timer play, auto space, etc.

Wireless remote control is included.
