  • TASCAM MD-CD1

  • ¥ 95,000 (around 2005)

Commentary

2U size MD recorder / CD player.

Dubbing is possible from CD to MD at up to 4 times speed.
In addition, continuous playback and independent playback are possible, and each deck can be treated as completely independent.

CD player part

The CD player supports playback of audio CD-R/RW discs and MP3 discs.

CD-TEXT display and ID3 tag display of MP3 files are possible.

Equipped with single / program / random playback, all tracks / single / program / random / repeat playback between A and B, fade-in / fade-out, timer play function independent of MD recorder.

Digital volume is mounted on the output.

Remote control is possible by connecting a PS/2 keyboard.

MD deck section

The MD deck part corresponds to the long play mode of LP2 and LP4.

It has Erase, Devide, Combine, Move, and Title editing functions.
You can also group and play multiple tracks.

Equipped with override function and auto-track function.

As playback functions, single playback, program playback, random playback, and repeat playback between all tracks / single / program / random / A and B are equipped.

Equipped with a sync recording function that starts recording automatically according to an input signal.
It is also equipped with recording functions such as a fade-in / fade-out recording function and a time machine recording function which can record up to 6 seconds in advance.

Digital volume is mounted on the input.

Equipped with timer play function which is independent of CD player.

Connect a PS/2 keyboard for remote control, editing, and title entry.

Other

The CD player and MD deck are equipped with independent pitch control.
The CD player has a pitch control function of up to ± 16%, a key control function of changing keys in units of ± 6 semitones without changing the tempo, and a key original function of changing the tempo by ± 16% without changing the key.
The MD deck is equipped with a pitch control up to ± 12.5%.

Playback functions such as auto-queue and auto-ready are installed in each CD and MD.

Equipped with independent CD output and MD input / output terminals.
In addition, the COMMON output jack for both CD and MD is mounted, and priority output can be set to this jack.

Equipped with a headphone port that can monitor CD/MD separately or both.

It is equipped with a constant velocity dubbing function that can reflect CD pitch control and key control.

Equipped with RS-232C terminal, external control is possible.

Wireless remote control is included.

The balanced input / output terminal is used by installing optional LA-MC1.
