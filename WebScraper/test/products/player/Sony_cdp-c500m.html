  • SONY CDP-C500M

  • ¥ 32,800 (released in 1989)

Commentary

CD changer that adopts roulette system.

The disc tray is a large rotating tray that can hold up to five discs simultaneously. This allows you to play one disc at a time, five discs in a row or five discs in a program. The disc tray also supports 8 cm single CD directly.
The roulette mechanism is designed to rotate in both directions so that you can search for the disk at the shortest distance.

It features a multi-disk program that allows you to tape edit across five disks.
Insert a disc and press the multi-program button to display Disc-1. Program the song you want to insert. Then switch the disc to display Disc-2. Repeat this operation to program the song.
The program time is accumulated continuously from the first image.

It also has an all-disk shuffle function that plays randomly through five disks.

Equipped with 5-mode repeat function of one disc for all songs, all discs for all songs, one disc shuffle, all disc program songs, and all disc shuffle.

The digital filter uses a 4 x oversampling digital filter, and the D/A converter uses a dual D/A converter with independent left and right channels.

The S-Servo III is used in the servo circuit to provide a smooth servo current by removing noise components with a low-pass filter. This stabilizes the power supply from the power supply and reduces the influence on the audio circuit.

The display section has a 5-disc music calendar.

It is equipped with AMS function that can quickly start a song.

The manual fader function allows you to fade in and out wherever you want.

Equipped with a headphone port with volume.

Wireless remote control is included.
You can perform music search from this remote control.
