Commentary

This is a limited version CD player based on C-711M and equipped with quality parts for the highest transparency.

To increase the purity of the music signal, we add luxury features such as careful selection of components, shielding, and specially ordered high-grade pin cables.

Uses CD mechanical shield construction.

Uses a shield transformer.
DuPont CORIAN DuPont CORIAN insulator, which has excellent rigidity and vibration damping.

Uses a very thick power cord with polarity display.

Equipped with LC-OFC class 1-pin cable (with spot connection and directional display)

The D/A converter is equipped with FPCS.

Continuous peak search function and CD editing and recording are supported with K-511M/K-W511M (not supported with K-511THM)

Can be linked with the system remote controller of A-911M/A-911MLTD/R-811M.
When used alone, it can be operated by the optional remote control RC-133C.
