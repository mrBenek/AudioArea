Commentary

Stereo cassette deck equipped with Computer Brain and Dolby HX Pro.

It is equipped with Dolby HX Pro which applies active servo bias theory.
The Dolby HX Pro keeps the total effective bias constant by adjusting the bias to a deep level when there is a large number of mid - and low-frequency components and to a shallow level when there is a large number of high-frequency components. As a result, the saturation phenomenon of tape, which is easily caused by high-level high-frequency signals, is suppressed, enabling clear recording.
Also, Dolby HX Pro works only when recording, so it can be played on any deck.

It is equipped with a computer brain function that automatically sets the bias value, recording sensitivity and equalizer value in order to bring out the performance of the tape.
With the Computer Brain function, there are 32 adjustment points for bias, equalizer, and recording sensitivity, each of which selects the optimum value for the tape from 30000 choices in about 16 seconds.
In addition, it is equipped with a 3-position data memory in which data adjusted by computer brain is stored separately by normal chrome metal.

The mechanism part uses a super-stabilized torsion dual capstan system.
In the dual capstan system, two capstan units are installed, each of which is slightly displaced and rotated to improve uneven tension in the cassette half, and the tape and head are always pressed together with a constant force.
In the superstabilized dual-capstan system, the capstan is further subjected to a special surface treatment using micrograin processing to reduce slippage between the tape and the capstan and improve head touch.

Equipped with a tape counter that displays the amount of remaining tapes in minutes and seconds.
Depending on the type of tape being used, the tape length switch can be toggled to count the remaining recording time from the C-46 tape to the C-120 tape. This counter also operates during fast forward and rewind.

DX Combination 3 head has been adopted for the head.

Equipped with a Dolby B / C type noise reduction system.

Equipped with ADMS which is an automatic degaussing system, it prevents the head from being magnetized.

It is equipped with an intro play function that automatically plays the head of a song every 8 seconds.

It has a music sensor function to find the head of a song.

Equipped with FL meter with peak hold.

It has a playback volume.

Equipped with a repeat mechanism.

It has a queue review function.

Equipped with auto / manual rec mute.

Equipped with an auto tape selector.

Equipped with an MPX filter ON/OFF switch.

Remote controller connection terminal is mounted.

Equipped with a timer standby mechanism, and by connecting an external timer, it is possible to record absence and play alarm clock.
