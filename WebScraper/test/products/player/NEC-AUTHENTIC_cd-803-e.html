  • ¥ 215,000 (around 1982)

Commentary

The first CD player announced by NEC.

In order to realize a CD-player, Line 5 control circuits such as pickup control circuit, laser control circuit, disk motor control circuit, feed motor control circuit and system control circuit are mounted.

It uses a coreless DD motor exclusively for disks and a pulse control motor for the pickup feed, enabling position control in the micron unit.
The servo system of the disk motor is equipped with an originally developed 3-stage action servo, which consists of a starting servo system using an action servo system and a twin servo system during operation, to realize a stable signal pickup.

The D/A converter uses a 16-bit D/A converter developed by NEC.
It quickly corrects errors caused by element variations and effects of operating temperature, and realizes a low distortion factor.

Equipped with NEC's original 16-bit digital filter.

Four types of play time display are available. By pressing the TIME key, you can display the real time of the currently playing song, the real time of the currently playing song, the play time of all songs on the disc, the remaining time of the currently playing song, and the remaining time of the entire disc.

With a built-in auto-start mechanism, you can connect it to an audio timer (sold separately) and turn on the auto-start switch on the back to start automatic performance at the timer set time.

It is equipped with functions such as a memory player that can program up to 99 songs randomly, a cue / review function for selected songs, memory / repeat of all songs, etc.

A remote controller is included.
