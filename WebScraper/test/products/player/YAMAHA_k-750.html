  • YAMAHA K-750/K-750S

  • ¥ 74,800 (around 1983)

Commentary

Reverse deck equipped with various functions.

It uses a rotary-head type Oatley berth that incorporates a Sendust recording / recombining head and an erasing head in the head block. It also uses an azimuth adjustment mechanism, a head center adjustment mechanism and a head anti-shake mechanism in which the FWD side and the RWD side are independent in order to realize high-quality recording and re-recording by preventing the head from shifting from the tape position, which tends to be a problem in reciprocating recording and re-recording.

A dedicated assist DC motor is provided for vertical movement and inversion of the head block, and a 2 + 1 motor separate drive system is adopted using a DC servo motor for driving the capstan and a flat torque DC motor for driving the reel to prevent mutual interference of each rotating system.
In addition, similar to the K-1000/K-2000, a non-clutch reel system has been adopted to speed up the operation response and reduce noise.
The flywheel uses a high precision dynamic balance type large flywheel with roundness of 0.1 μ and tip runout of 1 μ or less.

The amplifier section uses Yamaha's original pure current dam method. This method always supplies a constant current from the power supply to the amplifier. No signal current flows into the ground line or power supply line, so a clean power supply current can be obtained. This eliminates the negative influence of various non-linears existing inside and outside the circuit, and improves the ability to reproduce minute signals.
In addition, a low-noise operational amplifier is used for the recording amplifier, and the ± power supply is also used for metal tape.

Equipped with a Dolby B/C type noise reduction system.

Equipped with an auto fade out / in function at the tape end.
When this function is set to ON, auto-fade-out recording is started when the counter level indicator reaches 0.00, and auto-fade-in recording is started after reversing.

It is equipped with a program recording function, and by programming at the time of recording, tape recording can be done automatically according to it.

Equipped with fade-in / fade-out function.
Fade-in is set to approximately 8 seconds, and fade-out is set to approximately 4 seconds.

It is equipped with a function to display the remaining amount of the unrecorded. The remaining time of the tape is displayed on the counter with one touch operation, and at the same time, the part which can record is put in standby state.

Equipped with an automatic source change function, the source is automatically switched to the tuner, etc., even when the tape playback ends, and music playback can continue.

It has a monitor speaker out function so that you can check the sound while recording.
The K-750 has a speaker connector on the rear panel that allows you to monitor with a speaker without going through an amplifier.

It has a linear counter that can check the running time of the tape in minutes and seconds.

Equipped with a memory stop function.

Equipped with a REC return function that returns to the playback standby state after rewinding to the recording start point.

It is possible to start one song before and after.

Equipped with 3 types of reverse mode.

With a built-in blank skip function, you can automatically start fast-forwarding and start the next song when a silent part lasts for more than 15 seconds during playback.

It has a repeat function that allows you to repeat the same song or program up to 15 times.

Equipped with an intro-scan function that repeats intro-playback for about 10 seconds each.

Program playback is possible.

There are 2 kinds of variations, black model K 750 and silver model K 750 s.

It can be operated from a remote location by the optional wired remote control.
