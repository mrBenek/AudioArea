  • Technics RS-2760U

  • ¥ 108,000 (around 1976)

Commentary

Stereo cassette deck with direct drive system.

An HPF head has been adopted for the head.

Equipped with a level adjustable Dolby NR circuit.

Equipped with a peak check meter for accurate level adjustment.

It is equipped with functions such as memory rewind, tape selector, input selector and operation indicator lamp.

An electronic control system is used to prevent sparks and noise due to poor contact by using transistors instead of relays as switching elements.

Input / output levels can be adjusted independently on the left and right.

By combining with a timer (sold separately), it is possible to record absence.
