  • DIATONE DP-82B

  • ¥ 49,800 (Released in November 1969)

Commentary

An improved version of the DP-81B player system has been released.

The motor is a 4-pole, 16-coil condenser starting hysteresis synchronous motor.
We have also adopted a belt drive system using a polyurethane endless belt with precision polishing on both sides to achieve stable regeneration.

A precision processed 31 cm turntable is used.

A non-resonant static balanced pipe arm is adopted, and a plug-in head shell system of European type needle pressure direct reading type is adopted.
It is equipped with a lateral balancer and an inside four scan seller. In addition, it uses a direct-reading needle pressure application method that does not require a needle pressure gauge for needle pressure adjustment.

The cartridge is equipped as standard with a high-compliance type IM cartridge that adopts a newly designed holding mechanism.

* There may have been two versions with different details.
