  • SONY TC-V7

  • ¥ 68,000 (released in 1983)

Commentary

Auto reverse cassette deck with linear skating system.

It uses a rotobilateral system in which the head rotates 180 °, and realizes reverse reproduction with uniform characteristics in both directions.
A laser amorphous head is used for the recording / reproducing head.

It is equipped with an AMS function that can start up the front and back 9 songs.

It is equipped with a blank skip function that automatically searches for the next song by fast-forwarding when a silent part lasts for about 10 minutes during playback.

It is equipped with a cycle function that plays back and forth one tape up to 5 times continuously.

Equipped with a linear electronic counter that displays the amount of tape travel in minutes and seconds.

Memory start / stop is possible with the memory counter.

In addition to Dolby NR / B type, it is equipped with Dolby NR / C type.

Equipped with a centralized display by FL tube.

Equipped with Auto Tape Selector, Auto Play function and Rec Mute function with Auto Space.

The timer (sold separately) allows continuous recording and alarm clock playback.

Remote control via wired / wireless / foot remote control (sold separately).
In addition, synchronized play is possible with the synchronized remote control.
