  • SONY TC-PX100

  • ¥ 22,000 (released in 1998)

Commentary

This is a single reverse cassette deck with the same design as Sony's MD Pixy DHC-MD777/MD555.

In conjunction with DHC-MD777/MD555, you can enjoy CD synchronized recording and timer recording.

A pull-out type horizontal loading mechanism is adopted for the mechanism.

Equipped with a Dolby NR B/C type noise reduction system.

It is equipped with an AMS function that makes it easy to find the beginning of the current or next song.

Compatible with metal tape.
