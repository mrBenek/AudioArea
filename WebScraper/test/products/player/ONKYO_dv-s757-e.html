  • ONKYO DV-S757

  • ¥ 80,000 (released on September 18, 2001)

Commentary

THX Ultra certified DVD-Audio/Video player.

It has a 192 kHz / 24-bit D/A converter for audio and a 54 MHz / 10-bit video D/A converter for video.

It uses a 2-wavelength twin focus pickup compatible with CD-R/RW and MP3.

It uses a direct digital path circuit.

Equipped with video circuit off function.

Equipped with a setup navigator that enables easy and interactive initial setup.

Equipped with a Dolby digital and DTS decoder.

Equipped with a high-speed resume function and last memory function that are convenient for interrupting during playback.

It is equipped with a search function that allows quick playback by specifying a chapter or time.

It is equipped with a repeat play function that repeatedly plays your favorite tracks and chapters, a program play function that plays in your favorite order, and a random play function that plays in any order.

It is equipped with a condition memory function that can store up to 15 DVD settings you often watch.

Equipped with a screen saver function to prevent the display from burning.

It uses an RI terminal that enables the DV-S757 to be operated by remote controllers such as Onkyo AV amplifiers with RI marks.

Wireless remote control is included.
