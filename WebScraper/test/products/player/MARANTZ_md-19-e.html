  • Marantz MD-19

  • ¥ 75,000 (released in 1998)

Commentary

MD deck with Marantz's original audio technology.

Equipped with 20-bit ADC and 24-bit DAC.

It uses the latest version of ATRAC for 24-bit arithmetic processing.

The audio output uses the latest version of Marantz's original HDAM module.

It is equipped with an adaptive high-frequency control technology that improves the reproducibility of the high-frequency range and the natural sound spread.

Uses a low-noise, high-speed discrete power supply circuit.

Equipped with monitor mode which can use MD-19 as a D/A converter.

It is equipped with a sampling rate converter for various digital sources.

Equipped with a balanced volume.

CDTEXT can be displayed and recorded by digital connection with a CD player.

There are 2 color variations, gold and black.

Wireless remote control is included.
This remote control is equipped with a TEL key input function that enables you to enter Katakana characters with the same operation sense as a pager.
