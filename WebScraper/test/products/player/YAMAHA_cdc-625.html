  • YAMAHA CDC-625

  • ¥ 49,800 (released in 1992)

Commentary

This is a five disc CD changer that allows you to change discs while playing.

It uses a carousel type in which 5 disks are set in a large tray.
In addition, the Play Exchange feature allows you to open the tray and replace one disc while playing (not during programming).

It is equipped with functions such as a program performance that allows you to select up to 20 songs of your choice from a set of 1-5 discs and play them in your desired order, a single repeat that allows you to play one song repeatedly, a full repeat that allows you to play the entire disc repeatedly, and a random performance that allows you to play up to 99 songs by changing the order of songs at random.

It is equipped with functions such as direct selection, jump selection, disk skip, display switching, single mode / all mode switching, etc.

Equipped with a dimmer switch that can change the display brightness in 3 levels.

Equipped with an index search.

Equipped with a music calendar.

Equipped with a timer play function.

It has a headphone port.

Wireless remote control with remote control volume is included.
