  • SONY CDP-111

  • ¥ 145,000 (around 1983)

Commentary

Sony's second generation CD player

Data demodulation, error correction processing LSIs and memory ICs are used to demodulate data, correct errors, and absorb rotational irregularities. A 16-bit D/A converter converts digital data into analog signals.

Equipped with an index search function.

Equipped with an AMS function that allows you to quickly find the music you want to listen to.

A repeat play function that allows you to play all songs / one song / part repeatedly in three patterns.

A two speed music search in which you can listen to the contents of a song while fast forwarding.

Wireless remote control (RM-111) sold separately enables long-distance operation.

Auto pause function to wait at the end of each song (can be switched ON/OFF)

BEEP sound ON/OFF function.
