  • Marantz CD-16D

  • ¥ 160,000 (released in 1997)

Commentary

A CD player equipped with a digital input / output terminal.

The mechanism part is equipped with linear tracking mechanism CDM 12.1.

The D/A converter uses a bit stream D/A converter Triple 7.

It is equipped with a balanced output amplifier using Marantz's newly developed Dual HDAM.

Uses a copper-plated die-cast chassis, a 3-mm-thick copper-plated steel plate base plate, and a 3-mm-thick aluminum top cover.

Comes with an aluminum top remote controller that can control the output level.
