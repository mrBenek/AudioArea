  • SONY TC-835

  • ¥ 39,900 (released in 1975)

Commentary

An 8-track stereo deck capable of recording and playback.

Equipped with a 3-mode selector.
In EACH mode, it stops at the end of each program. In 4 → 1 mode, it stops at the fourth program. In CONT mode, continuous playback is performed.

It is equipped with a program track selection button and a display button which can freely select the program of any track.

Equipped with an auto-stop mechanism, it automatically stops when the program ends in each mode (except OCNT).

It includes a program display lamp, fast-forward button, illuminated level meter, and independent left and right recording and re-volume.

Microphone recording is also possible by using an optional microphone.
