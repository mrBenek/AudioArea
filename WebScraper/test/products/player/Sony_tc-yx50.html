  • SONY TC-YX50

  • ¥ 50,000 (released in 1982)

Commentary

Liberty series cassette deck with Dolby NR / C type.

In addition to Dolby NR / B type, Dolby NR / C type is newly installed.

Equipped with an auto tape selector, the type of tape is automatically determined by the detection hole of the cassette half and the tape position is automatically switched.

A 2-motor configuration full logic control system using dedicated DC motors for capstan drive and quick winding is adopted.

Equipped with a 12-element x2 LED peak program meter.

Equipped with Longs trawl slide volume and left / right level balance volume.

SD head is used for the head.

Designed according to the same design policy as the compact receiver STR-YX50 of the Liberty Series.

Remote control via wired / wireless / foot remote control (sold separately).
In addition, synchronized play is possible with the synchronized remote control.
