Commentary

A manual record player using belt drive system.

A 4-pole hysteresis synchronous motor is used as the drive motor.

Uses S-shaped pipe arm with knife edge support system that allows faithful tracing.

A MM type cartridge is used.

Thick board is used for the cabinet.

Equipped with an insulator to prevent howling.
