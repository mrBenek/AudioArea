Commentary

A multi-functional cassette deck with built-in electronic digital timer, capable of recording absence, sleep timer and alarm clock playback.

It has a built-in high-precision digital timer to further develop the absence recording function.
The digital timer section, which requires accuracy, employs an electronic system using newly developed MOS LSI (Ultra-Integrated Circuit). The start time can be set at one minute intervals and the error with respect to the set start time is also kept low.
In addition, the amount of time light can be switched to two levels.

With the Timer knob, you can set the time for recording your absence, playing an alarm clock, or operating as a sleep timer.
59 minutes can be set with 1H-ON, 118 minutes with 2H-ON, and 59 minutes with SLEEP. When the set time ends, the power to the deck and AC outlet is automatically turned off.
The DISPLAY knob is used to set the current time and recording start time.
The ADJUST knob advances the timer to the desired time. It can be set quickly to near the time set by FAST, while it can be set slowly by SLOW. The sleep timer operates by SLEEP START.

If the outlet is unplugged or there is a power failure, either AM or PM will flash to indicate the current time error when the power supply is restored after a lapse of time.

The drive section uses a DC-motor with a built-in generator with less torque ripple and vibration, and a capstan and flat belt with a precision finish to ensure stable tape drive.

It is equipped with a tape selector with a built-in automatic switching mechanism for chrome tapes. When a chrome tape with a detection hole is attached, the bias and equalizer are automatically switched to chrome regardless of the position of the tape selector.
Also, manual switching is possible for LH tape and Ferrichrome tape.

In order to securely fix the cassette, a 6-blade reel with less play between the tape and the hub is used.

Equipped with a memory stop on-off switch that is convenient for tracking.

Equipped with a tape mechanism protection device that prevents the mechanism from differentiating if the tape is not set correctly.

Equipped with a back illumination lamp with a tape end marker line that shows the remaining amount of tape at a glance.

Equipped with a full auto-stop mechanism that employs a Hall element.
