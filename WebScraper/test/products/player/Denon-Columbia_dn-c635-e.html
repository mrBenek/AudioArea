  • DENON DN-C635

  • ¥ 73,500 (tax included, sold on March 28, 2006)

Commentary

Professional CD player with balanced output and serial / parallel remote terminals.

Equipped with a sleep mode that reduces the burden on the CD mechanism when power is turned on for a long time.

It is equipped with an instant start function that enables instantaneous sound output.

The frame search function enables frame-by-frame searching and head-tracking, and the track select knob enables easy track selection.

Pitch control of ± 12% is possible with the pitch slider.

It is equipped with an end monitor function that plays only the end portion of the track and an end of message (EOM) function that signals the end of the track.

Up to 99 songs can be played.

Equipped with a shock proof memory of 10 seconds.

There was a wired remote control sold separately as an option.
