  • SONY MTL-10

  • ¥ 49,800 (released in 1983)

Commentary

A cassette player that can store 10 prerecorded cassette tapes and play them in the order you want.

It can be used by storing 10 volumes of tape in the cassette magazine. The tape can be played from any tape regardless of the k-ner number of the stored magazine. In addition, it is equipped with a random memory that can reserve up to 10 volumes, so it can be played continuously or repeatedly.
In addition, each volume can be selected by tape type and Dolby ON/OFF with the select switch on the side of the magazine.

The Oatley Berth uses a rotobilateral system in which the head is rotated by 180 ° at the end of the tape, so that the same head is used both ways.

It is equipped with AMS and blank skip function to realize smooth start of music.

Laser amorphous head is used for the head.

Uses a Dolby NR / B type noise reduction system.

All stages are DC amplifier configuration.

Equipped with a head cleaning window, the head, capstan, and pinch roller can be cleaned.

There are 3 variations of silver, blue and red.

A cassette magazine equivalent to the accessories was sold separately.
