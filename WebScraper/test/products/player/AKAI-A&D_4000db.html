Commentary

New4000 series tape deck with Dolby system.

The Dolby System is installed as a noise reduction system.

A 3-head system has been adopted for the head section, and dedicated heads are used for erasing, recording, and playback, enabling simultaneous monitoring during recording.
The VU meter is also linked with the monitor switch and automatically changes to the input level and playback level of the program source.

A 1 micron gap head is used for the head.

A 2-lever system is adopted in consideration of operability.

Equipped with a 2-position tape selector, it provides optimum frequency compensation for both general low-noise tapes and wide-range tapes. The equalizer curve is set by selecting Scotch #211 as the reference tape in the low-noise position and selecting a low-noise tape sold by Akai in the wide-range.

With sound-on-Sound and Sound-with-Sound recording capabilities, you can record on one track and transfer it to another track and make several recordings into one.

It uses a circuit design in which each input of the line and microphone is independent, and mixing recording is possible.
It is also possible to mix with the line input using the DIN input instead of the microphone.

Equipped with an automatic shut-off mechanism. When the tape reaches a wound or cut position, the power of the whole unit is turned off by the shut-off switch.

Equipped with a pause mechanism, tape travel can be stopped temporarily while recording / regenerating condition. If you release the pause button, recording / playback continues.

Equipped with a recording indicator lamp.

Equipped with a 4-digit tape counter.

There was a dust cover sold separately.
