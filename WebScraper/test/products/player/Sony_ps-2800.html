  • SONY PS-2800

  • ¥ 31,800 (around 1977)

Commentary

A record player with a built-in equalizer amplifier.

The equalizer amplifier section has an on/off switch that can be used as a normal player.

A 30 cm aluminum die-cast turntable is driven by a belt drive system, and a synchronous motor developed specifically for the turntable is used as the motor.
In addition, it uses a double-side polished urethane belt.

A new type of rubber sheet is used to suppress harmful vibrations of the phonograph record.
This rubber sheet is designed to fit the phonograph record into the sheet, reducing unnecessary vibration.

Static balance type universal tone arm is used for the arm.
It is equipped with an inside four scan seller and a lateral balancer that cancels the inside force according to the performance point of the record.

A MM type cartridge is used.

Uses an insulator that can be adjusted in height.

Equipped with auto-return and auto-cut mechanisms.

The output code is a low-capacity type with less high-frequency loss.

Comes with a removable acrylic dust cover.
