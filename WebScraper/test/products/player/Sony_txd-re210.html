  • SONY TXD-RE210

  • ¥ 40,000 (Released in September 1997)

Commentary

A CD / cassette deck that combines a CD player and an auto reverse cassette deck.

With a single CD sync button, you can easily check the disc → set the recording level → start recording.

It is equipped with a time edit function that automatically selects the longest song among the remaining songs when the next song does not fit in the remaining time of the tape.

It is equipped with a sound auto-focus function that detects the peak of the source and automatically sets the recording level.

Equipped with a fader button that allows you to fade in and fade out at the touch of a button when recording.

Equipped with a Dolby NR B/C type noise reduction system.

It is equipped with a multi-AMS function that can cue up to 30 songs before and after.

The CD player and deck are equipped with independent input / output terminals. Especially, the CD player is equipped with an optical digital output terminal.

Wireless remote control is included.
