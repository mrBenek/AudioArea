  • SONY TC-FX4

  • ¥ 43,800 (released in 1980)

Commentary

This is a cassette deck that supports synchronized play.

It is compatible with optional wireless remote control, wired remote control, and remote control unit for enjoying synchronized play.

It utilizes the operability of the logical control that allows position change without going through the stop mode, and uses the fine feather touch operation that realizes a lighter feeling of operation.

Uses a simple 2-motor configuration mechanism.

The head uses an SD head compatible with metal tape.

Uses Dolby NR / Complete IC.

Equipped with auto-play function.

Equipped with Rec Mute function with auto space.

Equipped with a timer standby switch.

Equipped with Line/Mic input switching.

Equipped with a twin level meter.
