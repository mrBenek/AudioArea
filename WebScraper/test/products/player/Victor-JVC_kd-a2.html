Audio Database

  • VICTOR/JVC KD-A2

  • ¥ 34,800 (around 1979)

Commentary

Component type cassette deck.

KD-A1 and KD-A2 have the same specifications except for the width.

Uses a dual-ball direct loading system (with Head cover) for more stable tape travel.

It is an erect full-view type that can confirm the tape running at a glance.

Equipped with a large backlit VU meter.

It has a tape selector with independent bias and equalizer.

Uses a Chronios head.
