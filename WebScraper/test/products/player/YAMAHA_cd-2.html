  • YAMAHA CD-2

  • ¥ 138,000 (around 1984)

Commentary

A CD player with digital filters.

The CD-2 uses an LSI with a built-in digital filter, which moves unwanted interfering signals and greatly reduces the load on the analog filter.

The power supply circuit uses oxygen-free copper wiring, a large low-impedance power transformer and a copper bus bar.

It is equipped with 12-track random memory and index search, 3-mode repeat playback to repeatedly play between any two points / all tracks / all memory, space play function to create 3-second interval between songs, etc., and 3-mode search function.

Wireless remote control is included.
