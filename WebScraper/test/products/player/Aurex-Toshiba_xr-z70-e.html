Commentary

The second Rolex CD player.

It has adopted a simple operation that can perform most operations just by pressing 8 key switches.

It is equipped with basic CD player functions such as random access, random memory function, skip play function and auto-repeat function, as well as auto-pause function which can be put into a pause state at the end of each song.

The XR-Z70K is not only a black model, but also a sound-quality model that takes advantage of the basic design of the XR-Z70 and refines the sound quality of analog circuits.

There was a wireless remote control sold separately.
