  • TASCAM CD-RW5000

  • ¥ 159,000 (around 1999)

Commentary

A commercial CD recorder capable of recording to CD-R, CD-RW, CD-R-DA, and CD-RW-DA.

In addition to the wired remote control, external control via Control I/O is also possible.

It is equipped with various input / output terminals such as analog balanced input / output (XLR type, with output trim function), analog unbalanced input / output (RCA pin), AES/EBU digital input (IEC60958 Type1), coaxial digital input / output (IEC60958 Type2) and optical digital input / output (TOS, IEC60958 Type2).

Equipped with an auto track function and a manual track increment function.

Equipped with Fs conversion function. If Fs = 44.1 kHz, the Fs converter will pass through.

Equipped with a sync start function.

Equipped with ERASE function (CD-RW only).

Program playback and repeat playback function.

SCMS Ready (Input : free, Record : SCMS)

L/R independent input level adjustment function.

Headphone port

2U rack-mount size.
