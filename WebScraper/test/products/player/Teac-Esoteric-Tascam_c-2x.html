  • TEAC C-2X

  • ¥ 165,000 (around 1981)

Commentary

Stereo cassette deck compatible with 2 speed and Dolby HX system.

It is compatible with 2 speed tape speeds, and can be switched between standard speed (4.8 cm) and double speed (9.5 cm). With double speed, the recording wavelengths on the magnetic surface are doubled. This increases the maximum recording level, the dynamic range, S / N ratio, and wow flutter.

The Dolby HX system is used to maximize the dynamic range of the tape.

Uses a 3-head 2-motor system.

Closed loop dual capstan system is adopted for the mechanism part.

The tape selector is equipped with a metal tape position.

Equipped with a dedicated test input position for tape adjustment.

Equipped with a tape slack removal mechanism.

Equipped with a tape monitor with CAL position.

Equipped with an auto-rewind mechanism convenient for repeated playback.

Equipped with a timer standby mechanism.

Equipped with a BIAS/EQ independent 3-stage switching mechanism.

Equipped with a differential type left / right interlocking volume.

Equipped with a REC MUTE mechanism.

Equipped with a Dolby NR system as a noise reduction system.

Equipped with a separately sold remote control unit connection terminal.

There was a rack handle as an option sold separately.
Can be mounted in an EIA (19 inch) standard rack.

The C-2X had a wide array of peripherals that could be upgraded to meet the needs of the application.

By using the plug-in BIAS/EQ card CX-8, fine adjustment can be made according to the tape to be used.

Dbx noise reduction unit RX-8B can be connected.

The MX-8 is a 4-input, 2-output audio mixer. Each input channel is equipped with a microphone (with 20 dB attenuator) / line switching mechanism, low-cut filter (100Hz/200Hz), and pan pot.

The CS-8 is a portable case that can mount C-2X, MX-8, and RX-8 simultaneously. The front and rear lids can be removed for use, so it can be used as a stationary system rack.

The TO-8 is a battery-powered test tone oscillator for adjusting the BIAS/EQ blank card CX-8, and can be used to adjust all tape decks, not just the C-2X.
The frequency can be selected from 400 Hz, 6.3 kHz and 12.5 kHz and the output can be selected from -10dB and -30dB.
