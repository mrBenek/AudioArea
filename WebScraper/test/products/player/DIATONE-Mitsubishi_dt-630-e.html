  • DIATONE DT-630

  • ¥ 62,800 (around 1979)

Commentary

Cassette deck equipped with MPSS function.

Equipped with an automatic head-tracking system (MPSS = Music Program Selecter System) that enables you to listen to several songs by skipping from the currently playing song. This system uses a no-signal detection method to detect the no-recording part between songs and operates.
After setting the automatic music selection switch to the MPSS position, press the Set Number of Songs button for the number of times of the music to be skipped. The number of songs set is digitally displayed in the Number of Songs display area. This allows you to automatically start playing up to nine songs after the current song and up to eight songs before the current song.

With a combination of a tape counter and auto-stop mechanism, it is equipped with a memory rewind mechanism that rewinds the tape to the set position and stops it. When playback / recording starts, turn on the memory switch and set the tape counter to 000. Press the rewind button after playback / recording to stop the tape at the set position.

It is equipped with the REC MUTE function. If you turn the control lever ON during recording, there will be no recording during the recording.

The newly developed SENDUST head is used for recording and reproducing heads, making it compatible with metal tapes.

Equipped with a 4-position tape selector.

The tape control button uses the ST mechanism (soft touch operation).
This ST mechanism operates the head housing using the driving force of the motor, so there is no mechanical load on the play button. It also realizes one touch recording, which can be recorded by pressing only one button.

15-point peak level indicator is used.

Equipped with IC Dolby circuit.
