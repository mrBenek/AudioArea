  • Audio Dynamics C-3

  • ¥ 40,000 (around the mid-1980s?)

Commentary

A full-component size CD player developed focusing on the basic performance of the Eurostyle component series.

A 3-beam laser pickup is used for the pickup part.

A 16-bit linear D/A converter is used for the D/A converter.
A seventh order passive filter is used in the filter circuit.

It is equipped with functions such as 16-track random program memory, 1-track / all-track repeat and skip search.

Wireless remote control is included.
