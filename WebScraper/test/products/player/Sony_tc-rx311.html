Commentary

Single reverse cassette deck with auto-calibration function.

It is equipped with an auto-calibration function that allows you to set the recording condition to match the tape with a single button so that you can record with the ability of various tapes.
This a result, bias calibration, which adjusts the bias value during recording, and level calibration, which adjusts the level difference during recording and playback due to the difference in tape sensitivity, can be performed simultaneously, making it easy to record according to the tape to be used.

In order to reduce hissing, Dolby B/C NR is incorporated in the noise reduction.
It also incorporates the Dolby HX-PRO, which controls the bias value in units of one thousandth of a second during recording and increases linearity in the high-frequency range.

When the deck is in a recording state or a recording pause state, it is equipped with a REC MUTE function with auto space that records approximately four seconds of silence on the tape and sends only that amount of silence to the recording pause state.
By simply pressing the REC MUTE button, you can create a tape with a well-organized space between songs, for example, when dubbing multiple CDs with subtly different space between songs onto a single tape.

It is equipped with a multi-AMS (automatic music sensor) that can cue up to 30 songs before and after counting from the currently playing song.

Equipped with 3-mode tape travel switching of one way, one round trip and repeat round trip (5 round trips).

An auto tape selector that automatically determines the type of tape (normal, high position, metal).

MPX filter switch that cuts the pilot signal for FM air check.

16-segment FL peak level meter.

4-digit Tape Counter with Reset / Memory Function

Stereo headphone jack

It is compatible with wireless remote control which enables basic operation of the deck with Sony's AV remote control.
