  • ONKYO CP-55A

  • ¥ 30,000 (around 1974)

Commentary

A belt drive type semi-auto player designed for basic performance.

A 4-pole synchronous motor is used for the drive part, and a special urethane belt is used for the belt.
In addition, a 30 cm aluminum die cast with good balance accuracy is used for the turntable.

The tone arm uses a static balance S-shaped pipe arm, and is equipped with an anti-skating system and lateral balancer.

A MM type cartridge is used.

The insulator adopts a shock absorption method.

Uses a removable fleece top type dust cover.

In order to support 4-channel playback, a low-volume connection code with less high-frequency attenuation is adopted.

Needle pressure scale is engraved on the counter weight part so that it can be accurately applied while reading the proper needle pressure.

Head shell can be removed with the international standard 4P connector.
