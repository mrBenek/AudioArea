  • Pioneer PD-M70

  • ¥ 79,800 (released in 1987)

Commentary

A CD changer that can play multiple CDs using a magazine that can hold six CDs.

It is equipped with 32 songs program play which can be freely programmed from 6 CDs, and random play function which plays from 6 CDs at random.

The included remote controller is equipped with a memory function that can store your own programs in 8 ways.

It also has a magazine for playing one piece of music so that it can be used as a normal CD player.
This magazine is the same type as a normal CD player in which the tray pops out when the eject button is pressed.

Equipped with digital level control that can be adjusted from 0 dB to -25dB in 1 dB steps using the level key.
You can eliminate variations in the volume of each CD.

The use of a disk stabilizer and a honeycomb chassis reduces unnecessary vibration.

Uses an oversampling digital filter.

You can share the magazine with Carrozzeria-branded in-car magazine-type CD players (KEX-M700, CDX-M100).

The accumulated program time can be displayed when the program is played back.

Digital output (output (pin jack).

Functions useful for tape editing such as pause programs and direct programs are enriched.
