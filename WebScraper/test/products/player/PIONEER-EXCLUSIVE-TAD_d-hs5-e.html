  • Pioneer D-HS5

  • ¥ 99,000 (released in late October 1996)

Commentary

DAT deck equipped with Hi-bit and Legato link conversion S.

The sampling frequency has been raised to 96 kHz, twice that of the conventional one, and it is equipped with high sampling (HS) mode which enables wide-band recording and playback of up to 44 kHz.
The rotation speed of the tape-speed drum has been increased to twice that of the conventional one, eliminating unnecessary ringing.

In order to achieve more faithful reproduction of the original sound, it is equipped with Hi-bit technology that focuses on minute level signals less than the number of quantization bits.
By re-quantizing 16-bit DAT data to 24-bit data, the dynamic range is theoretically expanded and quantization distortion is proposed to reproduce smooth waveforms.

It is equipped with Legato link conversion S which can reproduce the component over 20 khz even in the standard mode.
Based on the 1/f attenuation characteristics, the waveform of the original signal is estimated and reproduced for the component of 20 kHz or higher. In addition, an independent harmonic generation circuit is added to reproduce the overtones, and the reproduction band is expanded by adding them together with the time axis.

Equipped with DAC mode which can only use D/A converter function with Hi-bit Legato Link Conversion S.

An advanced 1-bit D/A converter is used for the D/A converter.

It is equipped with a character pack function that can record information such as song name, performer name and source title up to 60 characters in the start ID part of each song.
The recorded information is scrolled on the display at the start of each song.

It is equipped with a one day timer recording function. By inputting the current time, start time and end time, it is possible to record without an external timer.

It is equipped with a CD-Q code synchronized start ID recording function. When recording digital input from a CD, the start ID is automatically recorded, and even if there is no silence between songs, the start ID is recorded reliably, eliminating the recording error of the start ID.

Equipped with a TOC recording / playback function, the number of recorded songs and time data of each song are recorded at the beginning of the tape, and the management, operation and operation of recorded information like CD are possible.

Equipped with the last memory function, the settings of Auto ID, DAC mode and input switching are memorized when the power is turned off.

Equipped with 96 kHz High Sampling (HS) mode, 48 kHz Standard (SP) mode, 32 kHz Long Duration (LP) mode and 44.1 kHz Standard (SP) analog recording mode at the same frequency as CD.

The head uses a high-output AT head, and the signal reading ability is improved by adopting an all-track structure, improving abrasion resistance by glass welding, and ensuring stable tape touch by special polishing processing of the gap.

It is equipped with anti-vibration and anti-vibration measures such as panel stabilizer with protective rubber, honeycomb chassis, large insulator, bonnet anti-vibration structure, etc.

Wireless remote control is included.
