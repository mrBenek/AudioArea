  • Technics SL-35

  • ¥ 39,800 (around 1974)

Commentary

A full auto player equipped with an automatic repeat mechanism (memo repeat) unique to Technics.

Memo Repeat is a mechanism to repeat the set number of times by setting the memory repeat dial to the desired number of times of repeated playback (1-5 times, R (infinite)) and pulling the control lever to the start side.

High-sensitivity S-shaped pipe arm is used for the tone arm.

Equipped with MM type cartridge that can reproduce CD-4 only by changing needle tip.

The legs have built-in audio insulators.

Uses a low-capacity code that also supports CD-4.
