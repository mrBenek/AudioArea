  • TASCAM DA-30

  • ¥ 198,000 (around 1990)

Commentary

A DAT deck developed for recording masters and equipment business.

In addition to the long recording / playback function, it is equipped with AES/EBU digital interface for professional use and control I/O which can be operated by program using external computer, etc.

The sampling frequency is 48kHz/44.1 kHz (for recording and playback) and 32 kHz (digital input only for recording).

Analog balanced input / output terminals are mounted.

Equipped with left and right independent A/D and D/A circuits.

It can be mounted in an EIA-standard 19-inch rack.

A wired remote control unit is included.
