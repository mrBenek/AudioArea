  • Technics SL-3

  • ¥ 34,800 (around 1982)

Commentary

A linear tracking player with LP jacket size compact body and Technics player technology.

The arm uses a linear tracking arm that linearly moves the record board.
By means of an optical sensor arm deflection angle detection mechanism near the needle tip and microcomputer control, the tracking error angle is within ± 0.1 ° and the inside force is almost 0, achieving excellent running performance.
In addition, the arm bearing structure is a 4-point support gimbal suspension system with a pivot and ball bearing, and a dynamic balance system in which the center of gravity is aligned with the center of the arm rotation axis, realizing stable trace running without precise horizontal adjustment or needle pressure adjustment.

The cartridge uses the plug-in connector type MM cartridge, which is also used for the SL-6. In addition to the ease of installation, the accuracy of installation is improved and performance matching between the cartridge and the arm is achieved.

DC servo belt drive system is adopted for the drive part.

It is an electronically controlled full-auto type with all operations concentrated into 4 buttons. All operations can be performed with the upper cabinet closed. The rotation speed is automatically set according to the record size just by pressing the start button.
In addition, the arm can be controlled to move forward or backward, and it can be played repeatedly with the repeat button.

After the performance, the arm returns automatically.
