Commentary

This is a cassette deck that uses a metal compatible system and a plunger mechanism of 2-motor IC logic circuit.
It uses a plunger mechanism with IC and 2-motor configuration and realizes click-touch operability.

The mechanism section is equipped with a dual motor drive system.
This is a two motor system in which the capstan and the take-up reel are each driven by a dedicated motor. Compared to a single-motor system, this system works with more leeway and improves accuracy and durability.

The head uses a DX head developed assuming metal tape use.
This head consists of a Sendust guard and a magnetic pole with a special structure with a core laminated with a hard composite material. It realizes excellent electromagnetic conversion and abrasion resistance. In addition, there is allowance in the maximum magnetic flux density, which is a point when using metal tape, and it does not saturate even with the large bias current required by metal tape.
The linearity of the recording amplifier is superior to that of metal tape, and it is compatible with the dynamic range of metal tape.

Equipped with a pointer-type VU meter and a 3-point LED peak level indicator.
The indicator shows the maximum peak level value for each LH, CrO2, and metal tape on the scale of + 4, + 6, and + 10. The level can be set according to the tape.

If the tape selector is mounted and the knob is set to the position indicated by LH, CrO2 and metal, the bias value and the equalizer value will be set simultaneously.

Equipped with an auto-repeat mechanism. When the switch is set to the repeat side, when the tape is finished, it is automatically rewound and automatically played from the head of the tape.
Also, from the recording state, it will work when the tape ends, so it will move to playback immediately.

Equipped with timer standby mechanism.

It is equipped with Dolby system linked with Rec Mute and MPX filter.

By connecting a separately sold remote control unit, it can be operated from a remote location.
