  • AIWA AD-5310

  • ¥ 55,800 (around 1976)

Commentary

Stereo cassette deck with white oak style cabinet.

Equipped with a timer standby mechanism. By connecting a timer sold separately and setting the deck to a recording standby state, it is possible to record your absence. In addition, it is also possible to play an alarm clock by using it for regenerating condition.
In the standby state, the pinch roller and capstan are separated from each other to prevent deformation of the pinch roller.

By using a flywheel with large inertia energy and a DC servo motor with a built-in frequency generator, tape running is stabilized.

Equipped with an LED peak indicator, the distortion level can be indicated even for a pulsed signal that the VU meter can follow. The lighting level is set to + 7 dB.

The Dolby System is installed as a noise reduction system.
The AD 5310 is designed to turn off the Round Flip Filter when the Dolby is off.

The tape selector uses a 3-step switching system with independent bias and equalizer.

The head uses an FG head that uses ultra-hard ferrite with excellent wear resistance for both the core and guard parts.

It uses a quick review queue system that is convenient for tracking tapes.

An automatic line-input / microphone input switching mechanism is used to improve the SN ratio during line recording.

It uses a full auto-stop mechanism that stops the mechanism as soon as the tape ends, regardless of the state of recording, playback, rewind, or fast forward.

Equipped with tape run which can confirm tape running and Rec and Dolby NR display lamps which prevent operation error.

Comes with an acrylic dust cover.
