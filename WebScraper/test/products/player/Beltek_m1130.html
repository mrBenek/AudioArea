  • BELTEK M1130

  • ¥ 42,800 (around the early 1970 s)

Commentary

Stereo cassette deck equipped with autoloading mechanism.

An autoloading mechanism using unique technology is adopted for the mechanism.
In this mechanism, when a cassette is inserted, it automatically lowers to set the tape on the head, and when the eject button is pressed to eject the cassette, it pops out.

Ferrite head is used for the head.

The amplifier section uses a low-noise circuit using PNP transistors.

Equipped with a tape selector, it can be used with a bias / equalizer suitable for chrome, LH, and regular tapes.

It is equipped with a switch for switching between line input and microphone input during recording to improve the dynamic range and SN ratio.

By using an electromagnetic plunger, microswitch, relay, etc. for the operation button, smooth switching of operation is realized.

Equipped with an automatic stopping device, the head and pinch roller are automatically separated from the tape when the tape is finished winding, preventing excessive load on the drive mechanism and motor.

Equipped with a pose lock mechanism that allows instantaneous movement during recording.

Equipped with a memory counter.

Equipped with left and right separate type level meters.

Uses double volume knob for easy adjustment of left and right channels.

Remove the escalator of the cassette insertion part and press the illumination lamp switch to illuminate the head part and you can clean the head.

Equipped with an illuminated tape run display that shows the running state of the tape.
