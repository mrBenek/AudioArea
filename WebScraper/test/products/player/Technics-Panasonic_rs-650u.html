  • Technics RS-650U

  • ¥ 75,800 (around 1976)

Commentary

Cassette deck with horizontal drive 2-motor system.

The mechanism uses a 2-motor system in which the constant speed running motor and the fast-winding motor are separated.
An electronically controlled DC motor is also used for constant speed running.

LH head is adopted for the head.

Equipped with Dolby NR circuit in which MPX filter operates by switch IN.

The meter can perform peak check operation by switching the switch.

Equipped with a 2-step switching type tape selector with independent bias and equalizer.

Two types of muting circuits prevent noise generation during power ON/OFF, recording / reswitching, and STOP.

An electronic control system is used to prevent sparks and noise due to poor contact by using transistors instead of relays as switching elements.

Built-in memory rewind function.

Microphone input and line input can be mixed.

Recording of absence is possible by combining with an optional timer.

There was a wood case sold separately.
