Commentary

An auto reverse cassette deck equipped with a Dolby system.

It uses the Six in One Head, which integrates recording, playback, and erasing heads for both forward and reverse directions.

Equipped with an auto-reverse mechanism that employs newly developed dual capstan and mode select switch.
With the mode select switch, three types of this mechanism can be selected : Non-Reverse, which stops automatically at the end of one side winding, Reverse, which stops Oatley at the end of one side winding, and Repeat, which repeats Oatley at the end of one side winding.

In order to solve the hissing of cassette tapes, a Dolby noise reduction system is installed.
In the フォルテッシモ of music, hissing that is masked and unnoticeable becomes annoying when the program is pianissimo because the level of music and the level of hissing become close to each other. Therefore, the Dolby Noise Reduction System automatically raises the recording level when the program is pianissimo and lowers the recording level when the program is played back to create a level difference from the apparent hissing.
This reduces tape hiss and makes it possible to play back the musical quality of the program source more richly.

The mechanism uses an outer rotor type hysteresis synchronous motor with smooth rotation and less vibration.
In addition, we have adopted a polyurethane-based belt with less aging and reliable transmission, and a large double flywheel for the wheel.

Equipped with a 3-position tape selector, it can demonstrate the performance of general tape / Hi-Fi tape / Chrome tape.

It uses an independent stable power supply circuit that prevents bias fluctuations and provides stable performance.

The preamplifier uses a low-noise silicon transistor and a bipolar linear IC, both of which were developed specifically for audio applications.

Equipped with a full mechanical auto shut-off mechanism that returns the operation button when the tape is wound in any operation state.

Equipped with separate level volumes for recording and playback.

It has a headphone jack.

Equipped with a tape counter.

The level meter has a built-in lighting lamp.
