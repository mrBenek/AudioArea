  • ¥ 129,500 (released in 1988)

Commentary

The DAT deck is composed of almost the same structure as DTC-500ES and condensed into a minicomponent size.

Both A/D and D/A converters are equipped with a 2 fs type converter and a digital filter that reduce the load on the low-pass filter by doubling the sampling frequency. Both converters are integrated into a newly developed LSI, and the digital filter uses an A/D/A method that uses the same circuit for both A/D D/A conversions.
Since recording and playback can be used in the same circuit, phase correction and time axis correction are made easy.

Equipped with a digital peak margin display function, it is possible to display in digital units of dB how much memory level is available when recording.
The data used as the basis of this display is derived from the data of the digital processing circuit together with the data used for the level meter display to obtain accurate values.

With the subcode function, you can set the absolute time function, time search function, start ID and skip ID, rehearsal function, program number, renumber, erase function, RMS play by remote control, repeat function, etc.

Uses 2D. D. + 1 bsl motor mechanism.

It is equipped with two digital input terminals and optical digital output terminals which are coaxial with the optical system.

In addition to the absolute time display by subcode, it is equipped with a linear counter that can be switched to 4 modes.

-60dB ~ 0 dB Equipped with digital peak meter of each ch24 segment.

It is equipped with a cue & review function at two speeds, approximately 2 x speed and approximately 16 x speed.

Equipped with a blank search function that automatically searches for unrecorded parts that have never been recorded.

Equipped with a memory play function that rewind and play back to the counter 0 : 00 position and an autoplay function that plays back immediately after rewind.

When combined with the separately sold timer, it is possible to record your absence and play the alarm clock.

Equipped with a headphone port with variable output.

The side wood is removable.

Wireless remote control is included.
