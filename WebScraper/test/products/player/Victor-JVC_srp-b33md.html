  • VICTOR/JVC SRP-B33MD

  • ¥ 36,700 (around 1972?)

Commentary

A four channel record player developed for CD-4 (4 ch) playback.
It is a compatible type that can play back conventional records.

The cartridge is 4MD-1X for CD-4.

The tone arm uses a stay-balance type TH type tone arm.

Uses low-capacity special shield wire.

Equipped with push button type speed switching, overhang indication indicator, shell stand, special rubber isolator, accessory space, etc.
