  • ¥ 38,000 (released in 1996)

Commentary

CD player using PRO-BIT system.

Equipped with Yamaha's original PRO-BIT (Precise Reproduction of Original Bit) system.
PRO-BIT converts 16-bit data read from a CD into 20-bit data by calculating the correction value for quantization error based on the time series change of the data. Specifically, approximately 20000 patterns of Yamaha's 20-bit recorded data and 16-bit data obtained by truncating the lower four bits of the data are stored in an LSI and compared with 16-bit data read from a CD to calculate the correction value for 20-bit data. This greatly reduces the conversion error at the minute level.

The D/A converter is equipped with Yamaha's original I-PDM DAC.
In this method, each waveform pulse is handled independently, and even if the output pulse waveform is not ideal, distortion does not occur in principle, and high-purity analog signals can be obtained.

Uses LiD power supply with excellent low-impedance capability.

The servo circuit uses an intelligent digital servo to trace in an optimal state according to the state of the disk set in the tray.

It is equipped with 2 types of digital output which are coaxial with optical output.

It is equipped with a tape program function that can specify the order of songs when dubbing to tape or MD, and an autospace function that automatically inserts space between songs.
It is also equipped with a synchronous REC function and can be operated in conjunction with a Yamaha cassette deck.

Variable line-out terminal is mounted.

Equipped with 3-stage dimmer control, you can adjust the brightness of the display.

Wireless remote control is included.
