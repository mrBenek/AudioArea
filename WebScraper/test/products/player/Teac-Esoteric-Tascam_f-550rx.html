  • TEAC f-550RX

  • ¥ 125,000 (around 1979)

Commentary

Metal tape compatible cassette deck equipped with dbx system.

This is the first installation of the dbx system as a noise reduction system.
The dbx system compresses the dynamic range of the input signal to 1/2 in decibels for recording, and expands it twice during playback to ensure a high dynamic range within the limited dynamic range of the tape deck.
In addition to dbx, Dolby systems are also installed.

The specifications are compatible with metal tapes, and also compatible with high dynamic range program sources with many high frequency components.

Equipped with a peak level meter.
By electrically recording the peaks of the music signal and moving the pointer at high speed, an accurate set of recording levels is possible.

The input volume is independent on the left and right, making it easy to set recording levels.
In addition, it is a differential type that is precisely linked when one side is moved.

An FG servo DC motor is used as the drive motor for the mechanism.
This motor uses a built-in frequency generator (FG) to shut out changes in motor rotation caused by voltage and frequency fluctuations of the power supply.
In addition, high-precision capstan shafts, large flywheels, and precision polishing flat belt improve the transmission system and stabilize the tape speed.

A SENDUST head has been adopted for the recording / reproducing head.

The operation part uses soft touch operation with the unique button arrangement of the teac, and direct change is possible from any mode.

Equipped with the REC MUTE function.
If you press the REC MUTE switch during recording, the recording mode will change from that point to silent recording. If you press the Play button, the recording mode will return to the original recording mode. If you press the Pause button, the recording mode will become standby and wait for the next recording to resume.

Equipped with a timer mechanism, you can record messages or play an alarm clock by combining a commercially available audio timer and setting the timer mode to REC or PLAY.

Equipped with a memory stop function.
Set the counter to 000 to start the tape and press the rewind switch to automatically stop at 999.
You can check the repeated playback and recording of songs.

Equipped with line / microphone input selection switch.
This switch allows the line input to be sent directly to the recording amplifier without passing through the microphone amplifier, thus preventing distortion and noise when passing through the microphone amplifier.

There was a wired remote control RC-90 as an option sold separately.
By using RC-90, the same micro switch operation as the main body can be done at hand.

There was a rack handle as an option sold separately.
