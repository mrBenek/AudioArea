  • SONY CDP-701ES

  • ¥ 260,000 (around 1983)

Commentary

Sony's second generation CD player system.
From this generation, ES series of CD players were released.

Each function is controlled by five types of microcomputer.

It is equipped with an RMS function that allows direct selection of favorite music and random memory performance.

It is equipped with a location function that can call up the middle of a song freely by index No and time.

Equipped with an AMS function that allows you to start your head.

Equipped with a repeat function which can be selected in 3 patterns of all songs / one song / part.

Equipped with a music scan function that can skip and play only the intro part of a song and is convenient for searching for songs.

Equipped with a 2-speed music search function that allows you to listen to the contents of a song while fast forwarding.
