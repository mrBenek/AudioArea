Commentary

A horn / lens with a huge slant plate.

It consists of an exponential horn with an elliptical aperture and a large slant plate with a width of 91.4 cm.

It boasts excellent directional characteristics and is designed to be ideal for short distance PA.
When in use, install it on top of the enclosure using the supplied installation metal fitting 2505.

A 5 cm diameter screwdriver is usually used, but a 2.5 cm diameter screwdriver can also be used by using an optional 2327 adapter.
