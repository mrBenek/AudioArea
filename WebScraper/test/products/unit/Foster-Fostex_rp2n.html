Commentary

2-way coaxial type in-car 16 cm speaker.

A 16 cmCGM cone type woofer is mounted for the low band, and an RP type full drive type tweeter is mounted for the high band.

The tweeter has a directional variable mechanism so that the direction of the high frequency range can be freely set to the listener position.

Uses a compact design that allows installation of a front door.
