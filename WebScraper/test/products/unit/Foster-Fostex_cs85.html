  • FOSTEX CS85

  • ¥ 9,000 (1 pair, about 1992)

Commentary

A 8.5 cm cone type full-range speaker unit developed for in-vehicle use.

Uses an injection molding type cone.

It is a thin design considering the setting for the door. In addition, a special grill is attached.

It is also possible to add a CRT6RP for a two way system.
