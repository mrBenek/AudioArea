Commentary

Horn type tweeter unit.

It uses Coral's unique ultra-thin super duralumin diaphragm, ultra-precision heavyweight aluminum-alloy horn, casting magnet with less magnetic flux leakage, etc.

It can be fixed to the baffle by using the mounting flange AD-3 (sold separately).
