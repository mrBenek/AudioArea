Commentary

Large input type 2-way network.

The Altec network gives careful consideration to phase matching near the crossover joint, and smooth response can be obtained.
In addition, we have adopted a winding with a core that can be designed with less DC resistance, and we have retained excellent characteristics together with a precision grade attenuator.

It is a 2-section LC type with a large input type (250W) core volume and thick windings.

The high-frequency attenuator has an OFF position for 10-step switching in 2 dB steps.
