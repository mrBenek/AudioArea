Commentary

38 cm coaxial type unit combining double cone type speaker SP15B and horn type tweeter T35.

Built-in special high-pass filter and level control.

It was 16 Ω specification around 1970 and 8 Ω specification around 1972.

Mid-range kit BB4 (sold separately) can be used.
When using BB4 on the 15TRXB, it is necessary to use the crossover network X36 together.
