Commentary

A speaker system with the same configuration as A7-500-8.

A7 500 WII is a system released at the same time as A7 8 Ω, and the unit configuration has been changed to A7 500 w.
Also, there were 2 types of cabinets, Altec original and designated domestic cabinet.

The unit uses 38 cm cone type woofer 416-8A for the low range, and the horn type unit which combines driver unit 808-8A and horn 511B for the high range is mounted.
N501-8A is used for the network.

The cabinet is hand finished with a walnut and has a wooden grill on the front.
