  • SONY SS-555

  • ¥ 27,000 (a set of two cars, around 1979)

Commentary

This is a bookshelf speaker system that uses the bass reflex method.

A 16 cm cone type woofer using CARBOCON is mounted in the low range.

The high-frequency range is equipped with a 3 cm balanced drive type tweeter using Sony's original titanium.

It is equipped with a continuously variable level control for high frequencies.
