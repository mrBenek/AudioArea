Commentary

This is a tall boy type speaker system developed in response to demands for AV systems.

Two 18 cm cone type woofers are used for the low range.
It uses a vertical twin system with a double woofer and realizes a rich sound field and excellent sound image localization.

A 2.5 cm dome type tweeter with an alnico magnet is used for the high range.

The enclosure uses an elliptic curve round baffle to reduce diffraction phenomena.
It also has an input terminal that can be used for bi-wiring connection.
