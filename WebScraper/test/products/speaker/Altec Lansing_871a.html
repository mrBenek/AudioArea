Commentary

A speaker system with the same configuration as the 846A Valencia and 848A Flamenco.

The unit configuration is the same as A7-8.
A 38 cm cone type woofer 416-8A is installed in the low range, a horn type unit using a driver 807-8A and a horn 811B is installed in the high range, and N801-8A is used for the network.

The cabinet uses a bass reflex system.
The exterior is a luxurious furniture style with beautiful hickory material. The 871A was designed to be used with the front door open, and the front door opens to the right on the 871 ar and to the left on the 871 al.

The 885 (Milano) matching center cabinet, the (Milano) was sold separately and was designed to be combined to create a stereo ensemble system. 871
The 885 is finished with hickory material like the 871A, and has space for storing stereo components such as tuner, amplifier, record player, tape recorder, etc.

* 871A Milano was listed as 1971 release in stereo sound, but it was listed in the catalog in the latter half of 1970.
In addition, although the unit configuration was described in the above specification, there seemed to be another configuration such as using 416 type (416-Z?) woofer.
