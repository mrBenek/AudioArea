Commentary

A carefully tailored speaker system for custom production.

The low-range is equipped with an edgeless 30 cm cone woofer SLE30W.
This unit uses an SLE system that combines technologies such as an acoustic low-pass filter and a push-pull type double damper and realizes low fo, low 歪巻 sound quality and high input resistance.

Dome type Skoka FS70D is mounted in the middle area.
This unit is designed for use with edgeless and high-compliance woofers.

A ribbon-type tweeter FT3RP using the RP method is mounted in the high-frequency band.
The RP (Regular Phase) system is a system using a planar coil diaphragm. It is a full-surface drive system in which the entire diaphragm moves in the same phase. The basic structure is such that an extremely thin high-density planar coil diaphragm is held by a special support system between a pair of magnetic circuits with the same poles facing each other, so that the entire diaphragm is driven uniformly. Since the mass of the vibration system is small and it is driven uniformly, it acquires strength and wide dynamic range while maintaining the sound quality of condenser type.

Level control is provided for the middle and high frequencies.

The enclosure is sealed and has a rosewood finish on the exterior.
