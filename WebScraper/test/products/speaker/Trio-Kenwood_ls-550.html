  • KENWOOD LS-550

  • ¥ 55,000 (a set of two cars, around 1986)

Commentary

This speaker system uses PP cone woofer with enhanced rigidity.

It uses a 22 cm cone type woofer for the low band, a 5 cm semi-dome type squaker for the middle band, and a 2.5 cm dome type tweeter for the high band.
In addition, it has a magnetic shielding design so that it does not have any adverse effect even if it is placed near the television.

The enclosure is a bass reflex type and has a symmetrical design.

Equipped with level control.
