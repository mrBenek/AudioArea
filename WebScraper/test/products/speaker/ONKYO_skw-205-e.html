  • ONKYO SKW-205

  • ¥ 20,000 (1 unit, sold on June 23, 2000)

Commentary

A slim size subwoofer system with a built-in amplifier.

The unit uses a 20 cm cone type woofer.
The enclosure uses MDF.

In addition to the RCA pin input terminal, there is also an input / output terminal for speaker cable. An amplifier without a subwoofer output terminal can be used by connecting the SKW 205 between the speaker and the amplifier.

Equipped with a variable high-cut filter, auto standby ON/OFF function, music / movie mode switching function, and phase switching function.
