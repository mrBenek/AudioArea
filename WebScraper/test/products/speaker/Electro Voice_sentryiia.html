  • Electro-Voice SENTRY IIA

  • ¥ 126,000 (1 unit, around 1972)

Commentary

A monitor speaker system developed for professionals.

A specially developed 30 cm cone type woofer is installed in the low range.
A horn type tweeter using a VHF driver is installed in the high-frequency range.

The enclosure uses a bass reflex system.
The exterior has a four sided walnut finish so that it can be used vertically or horizontally, and the front has the same grill as the Patrician 800.

* Installation method described in the catalog

In a rectangular room, the low-frequency characteristics can be improved by placing it near the short side wall, and it can be extended to listening positions. For better stereo effects, choose an isosceles triangle between the listening point and both speakers with a vertical angle of 30 to 40 degrees.
In principle, the SENTRY IIA can be used horizontally, but it can also be used vertically, which is advantageous in terms of high-frequency directivity.
