Commentary

A speaker system that uses a dome unit.

A 30 cm cone type woofer is installed in the low range.
This unit uses soft urethane foam for the edge and has a high compliance design with a convex shape to improve linearity. In addition, a large 140 φ magnet is used for the magnetic circuit.

A 15 cm dome type skoker is installed in the middle area.
This unit was designed for the Dome Series (SB 500, SB 700). The diaphragm is made of corrosion-resistant aluminum alloy with high rigidity and light mass.
In addition, a 10-cm dome type tweeter using an aluminum alloy diaphragm is mounted in the high range as in the Skoker.

The mid-range and high-range are equipped with attenuators.
This attenuator is designed to be OFF when used in a multi-amplifier so that it can be used in a state of 0 insertion impedance.

The enclosure is sealed and the exterior is oil-finished with Oceania cheeks.

The input terminal on the back can be used in 2-way or 3-way multi-amplifier system depending on the switch.
