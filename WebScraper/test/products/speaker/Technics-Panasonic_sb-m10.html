Commentary

Compact 2-way speaker system.

A 14 cm cone type woofer is installed in the low range.
This unit uses a dynamic linear suspension to improve power linearity with excellent variable characteristics.

A 2.5 cm dome type tweeter is installed in the high range.

The enclosure uses a bass-reflex system using a 21 mm thick wooden cabinet.

It is equipped with an input terminal that can connect extremely thick cords and 4 mm plugs.

Anti-magnetic design is adopted.
