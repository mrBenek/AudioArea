Commentary

This is a monitor speaker system refined from the 612A system that has been used for many years.

The unit is equipped with 38 cm coaxial type unit.
The 612C Monitor uses different units depending on the age. Initially we used 604-8G, in 1979 we used 604-8H, in 1981 we used 604-8KS.

The enclosure 612C for the enclosure.
In 612C, the bass reflex port is precisely adjusted to improve the low-frequency characteristics.
