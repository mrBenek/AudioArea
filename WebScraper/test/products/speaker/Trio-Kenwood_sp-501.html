Audio Database

  • KENWOOD SP-501

  • ¥ 25,000 (one set of five, around 1997)

Commentary

Surround sound set with 5 small speakers for surround sound reproduction.

Each speaker unit is a 10 cm full-range cone unit.

The enclosure is a bass-reflex type and is compact in design so that it can be installed in any place.

Only 3 units for front and center are designed for magnetic resistance.
