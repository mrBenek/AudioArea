  • FOSTEX G750

  • ¥ 39,800 (a set of two cars, around 1998)

Commentary

This is a speaker system in which FOSTEC's speaker technology is condensed into an aluminum alloy body that has been miniaturized to the limit.

A 7 cm cone woofer is installed in the low range.
This unit was developed exclusively for the G750 and has an effective vibration radius of only 2.7 cm. However, it is designed to extend the low-sound reproduction band as much as possible.
The vibration system uses a push-pull double damper to make the stroke linear enough. In addition, the magnetic circuit uses an Alnico magnet equipped with an Alnico magnet, which is made to the maximum size for a 7 cm size, as an inner magnetic type.

A 1.6 cm soft dome type tweeter is installed in the high range.
The magnetic circuit is an internal magnetic type using an alnico magnet.

The enclosure has a four sided structure with extruded aluminum alloy, ensuring the required weight despite being small.
