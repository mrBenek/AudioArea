  • Technics SB-F6

  • ¥ 13,000 (1 unit, released in 1982)

Commentary

A 3-way speaker system with 100W input resistance.

It uses a 16 cm cone type woofer for the low range, a 5.5 cm cone type for the middle range, and a horn type tweeter using ceramic for the high range.

Linear phase design is adopted to improve sound image localization.

Uses TNRC with excellent resonance resistance.
