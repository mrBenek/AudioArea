Commentary

A speaker system using the same unit as the 846A Valencia.

There are 2 types of 848A, the former and the latter, and the model was changed in 1970.

The previous term has the same configuration as A7 and 16 Ω specification.
A 38 cm cone type woofer 416A is mounted in the low range, and a horn type unit using a driver 806A and a horn 811B is mounted in the high range. The network uses N-800G, which is different from the configuration of N-800D of A7.

The latter half has the same configuration as A7-8 and has been changed to 8 Ω specification.
A 38 cm cone type woofer 416-8A is installed in the low range, a horn type unit using a driver 807-8A and a horn 811B is installed in the high range, and N801-8A is used for the network.

The cabinet uses 860A with a bass reflex system.
The exterior design uses a classical Spanish style gridiron grill with an iron door pattern.

For the 848A, the 882B (FlamencoII type) was sold separately as the best matching center cabinet, and it was designed so that the stereo ensemble system could be constructed by combining it.
The 882B has a thick finish made of dedicated oak and has space for stereo components such as tuner, amplifier, record player, tape recorder, etc.
