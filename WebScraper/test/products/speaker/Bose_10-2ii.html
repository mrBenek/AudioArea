Commentary

This is a speaker system that expands the stereo listening area by using the stereo targeting array structure and the Kusu timmass method.

Bose's unique stereo targeting array is used.
In this technology, multiple driver units are installed close to each other, and the desired radiation pattern is controlled by arranging the interval, phase, angle, unit size, etc., so that correct stereo images can be reproduced over a wide range of listening rooms.

Bose's unique アクースティマス Base Technology is used to enhance low-frequency playback.
In this method, only the linear region of the woofer can be used structurally, so the distortion itself is minimized. In addition, the acoustic filter filters the harmonics generated from the cone, so high-quality bass reproduction is possible over a wide band.
The 10.2 ii has a 20 cm cone woofer inside the enclosure for low-frequency use.

The mid-range has a 20 cm cone mid-range, and the high-range has two 5.1 cm cone tweeters.
Magnetic fluid is used in this tweeter to improve input resistance.

A protective circuit is provided to protect the speakers from excessive input.
When an over-input is applied to the speaker, part of the circuit is opened and a resistor is inserted into the woofer circuit to reduce the peak of the signal to a safe level. This resistor is usually bypassed so that it does not affect the signal.
When the input signal level drops, the circuit automatically returns.
