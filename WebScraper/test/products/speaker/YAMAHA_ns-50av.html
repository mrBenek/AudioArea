  • YAMAHA NS-50AV

  • ¥ 37,000 (1 unit, released in 1991)

Commentary

A speaker system developed as a compact, high-performance, multi-channel standard monitor for AV multi-channel Purpose playback.

A 16 cm cone type woofer is used for the low range.
This unit uses the curved type seat cone used in the NS-10M and other models with an emphasis on the mid-range, improving the vocal band reproduction capability.

A 2.5 cm soft dome type tweeter is used for the high range.

It uses the right size design with the sub-woofer YST-SW50, and it matches like an integrated system when placed on the top plate.
Vibration isolation sheet of 2 mm thickness is attached so that vibration does not affect each other in systematization.
