  • YAMAHA NS-W1

  • ¥ 34,800 (1 unit, around 1982)

Commentary

Yamaha's first active subwoofer with a built-in amplifier.

The NS-W1 comes with a dedicated mixing adapter that can be easily added to your system by connecting speakers through the adapter.
This mixing adapter has an output terminal for the main speaker, so there is no problem if the main amplifier does not have speaker outputs A + B.

The unit is equipped with 25 cm cone type woofer.
This unit is mounted downward and is non-directional and emits sound pressure in all directions. Therefore, it can be installed in any place.

Built-in functions such as volume, filter, and high-frequency selector switch make it easy to match with the main speaker.
