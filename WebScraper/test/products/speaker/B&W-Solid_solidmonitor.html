  • SOLID SOLID Monitor

  • ¥ 48,000 (a set of two cars, around 1991)

Commentary

This is the first speaker system of Rocksolid Sound.

It is anti-magnetic design and waterproof for daily life, and can be set in places where conventional speakers could not be installed.

A 10 cm cone type woofer using fiber cone is mounted in the low range.
In addition, a polycarbonate cone tweeter is mounted in the high frequency range.

The enclosure uses a bass reflex system and the exterior comes in six color variations.

Die-cast is used for the stand part, and it is integrated with a unique fleece top type ball joint. This allows setting in various situations without adding parts sold separately.

The optional SOLID Array allows two sets of SOLID Monitors to be set in all directions.
