  • Marantz LS-60GX

  • ¥ 55,200 (a set of two cars, around 1989)

Commentary

This speaker system pursues rigidity of the entire system.

For the lower range, it uses an anti-magnetic 23 cm cone type unit that uses a bonded carbon fiber cloth. The frame uses an aluminum die-cast frame and is firmly fixed with an 8-point screw stopper.
In addition, a high heat-resistant large diameter long voice coil is driven by a double magnet type magnetic circuit.
For the high-frequency range, it uses a damperless polyetherimide 3 cm semi-dome unit, and for the ultra-high-frequency range, it uses a unimorph element 1.5 cm dome unit.

The enclosure uses a 20 mm mmMDF high-density board for the front baffle and a 25 mm high-density chipboard for the side and back plates. The enclosure is reinforced for higher rigidity.
