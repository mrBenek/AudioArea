  • KENWOOD DX-7

  • ¥ 77,000 (around 1991)

Commentary

Portable DAT recorder compatible with SCMS system.

The mechanism part is equipped with an ultra-small mechanism using a small drum with a diameter of 20 mm.
In addition, miniaturization and weight reduction have been achieved by adopting high-density double-sided mounting technology using a thin four layer substrate.

Coaxial digital input / output terminal is mounted.
A compact digital dubbing system can be constructed by combining the DPC-90 portable CD player.

The display section is equipped with a multi-function display that displays the track number, total elapsed time, recording level when recording from an analog signal, play / pause / repeat marks, REC/ST-ID display, and other information.

The body is made of aluminum or heat-resistant ABS resin, making it safe to use in a car.

Equipped with subcode function, auto start ID, ID dubbing, renumber, end ID, etc. can be set.
When recording from CDs or satellite broadcasting, Auto Start ID detects no signal and automatically writes the song number. The ID dubbing function allows you to manually write the ID for accurate cue.
The Remember function automatically sorts the music numbers even if you record from the middle. With the end ID function, you can write the end ID at the end of the recording section, so that you can handle the case where there is an extra tape.

It supports timer recording / playback. By connecting a timer sold separately and switching the timer switch, it is possible to record the absence and play the alarm clock.

Equipped with a key-hold switch, switching the switch disables button operation and prevents malfunction.

Multi-functional wired remote control is included.
With this remote control, you can manipulate mechanisms such as play / repeat, rec, pose, record hold, cue, cue & review, volume adjustment, and record and erase subcodes.
Also, this remote control has a headphone port.

By combining with the separately sold outdoor kit BP-A7, it is possible to play continuously for 100 minutes outdoors.

Enjoy in the car by combining the separately sold car power adapter DC-C5 and the car cassette adapter CAC-1.
