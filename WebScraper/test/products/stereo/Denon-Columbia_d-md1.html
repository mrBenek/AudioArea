Audio Database

  • DENON D-MD1

  • ¥ 110,000 (around 1996)

Commentary

System component with 3-disc CD changer and MD deck.

You can digitally record up to 32 songs with a 3-disc CD changer.
It also has a track edit function that is convenient for editing songs from a single CD.

Equipped with a sampling rate converter.

Equipped with a multi-jog dial that can be used for title input and MD editing functions.

It supports 2 times longer monaural recording mode and can record up to 148 minutes.

Uses an electromagnetic auto reverse cassette deck.

Uses an FM wide / AM stereo synthesizer tuner.

A two way speaker system using the bass reflex method is used for the speaker section.
