  • SANSUI MH9

  • ¥ 98,000 (around 1997)

Commentary

CD jacket size micro high component.

The MH9 consists of CD-9MH for the CD receiver, MD-9MH for the MD deck, and S-9MH for the speaker. In addition, the cassette deck D-9MH and the dedicated rack GX-9MH were sold separately as options.

* MH9 seems to be the same model as Victor Micro High Component UX-MD1.

CD Receiver CD-9MH
The amplifier has a maximum practical output of 20W + 20W.
It is also equipped with a superbus function and an electronic tone control that can be operated by remote control.

A 1-bit DAC is installed in the D/A conversion section of the CD player.
It is also equipped with functions such as a 15-track music calendar, 20-track program performance, repeat performance of one / all / program music, and random performance.

A wideband FM/AM tuner is used in the tuner section.
Equipped with area guide function and automatic tuning / manual tuning function.
Manual preset memories are also available, allowing up to 15 preset stations for each of FM and AM.

MD Deck MD-9MH
It is equipped with functions such as 20-track music calendar, 20-track program selection, 1-track / all-track / program-track repeat, random, and CD-sync recording.

Titles can be entered in Katakana, English and numeric characters.

Provides MD-specific editing features such as joins, divides, moves, erases, and all-erases.

Equipped with an auto title function.
This function allows automatic input of the radio station name when entering the area guide.

Speaker S-9MH
Two 10 cm cone type woofers are installed in the low range, and a double drive system is adopted.

A 5 cm cone type tweeter is installed in the high range.

The enclosure uses a bass reflex system and is an all-wood cabinet.

Cassette Deck D-9MH (Sold Separately)
Equipped with Oatley berth function.

Synchronous recording with CD/MD is possible.

Equipped with Dolby B NR as noise reduction.

Other
Platinum gold aluminum panels are used throughout the system.
It also uses a backlit display.

Equipped with an alarm clock / recording timer.
This function allows you to set the start and end time, source and volume. Note that timer recording of the optional D-9MH cassette deck is not available.

Wireless remote control is included.
