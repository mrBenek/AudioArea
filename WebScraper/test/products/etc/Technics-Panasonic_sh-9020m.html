  • Technics SH-9020M

  • ¥ 50,000 (around 1977)

Commentary

A peak / average meter unit developed for the purpose of reading as accurately as possible even at the recording level.

3 mode switching of peak / peak hold / average is possible, and 1 meter can be used for 3 different types according to the purpose.

The meter sensitivity can be switched between 0 dBm and 20 dBm.
An indication range from -50dB to + 10 dB is possible for each, and an indication range of 80 dB is secured by using a sensitivity change-over switch.

The input of the Route 3 can be switched with the push button.
You can read the power amp output, recording output, and preamplifier output by simply switching the buttons.
