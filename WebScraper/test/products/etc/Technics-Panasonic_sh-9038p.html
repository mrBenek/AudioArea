  • Technics SH-9038P

  • ¥ 80,000 (Released in November 1977)

Commentary

Programmable unit with built-in microcomputer that can program timer operation of 32 steps per week.

By combining the ST-9038T and SH-9038P, it is possible to automatically control FM tuning and air check on a weekly basis.

When the program mode is set to FMmemory, up to 8 SH-9038T receiving stations can be preset in the SH-9038P memory.

When the program mode is set to write position, you can store up to 32 addresses for the day of the week, time, broadcasting station, AC line, and ON/OFF program. You can control four AC outlets (SH-9038T, amplifier, deck 1 and 2) and automatically select the station.
Two of the AC lines can be turned ON/OFF independently.

When the program mode is set to the read position, the address of the memory at which the program is currently set is displayed in digital order by pressing the push button.

The set program is automatically executed in the auto position. In the manual position, you can select the channel or turn the AC line ON/OFF regardless of the program. When the programmed time comes during manual operation, the auto operation takes priority.

It uses a clock method with very low error in the power frequency synchronization method, and displays the day of the week with LED and the hours and minutes digitally.
For time adjustment, you can set the day of the week, hour and minute with separate buttons. If you set the time and set the switch to the clock position according to the time signal, you can adjust the time to the second unit.

Equipped with an add-on stopwatch function that can also measure the remaining amount of recording tapes.
With this function, the elapsed time after the start can be read in units of seconds up to 59 minutes 59 seconds. In addition, it is equipped with a hall function so that addition after the temporary stop is possible.

Die-casting cabinet is adopted.
