  • ¥ 7,500 (around 1976)

Commentary

MM cartridge with dual magnet power generation system.

It uses a dual magnet power generation system using two magnets. Since the electromagnetic path is constructed with the minimum required distance and magnetic efficiency is improved, the output voltage is also increased and stable characteristics are obtained.

Uses a special diamond needle with mirror finish.

The characteristics have been improved by using a single-end support system and a shielded power generation circuit.
