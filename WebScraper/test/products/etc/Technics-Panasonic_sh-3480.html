Commentary

A demodulator for CD-4 record playback that allows you to enjoy four completely separate sounds.

The equalizer circuit, which is important for demodulators, uses a high-performance IC to achieve low noise, high gain, wide dynamic range, and high stability with minimal fluctuation in performance.

Phono1 is equipped with 3 positions for semiconductor cartridge, Phono2 for MAG type cartridge, and Phono3 for other cartridge.

The separation adjustment required when replacing the cartridge can be adjusted to the optimum position by turning the adjustment knob on the front panel.

It is designed to supply bias current to the semiconductor cartridge connected to Phono1 when the bias switch on the back panel is turned ON.
If the MAG type cartridge is connected to the Phono1 by mistake, this bias current flows and is undesirable. To prevent such an error, we have adopted a mechanism in which the selector does not turn unless the Lock button is pressed when the Phono1 is selected.

Equipped with a COM terminal that can be shared by Phono1 and 2.
When a player is connected to this terminal, both MAG-type semiconductor cartridges can be used by simply changing the shell and the selector.

The function knob has three positions : 2 ch (for 2 ch record playback), 4 ch (for discrete 4 ch record playback), and External (for 4 ch source playback connected to 4 ch In).
If you set the 2 ch position, the same sound as the front is played from the rear.

A 4-channel volume is used, and the volume of all connected 4-channel sources can be adjusted with one volume.
In addition, it can be used as an audio muting because it uses a preset type that stops the rotation of the knob at that position once it is set.

Equipped with a Pre Out connector that can also be used as a recording connector.
If the output is taken from this jack, it functions as a 2-channel equalizer.
