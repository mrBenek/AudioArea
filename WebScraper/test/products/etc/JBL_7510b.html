  • JBL 7510B

  • ¥ 630,000 (around 1990)

Commentary

An automatic microphone mixer developed using JBL's original electronic technology.

The input can be expanded up to 24 channels by incorporating 7510 to 03 input modules (4 ch, input sold separately).

Equipped with 48 v phantom power supply function for condenser microphone.

The output has one unbalanced terminal.

Equipped with direct output terminals (unbalanced, 24 ch maximum), each input signal can be supplied to an external effect device.
An auxiliary input terminal (unbalanced, 1 ch) is also equipped to receive the pattern signal.

When the switch is in the "Auto" position, a digital lock circuit constantly compares the input common to all microphones (background noise) with a specific microphone input (signal sound), activates only the desired channels, and automatically adjusts the gain so that the output level remains constant regardless of the number of active channels.

It achieves an ultrafast rise time of 30 ns to 60 ns without affecting the rise of the PAL music signal.

Variable threshold level type.

The release time can be adjusted from 100 ms to 5s.

It can be used for multi-microphone recording as a high-speed, high-fidelity noise gate.
