Commentary

This PCM processor is equipped with the same 16-bit format as professional equipment, and the new function OVC enables long-time digital recording and playback.

In a digital recording system, an analog signal is converted (quantized) by a processor into a binary digital code composed only of 0 and 1 and recorded on tape.
In addition to the conventional 14-bit EIAJ format, the PCM-16E S is equipped with the same 16-bit format that is also used for compact disks and professional devices, improving dynamic range and distortion factor in addition to the conventional 14-bit EIAJ format. 501 es is equipped with 16-bit format which is also used for professional equipment.
In addition, 16-bit tapes can be played on other 14-bit systems as well as 14-bit tapes.

It is equipped with the newly developed OVC (Optimum Video Condition) adjustment function which enables the use of the VCR in the long-time mode.
In the past, when a VCR is used in the long-time mode, the reading level of the digital signal becomes narrower than that in the basic mode, which adversely affects the sound quality during playback. However, the PCM-501ES is equipped with an OVC function that can set the data reading level to an appropriate value in the long-time mode. The optimum point can be set by checking the OVC indicator of the 3-point LED display.
As a result, the PCM 501 es can be combined with almost any video deck, enabling up to eight hours of recording and playback.

An output terminal is mounted on the back panel that enables easy digital dubbing by connecting another VCR.

Sony's proprietary 16-bit A/D and D/A converters and high-performance, small-size LSIs enable high-fidelity design.

It is equipped with a peak program meter that covers a wide range from -50dB to 0 dB (reference level -15dB) by double indication method that displays peak hold and real time level simultaneously.
It is also equipped with an OVER display function that warns of input signal over-level and recording volume over-high, making it easier to set the level.

The peak program meter Rch is equipped with a tracking indicator that displays the tracking status, making it easy to adjust the tracking of the video deck used.

Equipped with a PB MUTE function that automatically mutes when a drop-out occurs and allows you to enjoy comfortable playback.

It is equipped with auto-emphasis function which makes PCM digital sound more beautiful.

It is equipped with a REC MUTE function that allows you to create an unsignalized portion during recording.

Equipped with a Monitor Out connector that allows image monitoring regardless of power ON/OFF.

Equipped with headphone volume terminal that can adjust the headphone level.
