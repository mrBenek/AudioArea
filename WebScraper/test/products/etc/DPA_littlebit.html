Commentary

D/A converter using bit stream 7350 chip.

The input format is the SONY/Philips Biphase Format (SPDIF) Standard and uses a 7350 bit stream chip.

Fully differential balanced mode with a sampling rate of 384 x.

The digital filter uses 8 x oversampling of 18 bits.

Dark green aluminum die-cast case is used for the case.

The power supply is equipped with separate regulators for digital and analog power.

Gold-plated terminals are used for input / output terminals.
