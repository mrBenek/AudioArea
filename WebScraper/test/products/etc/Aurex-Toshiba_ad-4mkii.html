  • Aurex AD-4mkII

  • ¥ 58,000 (around 1981)

Commentary

Address unit equipped with 4-channel address circuit.

The AD-4mkII has four built-in address circuits and can use both encoding and decoding addresses simultaneously. Therefore, when used in combination with a 3-head deck, it is possible to monitor decoded signals simultaneously while recording.

Since two decks can be connected, it is possible to dub deck to deck without passing through an address circuit. It is possible to directly dub address encoded signals or normal signals.
In addition, a recording terminal is mounted on the front, and address recording can be performed simultaneously on a total of three decks.

Equipped with a wide range peak meter of -40dB to + 12 dB. The attack time of this meter is 10 msec and quick response is possible.

The output level fine adjustment function is installed for the address reference signal, and the signal output level can be adjusted to the address reference level -3dB.

Encoding (address recording) The output is 300 mv and can be used even on open decks with low input sensitivity.

Input and output volumes are mounted. These volumes can be used even when the address is out.

The front terminal has priority over Deck 2 on the back for playback.
