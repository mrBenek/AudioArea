Commentary

MC type cartridge, an improved version of DL-301II.

The DL-301II uses an aluminum alloy metal body and is the world's first to allow four stops.
This provides a more secure attachment to the shell.

The magnets and terminal pins are made into a single frame with high-strength plastic, and a case with a precise light alloy finish is used to realize a rigid body.

The cantilever is equipped with an aluminum alloy tapered cantilever equivalent to DL-303, which realizes weight reduction while maintaining sufficient rigidity.

Uses samarium-cobalt magnet.
