Commentary

Audio timer with weekly 4 program function.

Timer reservation is a Weekly 4 program type, and each program can specify multiple desired days of the week at random.

The display uses a 2-color FL tube and is equipped with a dual display system.
Normally, the current time and next timer operation time are displayed, and the timer ON time and OFF time are displayed when the program is set or confirmed.

An operation guide system is adopted in which the display part to be set next flashes when the program is set, so that the program can be set easily.
In addition, it is equipped with a touch tone mechanism that can confirm whether the operation was reliable by electronic sound.

Three program modes can be specified for each program.
Three modes can be selected : Repeat, which operates every week with the same timer for the program contents ; One Time, which operates only for one week and retains the program contents after the operation ends and enters a memory save state ; and Memory, which operates only memory first and allows program set when necessary.

The AC outlet can be controlled independently for Manual, Timer and Sleep modes.

Equipped with a sleep timer mechanism that can be set every minute from 1 minute to 99 minutes.

Equipped with a one touch program check mechanism that allows one touch confirmation of program contents.

Equipped with an auto-dimmer mechanism that automatically adjusts display illumination according to the brightness of the room.

Equipped with shift keys for both forward and reverse directions, which are convenient for program correction.

Equipped with battery (006P, DC9V) backup function, memory can be retained in case of minor power failure or movement.

The time display is 12/24 hour switching type.

Equipped with a second set key that can reset the current time for 0 seconds with one touch.

Equipped with an automatic return mechanism that automatically returns to the current time mode even if the operation is stopped in the middle of the program.
