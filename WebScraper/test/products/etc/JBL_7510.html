  • ¥ 700,000 (around 1983)

Commentary

An automatic microphone mixer developed using JBL's original electronic technology.

Equipped with 4-channel microphone input, it can be expanded up to 24 channels in 4-channel unit by adding separately sold modular unit 7510-01.
The input is a balanced type with 48 v phantom power supply function for condenser microphone.

It has a floating output, a direct output (unbalanced, 24 channels maximum) for supplying each input signal to an external effector device, and an auxiliary input (unbalanced, 1 channel maximum) for receiving the return signal from an external effector device.

The 7510 is designed to function as a normal manual mixer when the toggle switch on the panel is in the ON position, and to display its true value in the AUTO or PRIORITY position.

In the AUTO position, the digital logic circuit constantly compares the input common to all microphones (background noise) with the input to a specific microphone (signal sound), and operates only the desired channels. The digital logic circuit automatically controls the gain so that the output level remains constant regardless of the number of active channels. This allows a constant feedback margin to be maintained without dynamic range fluctuations or breathing, which cannot be avoided with conventional compressors and mixers. The rise time is 30 ms to 60 ms, reducing the influence on the rise of the local music signal. The threshold level is adjustable.

In the PRIORITY position, the channel set to this position takes precedence over other channels. This allows the chairman's remarks to be delivered in a meeting with priority over other channels.

It can be used as a high-speed, high-fidelity noise gate for multi-microphone recording. It can also be used as a concert SR to increase the system gain up to howling, and it can be used as a preset without adjustment.
