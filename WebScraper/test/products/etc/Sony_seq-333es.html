  • SONY SEQ-333ES

  • ¥ 79,800 (released in 1988)

Commentary

A 10-band graphic equalizer with an automatic sound field correction function.

Equipped with an automatic sound field correction function that automatically measures and flattens the frequency response in the listening position using the included microphone and the built-in pink noise generator.

It is possible to obtain correction characteristics by averaging characteristics at up to 8 locations.

Up to 4 patterns of correction characteristics or characteristics set by yourself can be stored.

It is equipped with 3 types of fixed memory set by a recording engineer and a flat position to turn off the equalizer with one touch operation.

It is equipped with a real-time analyzer display that displays the frequency characteristics at that time.

Equipped with a character switch that can invert the correction curve around 0 dB and can also be used as a hissing reduction system.
