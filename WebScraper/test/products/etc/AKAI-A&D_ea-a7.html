Commentary

This graphic equalizer incorporates feather touch operation in all operations.

The feather touch operation realizes a soft feeling of operation.

The EQ section can be used either as an interlocking can control both left and right channels simultaneously by operating the channel selector, or an independent type that can adjust one channel separately. The band is composed of seven bands and can be continuously adjusted from + 10 dB to -10dB.

Each level control button uses a high-precision electronic volume.
In addition, it is equipped with REV function which can invert characteristics with one touch operation.

Equipped with an FL display, it can be switched between 3 modes : curve display of equalization characteristics, spectrum analyzer to inform energy balance for each frequency in real time, and dual display to show curve display and spectrum display simultaneously.
In addition to the 3-mode display system, all control information such as memory and input source status can be displayed centrally.

It is equipped with a computer-controlled memory system and can store up to three types of equalization characteristics and display modes. It is equipped with a fourth memory mechanism, Last One Memory, which can store correction curves when the power is turned off.
The memory protection system employs a rechargeable memory backup circuit using a super capacitor, which can retain memory contents for approximately three weeks.

Two tape decks and one line input can be connected. Also, by switching the REC monitor selector, it is possible to copy from deck to deck and to compare and watch between the sending side and the recording side with one touch.

Equipped with Akai's original signal bypass circuit mechanism that allows the line input signal to pass through as it is even when the power switch is turned off.

Equipped with a DEFEAT button that flattens the compensation curve with one touch.
