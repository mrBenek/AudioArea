  • AKAI RT-100

  • ¥ 8,800 (around 1983)

Commentary

Analog 24-hour program timer.

It is a multi-stage operation type that can record messages 48 times per day and 96 times per day at 15 minute intervals.

The timer set operation can be performed simply by inserting the set pin at the desired time to turn ON/OFF.
