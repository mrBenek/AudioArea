  • Aurex AD-15/AD-15K

  • ¥ 45,000 (around 1978)

Commentary

An address unit that can use the Automatic Dynamic Range Expansion System (adres) on a cassette or tape deck.

The address unit is connected to the input / output terminal of the tape deck and can be used as an encoder and decoder to improve the noise level by 30 dB.

When used in combination with a deck, the input signal levels for encoding and decoding must be matched to the output level of the deck. The AD-15 has a calibration function so that the level can be adjusted.
First, turn ON the calibration switch of the address unit and record on the tape deck at a level of -3dB for about 10 seconds while setting the function of the unit to Record.
Next, set the unit function to Play, maximize the output level adjustment knob on the tape deck, and play back the recorded calibration signal.
Then, adjust the semi-fixed volume of the CAL-PLAY with a flat-blade screwdriver so that both LEDs of the calibration indicator of the address unit light up.

This system consists of a variable response circuit (VRA), a level sensor that records control signals to control the VRA, and a weighting network that provides frequency characteristics.

There are 2 kinds of color variations, AD-15 of walkers or rollators for the elderly Ra and AD-15K of BlackCar Ra.
