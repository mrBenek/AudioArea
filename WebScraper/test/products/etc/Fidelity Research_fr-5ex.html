  • Fidelity-Research FR-5EX

  • ¥ 18,800 (around the early 1970 s)

Commentary

MM type cartridge with high performance vibration system based on FR-5.

A ring-shaped toroidal core is adopted in order to eliminate harmful distortion of magnetic materials.

The FR-5EX achieves an excellent high-frequency response of 50 kHz by improving the vibration system.
In addition, the only change from FR-5/FR-5E is the vibration system, so it is possible to improve the performance of FR-5/FR-5E by using the needle tip of FR-5EX.
