Commentary

MC type cartridge, an improved version of DL-103R.

There was a model called DL-103R/T which is a limited 1000 set of DL-103R and includes a booster transformer AU-103 as a set.

The DL 103 r uses 99.9999% (6N) high-purity copper wire for the power generation coil while following the basic technology of the DL 103.
