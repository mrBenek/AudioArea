Commentary

Pre-main amplifier that adopts TRAIT.

This is the first time in its class to be equipped with TRAIT, which succeeded in integrating temperature detection and compensation circuits into one chip for the first time in the world.
In the conventional configuration in which the temperature detection transistor is attached to the radiator externally, the operating point always changes depending on the output size, resulting in a mismatch in the amplification factor of the music signal and distortion especially in the case of small signals.
In TRAIT, the temperature detection element detects the temperature of the transistors in almost real time because the transistors are placed on the same chip. This temperature detection element enables the realization of a thermal feedback system in which the operating state of the transistors is always ideal.

In order to solve the problem of ground line, pure signal ground line is adopted.
This allows only a pure signal to flow through the main ground line, reducing the noise components that pollute the music signal.

With the aim of minimizing the signal path, it uses logical flow construction with internal layout along the signal flow. This reduces the interference between the large signal part and the small signal part or power supply part, and improves sound quality.
The printed circuit the printed circuit board is also fixed in a soft manner so that it has a vibration resistant structure that is less susceptible to vibration.

A source direct circuit is installed to bypass the tone control and balance control circuits when they are not used.

It has a phono equalizer jack for MM.

Equipped with a loudness control.

Wireless remote control is included.
