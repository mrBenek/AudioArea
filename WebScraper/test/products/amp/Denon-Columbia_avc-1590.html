Commentary

AV amplifier equipped with 32 bit floating point type DSP.


Surround circuits of the same quality are used for all 7 channels, enabling 96 kHz processing from audio signal input to DSP processing and D/A output, enabling more accurate and precise surround processing.

A 32-bit floating point type DSP (third generation SHARC) is used for the DSP for surround circuit processing.
This DSP is compatible with 96 kHz signal input, and the signal processing capability is more than twice that of the conventional model, so the accuracy of various surround decoding processing is also improved.

Uses a 96 kHz / 24-bit digital interface receiver.

All seven channels of the power amplifier are fully discrete and have the same response. In addition, it realizes wide-range playback capable of up to 100 kHz playback, enabling not only impressive multi-channel surround playback but also high-quality 2-channel stereo playback.

The power supply section is equipped with a large power transformer and a large-capacity block capacitor to ensure stable power supply.
In addition, the chassis supporting large transformers is designed to be more integrated with the chassis combined with a high-rigidity transformer frame in consideration of sound insulation.


It supports a wide range of surround formats, including 6.1 channel surround formats such as DTS-ES and Dolby Digital EX, and surround formats such as DTS96/24 and Dolby Pro LogicIIx.


Equipped with a video conversion function to convert between signals connected to the image input terminal or S-image input terminal.

Equipped with an on-screen display function, you can perform various setting operations while viewing the monitor screen.
In addition, the front panel display can display the setting operation contents simultaneously, so the setup can be performed even when the monitor is turned off.

Equipped with an auto surround mode function.
This function allows you to automatically store the surround mode for each of the three input signals : Analog / PCM2ch, Bitstream (Dolby Digital, DTS, AAC) 2ch, and Bitstream multi-channel.
When the type of input signal changes, the surround mode that is most suitable for each signal is automatically set.

The subwoofer has eight selectable crossover frequencies that can be optimized for your speaker system.

The built-in Rec out selector allows you to watch movies and audio while recording.

Equipped with Route 3 D-image input port.
This connector supports D4 video.

8 ch pre-out terminal is mounted.

It uses a newly designed aluminum front panel.

Separate switches are installed for main power ON / OFF and on / standby.

Wireless remote control is included.
