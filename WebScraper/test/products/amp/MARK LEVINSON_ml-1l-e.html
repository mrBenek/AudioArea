Commentary

A control amplifier based on JC-2 and improved in spring 1977.

Complimentary service push-pull circuit of pure class A operation is adopted for all stages.

LEMO connector is used for input / output terminal.

As for the phono amplifier gain and input impedance, in addition to the standard System A3, there were System D5 and System D6 which can be directly connected to MC cartridge and System A4E which is ideal for EMT cartridge, etc.

The power supply unit uses a separate housing.
