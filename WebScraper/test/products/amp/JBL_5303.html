  • JBL 5303

  • ¥ 214,800 (1 unit, around 1973)

Commentary

Low noise, low distortion 3-channel mixer / preamplifier.

It has two microphone inputs and one high-level input, and you can add two microphone inputs by using the optional 5190.
An unbalanced high-impedance input can be connected to each input terminal, and an XLR-type connector is provided for connecting to a microphone.
