Commentary

An AV receiver equipped with Dolby Surround AC-3 as a surround system.

It supports digital signals in the Dolby Surround AC-3 format.
Compared with Dolby Pro Logic, which decodes 2 ch analog signals, the 5.1 ch digital signal, including stereo surround sound, realizes excellent reproduction in terms of sound quality, spatial spread and dynamic range.
You can also enjoy other surround modes such as Dolby Pro Logic Surround, Dolby 3 stereo and DSP Logic.

The amplifier section uses a discrete amplifier configuration with full-scale component design for all channels.

Color OSD (On-Screen Display) is adopted, and when it is set in GUI mode, it can be operated while viewing the screen.

With the included remote control, multiple devices can be operated with one remote control using the setup code.
In addition, it has a MACRO key that can record continuous operations.

Equipped with AM stereo compatible FM/AM tuner.
