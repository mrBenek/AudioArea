Commentary

An AV amplifier compatible with the Dolby Digital DVD format.

Built-in decoder using newly developed Dolby Digital 1-chip IC enables 5.1 channel surround sound playback. Built-in Dolby Pro Logic and DSP functions enable 5-mode sound field control.
The DSP function can be used in conjunction with Dolby Digital and ProLogic playback.

The amplifier section is equipped with a 5-ch discrete power amplifier of 100W.

The built-in speaker setting function allows you to set the appropriate playback conditions according to your speaker system.
It is also equipped with functions such as a low frequency control normalization of disabled and elderly people participating in society that enables you to tune various low-frequency playback parameters according to the viewing environment, a dynamic range compression function that supports the compression of the dynamic range on the soft side, and a dialog function that adjusts the volume of sources of different levels.

Equipped with an AC-3 RF input required for Dolby Digital LD playback and an AC-3/PCM Digital connector (2 coaxial cables, 1 optical cable) to which Dolby Digital media such as DVD and MD can be connected.

There are four inputs and one outputs for the audio jacks, an input Line 5 and an output Route 3 for the video jacks, and an input Line 5 and an output Route 3 for the S jacks.
In addition, all channel pre-out jacks, including the subwoofer output, are mounted.

It uses high-sound parts such as digital / analog independent winding transformer.

The VSX-D7 is equipped with a learning function that can handle non-preset models and functions, as well as a multi-operation function that enables one touch operation from power-on to play.

Multi-control remote controller is included.
This remote control contains preset remote control codes from major manufacturers such as LD, DVD, VCR, TV, etc., so that centralized control of the AV system is possible by simply calling up the codes.
