  • SONY TA-E7B

  • ¥ 150,000 (around 1977)

Commentary

A high-end preamplifier with improved physical characteristics, simple signal path and rich control functions.

The main signal transmission path consists of an equalizer amplifier and a flat amplifier, which is extremely simple structure. The physical characteristics are improved by using a DC amplifier.

The first stage employs a current mirror differential amplifier using a dual FET, and the second stage employs a two stage direct-coupled NF type equalizer amplifier with a constant current load and Darlington connection of PNP transistors.
A DC feedback circuit using FET is adopted to stabilize the operating point of the amplifier stage.

The flat amplifier section is a two stage direct-coupled DC amplifier with differential amplification using FET at the first stage and PNP transistor with constant current load at the second stage.
Direct coupling method is adopted for the input side.

The head amplifier section for the MC cartridge has the same circuit configuration as the head amplifier HA-55. In the first stage, low-noise LEC transistors are connected in parallel to lower the equivalent noise resistance and improve the SN ratio.

An FET buffer power supply is used to bring out the advantages of each amplifier stage.

Low crosstalk and improved SN ratio are achieved by simplifying the Phono input and signal path through relay switching.

It is equipped with a large level meter of 3-mode indication system with sampling and holding function which can accurately read the change of peak level.

Equipped with functions such as function selector, tape monitor switch, tape copy switch, mode selector, balance control, muting switch, attenuator, tone control, high filter, low filter, etc.

Built-in electronic muting circuit to suppress unpleasant shock noise generated when power switch is turned ON / OFF.
