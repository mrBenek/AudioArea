  • ONKYO Integra TX-DS989

  • ¥ 350,000 (released on April 4, 2000)

Commentary

AV amplifier equipped with THX Ultra certified 7 ch amplifier.

The TX-DS989 is available in early and late models.
The latter model is equipped with Dolby Pro Logic II, DTS-ES and MPEG-2 AAC decoder in addition to the same configuration as the former model, and each mark is written on the front panel.
In addition, during the period from April 2001 to April 2003, an upgrade service was provided for the early model, and the early model was compatible with Dolby Pro Logic II, DTS-ES, and MPEG-2 AAC.

Equipped with an THX Ultra certified 7-channel amplifier, it realizes the spread of a front sound field that perfectly localizes to the image targeted by THX and a rear sound field that creates natural surround feeling.
In addition to the conventional 5.1 channels, the THX Surround EX decoder uses a surround back channel to further enhance the sense of diffusion in the rear sound field. This enables a more theater-like playback environment with two surround back channels (surround L/R) and 7.1 channels including a subwoofer.

Equipped with an RS 232 terminal, it can be used for DSP upgrades and home integrated control systems such as the AMX system.
It also includes an expansion board slot for the IEEE1394 terminal, which provides power and voice information through a single cable.

It uses a design in which buttons for setting are hidden inside the electric slide door.
Various settings can be made by on-screen display while watching the display. Each input selector can automatically store the previously selected listening mode and its parameters. In addition, the automatic listening mode selection function linked to input signal formats such as DTS, PCM, and analog, and the intelligent volume function that can adjust the signal level in advance depending on the input device or source can be set for each input selector.
You can also lock the setting.

It features two 24-bit crystal DSP processors and a 192 kHz / 24-bit D/A converter.
In addition, high-quality parts such as Maxim's video amplifier used in professional equipment and toroidal transformer which is superior in low impedance design are used.

Wide Range Amplifier Technology WRAT compatible with DVD-Audio and SACD has been introduced.

Equipped with a linear optimal gain volume circuit that improves S/N with optimal attenuation control.

Equipped with a non-scaling configuration to prevent dynamic range and S/N degradation during downmix.

It is equipped with digital upsampling function which also supports analog source.

It is equipped with an AV sync function (up to 3 frames) to compensate for the minute gap between video and audio that occurs when using a line doubler.

It is equipped with Theater-Dimensional mode which reproduces virtual surround with only the front speaker.

Equipped with power on volume function that can set the level when the power is ON.

Equipped with a maximum volume function that can set the upper limit of the volume level.

The LFE level can be set according to the signal format.

Built-in 40-station preset FM/AM tuner.

Remote controller with learning & macro function is included.
In addition to learning the remote control code of another connected device and memorizing continuous remote control operations using the macro function, this remote control can also illuminate buttons with the back light.
