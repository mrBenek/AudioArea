Commentary

50C-A10 is newly adopted to improve the output limit of SQ38D.

It was developed as the latest model without using the transistor which had started to spread and without using even a single transistor. The initial letter F of Final is added at the end of the model number to mean the last of the 38 series.

50CA10 newly developed by NEC is used for the output tube.
This tube can obtain an output of 30E by push-pull, which improves the output limit of the famous triode 6RA8.

The output transformer is OY15 type.
The SQ38 uses a slightly smaller OY14 type, and the SQ38F was the first to use this OY15 type transformer as a finished amplifier.

The power amplifier section is the same Mulard type as the SQ38, and the preamplifier section is further improved.
