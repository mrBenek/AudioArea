  • Pioneer SM-70

  • ¥ 25,000 (around 1969)

Commentary

An affordable power amplifier using silicon transistors.

It employs a quasi complimentary service type direct-coupled drive using a 3-fold diffused mesa silicon power transistor.

Equipped with two output terminals.
In addition, the input sensitivity can be switched to 3 positions so that it can be matched to the preamplifier to be combined.
