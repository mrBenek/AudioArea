Commentary

This tube type stereo power amplifier was developed as the highest model of the VT series.

The unique output transformer also supports ultra-linear connection and cathode feedback to ensure extremely low drive impedance and highly efficient and stable operation.

A highly reliable large beam tube 6550C is carefully selected for the power stage. The 6550C has a 4-parallel push-pull configuration with 8 left and right channels for a total of 16 channels. Therefore, the output impedance is sufficiently low, and by reducing the number of turns of the transformer, an excellent wide range and constant loss are simultaneously realized.
The signal system has a balanced configuration for all stages. It is equipped in the input stage with a constant current source, the shortest signal path, and the LR independent dual monaural configuration to enhance the total perfection.

A 3-speed -speed silent fan in the center of the chassis to properly cool the vacuum tubes and assist in stable operation when needed.

There is no change in load condition or input sensitivity when either one balanced or one unbalanced input is used.
The impedance of this input circuit is set so that it can be used with either a tube type or a semiconductor type amplifier, so that there is no mismatch when a semiconductor type channel divider is connected.
