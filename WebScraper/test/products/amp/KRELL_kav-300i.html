Commentary

An integrated amplifier which is a minor change model of K300i.

The power amplifier uses 12 specially made metal-can-type power transistors jointly developed with Motorola. Stable current supplying capability is obtained by operating Pc250W class power transistors in 3 parallel / full complimentary service configuration.

The stages of the preamplifier are fully discrete / full complimentary service. All circuits from the first stage of the line input to the previous stage of the output stage are pure class A, and sound quality is enhanced by our unique ultra-low NFB and ultra-wide band design.
The preamp section is fed by a dedicated constant voltage regulator, which is completely independent of the output stage, to eliminate adverse effects on the reproduced sound due to mutual interference.

The volume control uses a 152 step analog volume by applying a multiplexing DAC element.
The volume is attenuated by switching the high-precision R2R ladder resistor built into the DAC, so that transients and phase changes that are unavoidable with a normal variable resistor are small, and a low-colorization sound is realized at any volume position.
The minimum step interval is 0.2 dB and can be controlled from the remote control.

The power supply section contains a large toroidal transformer with a large capacity of 400 VA and a computer-grade filter condenser.

It is equipped with one XLR balance system and single end Route 3 input terminals. It is equipped with one tape loop system and preamplifier terminals which can be used for multi-amplifier, etc.
One single-ended input can also be switched to the theater throughput for connecting a surround processor, and if the K300i is built into a home theater system, it can be used as a single processor for volume control and other operations.
The source selector is provided by a highly reliable, completely sealed relay located at the shortest distance from the terminal panel to eliminate the routing of signal lines inside the enclosure, and the relay itself is controlled by the control voltage of the system control section, which is electrically and spatially independent of the signal path.

The internal layout uses dual monaural construction.

It comes with a wireless remote control of aluminum block carving.
This remote controller is black anodized and lettering is toreutics processed. It provides centralized control not only for all functions including power ON/OFF, but also for other remote-controlled Kroell products such as power amplifiers.
