Commentary

A control amplifier equipped with a pure current dam and a new linear transfer, inheriting the design concept of the high-grade C-70.

The circuit of C-50 is a pure A-class system with a new linear transfer of DC configuration. It is also composed of a unit amplifier equipped with a pure current dam which is an advanced type of pure current servo.

In each unit amplifier except for 6 db amplifier, tone control amplifier and headphone amplifier, pure current dam which is an advanced type of pure current servo is adopted.
In this circuit, a semiconductor inductance in series with the power supply and a high-capacity electrolytic capacitor cut off the power supply with the non-linear component flowing in the power supply line at several Hz so that only the DC component flows. Thus, each unit amplifier is cut off from the power supply when AC is low, and at the same time, the change of the current in the ground line is suppressed.
In principle, this pure current dam operates without the need for a bypass current, making it an efficient and excellent power supply system.

Unit amplifiers, excluding buffer amplifiers and headphone amplifiers, are equipped with new linear transfer circuits.
This circuit works so that the voltage drop across the emitter resistor between the two elements is always constant, and the power supply supplies the same distortion-free current as the output signal. This eliminates the effect of current distortion caused by the gm curve of the element, which cannot be avoided with conventional pure class A bias circuits.

Real-time equalizer developed in C-70 is used in the equalizer section.
This circuit uses two RIAA elements and uses an NF-CR type configuration to achieve ideal equalization in terms of frequency and time. This equalizer also has excellent RIAA deviation, combined with the use of the original low-noise FET to achieve stable clear reproduction.

In the MC head amplifier, an original low-noise transistor with excellent audibility is used in complimentary service, and an excellent SN ratio is obtained.

The newly developed logic control is used for the input selector. This method uses a newly designed plunger switch to improve the operability of the selector operation.
A highly reliable gold-clad relay is used in the disk direct and muting switches. These two switches have a built-in memory that can be restored to the previous state of use at the next power ON.

Equipped with a disk direct switch.
This switch allows you to pass the Input Selector, Mode Selector and Tone Control and simply output the output from the equalizer amplifier to the Pre out.

The tone control section is a continuously variable turnover frequency system.

It uses high-quality parts such as electrolytic capacitors in plastic cases, high-precision RIAA elements, non-magnetic gold plated pin jacks, and oxygen-free copper power cords with polarity.

Yamaha's original high-precision 4-series volume is used.
This is done by attenuating before and after the flat amplifier. The resistance value of the volume can be made smaller than that of the normal one, and SN failure in actual use is improved.

Equipped with a Direct Phono MC terminal that is directly connected from the input terminal to the MC head amplifier.
