  • ONKYO Integra TX-DS989Ver2

  • ¥ 380,000 (released on June 19, 2002)

Commentary

An AV amplifier based on TX-DS989 with improved decoding function and sound quality.

Equipped with an THX Ultra2 certified 7-channel amplifier. Basic functions have been improved from the conventional THX Ultra by improving the power amplifier section, and surround modes have been added to support 7.1 channel speaker settings such as THX Ultra2 Cinema mode and THX Music mode.
In addition, it is equipped with an THX Surround EX decoder that enables a 7.1 channel playback environment including two surround back channels and a subwoofer.

In addition to 5.1 channel audio formats such as Dolby Digital, Dolby Pro Logic II, DTS and MPEG-2 AAC, it is equipped with Dolby Digital EX and DTS-ES (Discrete 6.1 / Matrix 6.1/Neo : 6) decoder which realizes more realistic feeling of theater by adding surround back channel and DTS-ES (Discrete / Matrix / Neo : 6).
It also supports DTS96 / 24, which enables 5.1 channel playback at 96kHz/24bit.

Equipped with an RS232 terminal, it can be used for home integrated control systems such as AMX.
It also has an expansion board slot for the IEEE1394 terminal that enables power supply and voice information transmission with a single cable.

Various settings can be made in the on-screen display while watching the display.

Each input selector can store the previously selected listening mode and parameters, as well as an automatic listening mode selection function linked to the input signal format (DTS, PCM, analog, etc.) and an intelligent volume function that can adjust the signal level in advance depending on the input device or source.
You can also lock the settings.

The amplifier circuit is designed to stabilize the ground potential by carefully selecting the parts to be used in order to improve the reproducibility of minute audio information.
The digital audio processing circuit is equipped with two 24 bitDSP processors manufactured by Cirrus Logic, the DAC is equipped with a 192kHz/24bit D/A converter, and the video circuit is equipped with a Maxim video amplifier that is used in professional equipment.
The power supply parts use a high-capacity toroidal transformer which is superior in low impedance design.

Wide Range Amplifier Technology WRAT for DVD-Audio and SACD.

Equipped with a linear optimal gain volume circuit that improves S/N with optimal attenuation control.

Equipped with non-scaling configuration to prevent dynamic range and S/N degradation caused by downmix.

It is equipped with digital upsampling function which also supports analog source.

It is equipped with an AV sync function (up to 3 frames) to compensate for the minute gap between video and audio that occurs when using a line doubler.

It is equipped with Theater-Dimensional mode which reproduces virtual surround with only the front speaker.

Equipped with power on volume function that can set the level when the power is ON.

Equipped with a maximum volume that can set the upper limit of the volume level.

Equipped with setting function of LFE level according to signal format.

There are two ways to display volume levels.

It is equipped with a function to display naming (select alphabet, etc.) of each input source.

Equipped with a 40-station preset FM/AM tuner.

Remote controller with learning & macro function is included.
In addition to learning the remote control code of another connected device and memorizing continuous remote control operations using the macro function, this remote control can also illuminate buttons with the back light.
