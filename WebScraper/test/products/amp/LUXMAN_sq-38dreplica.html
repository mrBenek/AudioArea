Commentary

Pre-main amplifier reproduced from SQ-38D which is the representative vacuum tube amplifier of Lux old days.

SQ-38D uses OY type output transformer.
This transformer was created in 1964 with more than 30 years experience in transformer making. It was developed by adding handicraft winding technology to the Orient Core Material.

The original SQ-38D was a 10W + 10W amplifier using 6RA8 PP for the output stage, but it has been changed in consideration of the current speaker situation and the characteristics of the input source when the reprint edition is released.
In the reprinted model, 12AU7 is used for the first stage and driver stage, and 6BQ5 is used for the output tube. Combined with an OY-type output transformer, the output of 12W + 12W is secured.
In addition, the input is equipped with the Line 5 input corresponding to the modern input source while inheriting the original design concept, and the phono amplifier is assembled with the circuit by the vacuum tube.

The tone control uses a 12AX7 vacuum tube circuit.

A transformer is used for the MC booster circuit.
