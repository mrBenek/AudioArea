Commentary

A mixer / preamplifier with two programming channels and a Route 6 microphone input.

Channels 1 to 6 are microphone circuits with an input impedance of 50 to 600 Ω, and have built-in microphone input transformer and output transformer.
Each microphone channel is equipped with an indicator lamp so that distortion can be checked before it reaches an audible level.

The program channel has an input impedance of 50k Ω unbalanced and can be changed to a 600 Ω balanced type by using a plug-in transformer 5196.
One of the program inputs can also be used for input from the magnetic cartridge by adding a preamplifier module for the 5192 magnetic cartridge.

With built-in monitoring and gain control circuits, both front and rear panels can be monitored using an auxiliary amplifier or high-impedance headphones.

The equalizer controls can be boosted or cut in 3 dB steps up to 15 dB. The low-frequency settings control frequencies below 300 Hz, and the high-frequency settings control frequencies above 2 kHz.
