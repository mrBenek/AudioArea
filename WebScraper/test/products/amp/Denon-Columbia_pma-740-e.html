Commentary

Stereo pre-main amplifier with non-feedback 0 db amplifier.

Equipped with a non-feedback 0 db power amplifier that eliminates negative feedback loops.
This system is an application of the non-feedback technology developed in POA-8000, and uses a direct distortion servo circuit. The direct distortion servo circuit compares the input signal with the signal obtained from the output side (main signal + distortion component) to detect only the distortion component, and then applies the inverted phase of this distortion component to the input side.
This action drives the static distortion into the 0 region, and the noise appearing at the output side of the amplifier is treated as a difference from the input signal and is removed in the same way as the distortion component.
As a result, there is no signal feedback and no time delay, so the dynamic distortion which affects the auditory quality is suppressed. In addition, since there is no NFB circuit, it is not affected by the back electromotive force from the speaker, so the increase and generation of the distortion when the speaker is connected are suppressed.

A Pc150W power transistor is used for the power amplifier section.

In order to cope with the high dynamic range of digital sources, we have adopted a large transformer with multiple shields, a large capacitor of 20,000 μ F, and a fast recovery diode for the power supply.

The equalizer section is equipped with a unique super-equalizer to achieve a wide bandwidth.

Built-in high-speed protection circuit.
