  • YAMAHA AVX-2200DSP

  • ¥ 220,000 (released in 1992)

Commentary

This AV amplifier is equipped with all the power amplifiers for the 7 channels required for CINE-DSP playback. It also pursues the highest level of sound quality, characteristics, and sound field playback function. It was developed with the aim of becoming an integrated top.

The DSP LSI is equipped with the YSS 213, which was developed and adopted in the AVC 3000 DSP. All digital processing is adopted from input to output for signal processing, and the internal operation word length of 30 bits is realized.
The YSS 213 has both a Dolby Pro Logic decoder and a two sound field DSP processing function, an external expansion interface and an equalizer built into one chip. In addition to the YSS 203 and YM 3413 of the AVX 2000 DSP, it incorporates peripheral circuits and functions. The accuracy and reliability of the Dolby Pro Logic decoder itself has also been improved by all digitization.
The AVX 2200 DSP is equipped with two DSP LSI (YM 3413) by utilizing the external expansion interface function of the YSS 213, and achieves three times the sound field processing capacity of the AVX 2000 DSP.

In order to emphasize the originality of the input signal, during 2-channel stereo playback and HiFi-DSP sound field playback with the effect off, the main channel signal is output as it is through the analog path.
For CINE-DSP and Dolby Pro Logic playback, the input trim is adjusted to the optimum operating level of the A/D converter. The 19-bit A/D system converts the digital signal to the optimum operating level of the 19-bit A/D converter, and the signal is input to the YSS-213. The 18-bit D / A converter converts the DSP-processed digital signal to analog.

The A/D converter uses the 19-bit DSP tracking control A/D system, which is also used in the AVC 3000 DSP. As a result, low-level linearity of 19-bit equivalent and dynamic range of 114 dB are obtained.
The D/A converter is equipped with a compound interpolation type 18bitD/A system with excellent jitter resistance and reproducibility of small signals.

In the power amplifier section, all 7 channels are equipped with an all-discrete power amplifier.
High-quality parts such as specially tuned block Chemicon, mica capacitors, and audio resistors are used. In addition, power transistors and large power transformers with sufficient margin are used.

All input terminals are gold-plated, large speaker terminals compatible with banana plugs and thick cables, and OFC power cable with polarity are used.

Sound field programs include the CINE-DSP11 program and Dolby Pro Logic for AV source.
For music sources, 12 HiFi-DSPs are installed, which are sound field programs measured at concert halls and jazz clubs.

All 7 channels are equipped with pre-out (L/C/R) and line-out (front / rear) terminals, and an external amplifier can be used.
In addition to the mono output which mixes L/C/R, the super woofer output has a split super woofer terminal which distributes C to the L/R.

In addition to the normal test tone function, the test tone function for the super woofer is equipped.

All 11 voice input systems are equipped with an input trim function that can adjust the input sensitivity within a range of 0 to + 6 dB (2 dB step).

Some parameters can be adjusted so that the sound field effect can be changed. A parameter initialization function is also provided so that the changed parameters can be reset to the default values for each program.

It is equipped with a memory guard function. If you turn the memory guard switch on, you can lock and operate only functions that are necessary for daily use. This prevents you from accidentally adjusting the speaker level or changing program parameters.

The center channel is equipped with a 5-band graphic equalizer to adjust the tone of the center speaker.
In addition, bus extension and L/C/R-linked 2-band tone control are installed.

A learning type remote controller is included.

There were 2 types, titanium color and BlackCar color.
