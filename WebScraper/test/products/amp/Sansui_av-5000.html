Commentary

An AV amplifier that supports Dolby Pro Logic Surround.

Equipped with a Dolby Pro Logic Surround Decoder.
It is equipped with five surround modes including Dolby Prologic, which applies a phase shifter to the rear signal to reproduce the movie theater's Dolby surround effect.

It has a built-in FM/AM tuner that can be preset randomly.

The main body panel and remote controller are equipped with multi-function key controls for center, rear speaker and surround adjustment.

Includes a 52-key wireless remote control.
With this remote control, you can not only operate the main unit but also perform basic control of the CD - α 307 connected to the system.
