Commentary

A preamplifier with dual monaural design.

The internal circuit is a dual-mono design, and both the amplifier and power supply have completely separate and independent LRs, thereby eliminating crosstalk between the two channels.

Equipped with 5 types of phono input sensitivity adjustment mechanism, the optimum sensitivity for the cartridge can be selected.

The volume uses a 23-step attenuator type.
In addition, a discrete low-capacity register with self-cleaning contact function is adopted.

It is compatible with MM/MC cartridge by adopting plug-in type module substrate.

It is possible to easily dub by using 2 tape recorders.

There was an optional remote control function.
The volume control and input signal can be switched by a circuit composed of a relay and a resistor. Although 40 dB is the standard setting for volume adjustment, it is also possible to select this in advance using the volume control on the main unit.
