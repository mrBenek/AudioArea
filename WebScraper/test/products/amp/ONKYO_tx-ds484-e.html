  • ONKYO TX-DS484

  • ¥ 50,000 (released on April 19, 2000)

Commentary

In addition to the Dolby Digital, this is the first AV amplifier in this class equipped with a DTS decoder.

It is equipped with wide range technology WRAT which also increases S/N of audibility.

It features a 24-bit crystal DSP processor and a 96 kHz / 24-bitD / A converter.

The LFE level can be set according to the signal format.

Equipped with power on volume function that can set the level when the power is ON.

Equipped with a late night mode to improve the sound field at low volume.

The surround mode can be stored in each input selector.

All channels have a banana plug-compatible screw-type speaker terminal.

Wireless remote control is included.
