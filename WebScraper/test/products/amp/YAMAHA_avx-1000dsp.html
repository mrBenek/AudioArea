  • YAMAHA AVX-1000DSP

  • ¥ 138,000 (released in 1993)

Commentary

This AV amplifier was developed as an amplifier that allows you to easily enjoy the pleasure of CINEDSP.

The DSP-LSI is equipped with the YSS 223 developed for the AVX-1000DSP. This LSI has a built-in 2chA/D and 4chD/A converter in one chip. It also has the functions of the YSS 203 and YM 3413 that were used in the AVX-2000DSP. Input / output is done in analog form, and internal signal processing is done in full digital form.

Sound field programs include 11 CINE-DSPs and Dolby Pro Logic for AV, and 11 HiFi-DSPs for music sources.

All 7 channels of the power amplifier are mounted in discrete configuration.
In addition, all input / output terminals are gold-plated, and OFC power cables with polarity are used to improve quality.

It is equipped with a front mix switch. By switching this switch to the on/5ch side, you can enjoy CINEDSP even in 5 ch playback with the same speaker arrangement as Dolby Pro Logic playback.

Equipped with a test tone function.

The center channel is equipped with a 5-band graphic equalizer, enabling you to tune the sound with the L/R speakers.

Part of the parameter is released and the sound field program can be adjusted.
With the parameter initialization function, each program can be reset to the default value with one touch operation.

It is equipped with a memory guard function. If you turn on the memory guard switch, functions other than those necessary for daily use are locked and you do not need to worry about changing the set value accidentally.

Speaker terminal for banana plug is adopted.
In addition, pin output terminal for super woofer is mounted.

Dual center speakers are supported.

A learning remote controller is included.

There were 2 types, titanium color and BlackCar color.
