Commentary

A stereo power amplifier with reduced dynamic interference distortion.

In order to reduce dynamic interference distortion, the AB and LR independent power supply system is adopted, which is further developed from the A-class and B-class independent two power supply system adopted by VICTOR/JVC.
Dynamic interference distortion occurs when the current consumption of the class B power amplifier stage (output stage) changes drastically according to the signal level change and interferes with the class A amplifier stage before the output stage where the voltage is amplified. To fundamentally solve this problem, in the class A and class B independent two power supply system, the power supply for the class A stage and the class B stage is separated from the transformer itself, and the power supply for the class A stage is made constant voltage. This prevents the voltage fluctuation of the class B stage from being directly transmitted to the class A stage, thus preventing dynamic interference distortion.
The AB / LR power supply system adopted for the M-3030 has been developed to further reduce distortion by using the power supply for the B stage exclusively for the left and right channels.

Twin monaural configuration is adopted to reduce static crosstalk such as jumping interference between board patterns and signal wires, potential difference between ground lines, jumping interference due to chassis structure, etc.

An ICL (input, capacitor-less) circuit has been adopted in which the coupling and condenser of the input section are eliminated.

Equipped with left and right independent level control.

It is equipped with two input terminals, Directo and Subsonic.
