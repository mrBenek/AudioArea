  • Technics ST-K808

Commentary

FM/AM tuner preamplifier that combines a quartz synthesizer FM/AM tuner, a microcomputer audio timer, and a preamplifier into one.

The multi-functional LSI that integrates 18000 elements on a single chip, including the quartz synthesizer, controls the flow of music signals, the flow of control signals from the space control system, and the power on/off all in an instant and accurately.

The tuner section uses a quartz synthesizer system that achieves high equivalent accuracy.
In addition, 8 preset FM/AM stations can be set. It is equipped with functions such as battery backup to protect the contents of memory, and last one memory which can be enjoyed by the receiving station from the previous time when power is turned on.

Timer programming is available for a total of three operations : two operations for every day designation to operate the system at a fixed time every day and one operation for once designation.

The timer programming function allows you to specify the on/off time, select the music source, and select a preset station for each of the three every-once operations.

Equipped with a clock call button to check the current time.

The preamplifier section is designed to reduce noise by examining the elements and circuit constants.

Microphone mixing with each program source is possible, and the mixing recording output terminal is installed so that mixed sound can be recorded as it is.

When combined with the space control system SH-R808, stereo operation is available at hand.
