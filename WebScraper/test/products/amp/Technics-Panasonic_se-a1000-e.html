  • Technics SE-A1000

  • ¥ 70,000 (around 1995)

Commentary

Power amplifier that adopts MOS class AA amplifier.

A MOS class AA amplifier is used. A pure class A voltage control amplifier using a MOS FET with excellent frequency characteristics and linearity and a current supply amplifier composed of a bipolar transistor with driving power are connected by a class AA bridge to improve sound quality.

The electrolytic capacitor for smoothing the power supply circuit uses a bamboo fiber mixed separator, which is combined with carefully selected aluminum foil and improved electrolytic solution to obtain pure driving force.

It uses a large analog power meter that can read directly from a 0.001W minute signal.

The front panel uses high-density hybrid material that suppresses vibration.

Gold-plated input terminal is used.

OCC wire is used for internal wiring.

Speaker terminal compatible with banana plug is adopted.

Uses OFC power cord with polarity display.
