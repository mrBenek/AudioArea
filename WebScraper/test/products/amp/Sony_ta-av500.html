  • SONY TA-AV500

  • ¥ 59,800 (released in 1988)

    　
Commentary

An AV amplifier that connects various AV equipment and enables centralized control from one unit.

Surround sound playback is possible with a surround processor and rear amplifier built in.
The surround mode is equipped with three modes : Dolby, which supports Dolby surround software, a concert hall where you can enjoy the reverberation of a concert hall, and simulated for mono source.

Four types of audio equipment can be connected, and the Rec out jack outputs audio from the selected video equipment as well as audio from the audio equipment.

Four video devices can be connected. The 1/2 system is equipped with an input / output terminal, and the 3/4 system is used as an input terminal for a dedicated player.
In addition, S-video input / output terminals are mounted on Video1-3 and monitor terminals.

The monitor output for connecting to a TV is equipped with an image output for sending out the image of each device as it is, and an RF output terminal for connecting to a TV without a video input terminal.

It is equipped with a mix function that allows you to select video and audio independently.

Contains a connector for the graphic equalizer.

Two speaker outputs are installed.

It comes with a programmable remote controller that is compatible with the Sony remote controller and the main unit.
Even if the device to be combined is not Sony, if it is an infrared remote control compatible device, it can be operated by reading the signal of the remote control and storing it.
