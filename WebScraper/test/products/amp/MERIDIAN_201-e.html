Commentary

This is a preamplifier that has been improved over a long period of time, made simple and straight, uses carefully selected parts, and is equipped with new functions and possibilities.

The input offset function allows you to set the same volume level to match the input level of each source. You can also set the upper limit of the volume level and the muting level. You can each input source.

By incorporating the System2 module (sold separately), the preamp function for two units can be provided.
For example, two different music sources, such as CDs and FM broadcasts, can be controlled independently and played in separate rooms.

It can be connected to CD Player 207 mkII or FM Tuner 204 with an optional network cable.
P lead is used for connection with 207 mkII, and Q lead is used for connection with 204.
