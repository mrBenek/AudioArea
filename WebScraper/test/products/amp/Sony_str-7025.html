  • SONY STR-7025

  • ¥ 57,800 (around 1975)

Commentary

AM/FM stereo receiver with improved operability.

A newly developed junction FET has been adopted for the FM front end to improve the receiving capability.
In addition, a solid-state filter is used in the IF section to sharpen the separation of adjacent channels.

Frequency number line type varicon is used, and more accurate tuning is possible by dialing at equal intervals.

Solid state filter is used in the AM tuner section.

The power amplifier section uses a two power direct coupling system using a hybrid power IC.

The tone control circuit uses NF type.

It has a built-in microphone amplifier for mixing.

Equipped with a loudness switch and a filter switch.
