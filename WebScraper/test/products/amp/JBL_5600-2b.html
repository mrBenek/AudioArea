Commentary

A mixer / preamplifier that is suitable for PA in conference rooms and halls.

It has four microphone inputs and two program inputs, and you can add an optional 5190B microphone preamp to use two more microphones.
When using 5190B, remove the top left cover of the front panel.

All inputs are unbalanced high-impedance type.
The microphone channel can be changed to a balanced low impedance by using a 5901 microphone input transformer, and can be converted to a 50k Ω unbalanced program input by using a 5904T pad.

Each of the two program channels has a socket for a 5195 matching / bridge transformer and can be converted to a 15k Ω bridge or 600 Ω matching balanced input.
You can also use a preamplifier for a 5191 magnetic cartridge / tape head.

Equipped with tone control and cue control.

The VU meter can be adjusted from the rear.
