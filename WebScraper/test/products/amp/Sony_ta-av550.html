Commentary

An AV amplifier with Dolby Pro Logic Surround.

Equipped with digital delay Dolby ProLogic Round.
With the TA-AV550, you can enjoy four pro-logic modes, including the 3 ch logic mode, which allows you to enjoy Dolby surround sound without the rear (surround) speakers.
Normal mode is selected when a small speaker is used as the center speaker and plays with five speakers. Wide mode is selected when a center speaker is the same size as the front speaker. Phantom mode plays surround with four speakers, front and rear.

The TA-AV550 has a built-in power amplifier for 4 channels, and an external amplifier is required for the center speaker.

In addition to the Pro-Logic mode, it is equipped with a hall mode in which you can enjoy a stereo source with a realistic sensation like a concert hall, and a simulation mode in which the sound of a monaural source is expanded.

It has five input jacks for audio and an adapter jack to which a graphic equalizer can be connected.
Image input is installed in the Line 5, and 4 systems are also compatible with the S-image connector.

It comes with a remote control with a learning function. This remote control can memorize the signal of the remote control if it is an infrared compatible device.
You can also set the volume of the master and rear and select the delay time to set the best surround condition in the listening position.
