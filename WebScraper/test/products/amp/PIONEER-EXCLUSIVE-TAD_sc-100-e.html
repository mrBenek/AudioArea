Commentary

A high-end preamplifier that uses silicon transistors for everything.

Equipped with Phono which can alternately use Route 3 pickup, MC cartridge can also be used with built-in MC amplifier, and Phono2 can switch input impedance to 3 stages.
The input selector is divided into two systems, a low-level system and a high-level system, so you can pass through positions that are not used when selecting.

The equalizer amplifier is a 3-stage direct-coupled E-E feedback type using low-noise silicon transistors.

The volume is a 2 dB step real attenuator type.
It also comes with a 15 dB 3-step lever attenuator that can be used as a muting switch.

By using a constant-voltage power supply with a zener diode, it is designed to follow power changes with a margin.

For tone control, Bass and Treble are independent 2.5 dB step 11 contact rotary type is adopted.
The filter is an electronic filter that sharply cuts noise without affecting sound quality. The cutoff frequency can be switched between two levels for both Low and High.
