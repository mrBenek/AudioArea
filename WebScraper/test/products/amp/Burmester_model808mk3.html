Commentary

This is the third generation preamplifier of the Model808, which was developed as a reference unit.

The Model808mk3 consists of a base unit for control functions, an independent power supply box, and input / output modules connected to the base unit for optimum gain adjustment.
Up to six input modules can be a maximum of six modules, and output modules can be balanced or unbalanced for each input.

There were eight input modules : balanced phono MC, unbalanced phono MC, unbalanced phono MM, tuner, CD (standard equipment), tape, aux and DAT.
In addition, all input modules except the phono type are equipped with a changeover switch that can be used for both balanced and unbalanced types.

Because of the ultra-low output impedance design, the headphones can be connected directly to the output terminal.

Equipped with phono adapter plug, level controller for each module, sine wave generator and digital display.

2-way signal monitoring enables tape dubbing.
During dubbing, the signal path is greatly shortened through a relay switch.

High-precision tracking and discrete volume control provide excellent channel balance at ultra-low power.

ON/OFF of the power amplifier and クティブスピーカー can be remotely controlled via an independent power box supplied with 10 VDC.

Remote control for input / output contrast can be installed as an option.
