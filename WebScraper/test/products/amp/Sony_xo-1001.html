  • ¥ 128,000 (released in 1982)

Commentary

This is the world's first stereo deck receiver with a barcode reader program timer.

The timer uses a bar code reader input method that requires only rubbing the input sheet.

Equipped with weekly timer function.

Equipped with a program priority function to enter the set mode at the reserved time.

Equipped with linear electronic counter with subtraction function.

The tuner section uses a quartz-lock digital synthesizer system and can preset eight FM and eight AM stations.
