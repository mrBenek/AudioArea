  • DBX BX-3

  • ¥ 250,000 (released in 1986)

Commentary

A 4-channel power amplifier with BTL connection.

The BX-3 is configured as an independent four power stage configuration.
This configuration allows you to select the output mode using the switch, so you can play two channels with a bridge (BTL) connection, or a three channel system that combines a two channel system with a BTL connected subwoofer.

The power supply section uses a large toroidal transformer with extremely thick bifilar winding with excellent regulation.

Four 150W high-power transistors are placed in each power stage in a parallel push-pull configuration so that 40A of current can be instantaneously sent to the speakers while keeping the output impedance low.
The first stage has a dual-FET direct input and all stages have a direct DC amplifier.
