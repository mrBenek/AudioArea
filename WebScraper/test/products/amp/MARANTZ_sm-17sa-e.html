  • Marantz SM-17SA

  • ¥ 90,000 (Released in September 1999)

Commentary

A flexible stereo power amplifier that can be used not only independently but also in combination with the PM-17SA.

The circuit configuration is designed for compatibility with the PM-17SA and uses a current feedback circuit similar to the PM-17SA.
In the current feedback type, phase compensation is light by lowering the feedback impedance, and high-speed amplifier is realized. In addition, stability against load fluctuation is high, and it can drive any speaker system powerfully.

Marantz's original voltage amplification module adjustable HDAM circuit is mounted.

A super-ring core toroidal transformer is used for the power supply.

The internal layout uses twin monaural construction to eliminate crosstalk caused by channel interaction.

The speaker terminal is made by WBT of Germany.

Equipped with a warm-up meter that can check the heat-up state.

The chassis has a highly rigid body structure made of aluminum.

It can be used not only as a stereo power amplifier but also as a monaural power amplifier using a BTL connection.
Equipped with a newly developed BTL-SA line amplifier that uses a high-slew rate, low-distortion SA buffer circuit, it demonstrates its ability when using BTL.
In addition, line output terminals for both forward and reverse phases are mounted, making the system highly expandable.

Can be used alone or in combination with PM-17SA.

An inlet type AC cable is used.
