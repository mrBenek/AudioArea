Commentary

A preamplifier that was born as the highest peak of modern amplifier.

The LNP-2L has been replaced with the original ML module specifications from the Burwen Corporation which was used in the LNP-2.

The amplifier consists of three stages : a phono amplifier, an input amplifier, and an output amplifier. Each stage has one module per channel, and each stage has one module for the meter. There are eight modules in total.

The LEMO connector is a CAMAC-type connector used for special applications such as medical devices and physical measuring instruments. The LEMO connector is also used for the tape input / output and main out output. The LEMO connector is also used for the tape input / output and main out output.

We put top-of-the-line products into parts such as volume and changeover switch.
The meter is made by Weston and can be switched to VU or peak with a special module. In the HF + position, it can be adjusted to high frequency so that it can be used in cassette recording.

The power supply unit (PLS-153L) is separated.
