Commentary

This is a preamplifier that improves quality by thoroughly pursuing each item such as the circuit configuration of each block, each element constituting the circuit, construction including the arrangement, etc., and drawing out the performance of each block to the maximum extent.

The TA-E88 combines two monaural amplifiers and is designed to be as simple as possible. For this reason, the input / output terminals are set on the top surface, and control components such as attenuators and switches are mounted on the printed circuit board.
This provides a layout in which each circuit is connected in a straight line along the signal path.

The volume for attenuators and balancers uses an ultra-low noise, high-precision volume that uses a gold-evaporated conductor on the sliding part.
We also use carefully selected devices such as a highly reliable selector switch with silver-clad contacts and a structure that eliminates unnecessary vibration through a coupler, a non-resonant capacitor, high-precision metal-coated resistor, and semiconductors.

The head amplifier realizes low distortion by connecting several LEC transistors with low noise characteristics in parallel.
In addition, circuit technologies such as differential circuits using dual FETs, cascode connections, current mirrors and diamond circuits are incorporated into the equalizer amplifier and flat amplifier, while the input buffer amplifier uses almost the same circuit configuration as the equalizer amplifier except for the output stage.
All these circuits are DC amplifier configuration.

The power supply section uses dedicated power transformers for the left and right channels respectively, and four independent power supply systems using FET buffer power supply are adopted.

Equipped with a cartridge load selector that can switch between load resistance of 10k Ω ~ 100k Ω and load capacity of 100 pF ~ 500 pF.

Equipped with a low-cut filter switch for the Phono signal.

Equipped with a quick access type function selector, 2-system tape monitor switch, and mode switch.

Equipped with an output level selector switch that can switch the output level between 0 dB, -10dB, -20dB and OFF.
