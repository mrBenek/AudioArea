  • YAMAHA CA-1000II

  • ¥ 125,000 (around 1976)

Commentary

Pre-main amplifier which is an improved version of CA-1000.

The power amplifier section uses a pure complimentary service OCL circuit directly connected to all stages.

The equalizer amplifier consists of an input stage using Yamaha's original dual FET and a class A SEPP output stage using a transistor.

Built-in class-A / class-B switching circuit provides high output for class-B and high transparency for class-A.

All circuits are analyzed by CAD method using a computer.
