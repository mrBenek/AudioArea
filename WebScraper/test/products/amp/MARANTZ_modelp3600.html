Commentary

A control amplifier which is a selection item of Model3600B.

The Model3600B is subjected to a thorough examination, and the items that show particularly excellent measured values are selected with the name of a professional.
The front panel uses a thick 8 mm solid panel and is a 19-inch rack mount type.

In designing the basic circuit, the input source as much as possible and each component on the circuit are programmed into a computer, and simulation is carried out many times to obtain the highest value in each characteristic such as harmonic distortion, intermodulation distortion, S/N ratio, etc..
In addition, listening tests are conducted using the know-how obtained through experience and ears.

The rear panel has terminals for the main and remote speakers, which can be switched by the control amplifier.

Equipped with scope output, input waveform can be observed in combination with Tuner Model150.

There was a wood cabinet of mahogany as an option sold separately.
