Commentary

AV amplifier with THX Select certified 6-channel amplifier.

Equipped with THX Surround EX decoder, it can play back one surround back channel (monaural) and 6.1 channels including sub-woofers.
In addition, a 7.1 ch pre-output terminal is mounted for audio output, and surround back can be played back in 2 ch by connecting to an external power amplifier.

In addition to Dolby Digital, it is equipped with Dolby Pro Logic II which realizes 5.1 ch playback of various sources, 6.1 ch DTS-ES which adds surround back channel to conventional DTS 5.1 ch, MPEG-2 AAC and other decoders such as MPEG-2 AAC.

You can make various settings on-screen while viewing the display using the easy-to-use cursor buttons.
In addition, the Basic menu and the Advanced menu for custom setting are installed to improve usability.

It features two 24 bit crystal DSP processors and a 192kHz/24bit D/A converter.
It also uses high-quality parts such as an audio / video separation board that protects signal purity by preventing noise contamination, and large aluminum extruded heat sink separated for front & center and surround & surround back.

It is equipped with a D4 terminal in addition to a component video terminal, a digital audio input / output terminal and a 7.1 channel input / output terminal.
In addition, the video signal input from the composite terminal can be output from the S terminal and vice versa.

Equipped with wide-range amplifier technology WRAT for DVD-Audio and SACD.

Equipped with a linear optimal gain volume circuit that improves S/N with optimal attenuation control.

Equipped with a non-scaling configuration function to prevent degradation of dynamic range and S/N during downmix.

Equipped with a speaker impedance setting function that can be used with 4 Ω speakers.

It is equipped with digital upsampling function which also supports analog source.

Equipped with equipped with an A/V sync that compensates for the minute gap between video and audio that occurs when using a line doubler.

Onkyo's unique surround modes include Theater-Dimensional, Mono Movie, Enhanced7, Orchestra, Umplugged, Studio-Mix, TV Logic, and All Ch Stereo.

Equipped with power on volume function that can set the level when the power is ON.

Equipped with a maximum volume that can set the upper limit of the volume level.

Equipped with a listening mode memory function that automatically memorizes the listening mode for each input source type.

Equipped with an intelligent volume function that automatically adjusts the volume level of each media to a constant level.

The LFE level can be set according to the signal format.

There are two ways to display volume levels.

It is equipped with a function to display naming (select alphabet, etc.) of each input source.

Built-in 40-station preset FM/AM tuner.

It comes with a remote control equipped with learning & macro function that includes operation codes for AV equipment of major manufacturers in advance.
In addition, this remote control is capable of self-illumination of buttons by the back light.
