  • SONY AVU-700

  • ¥ 88,000 (released in 1995)

    　
Commentary

An AV amplifier that realizes AV control with one button remote control.

Wireless Remote Control RM-VP1 with one button operation is included.
The AVU-700 is equipped with a GUI called Visual Touch that uses the display as the control panel of the remote control. With this GUI, it is possible to operate the amplifier itself and other AV devices with a single button.
When a remote control button is touched, a menu screen and a pointer indicating the position appear on the display. The pointer moves according to the movement of the hand holding the remote control. You can operate the remote control as if you were operating a PC mouse.

If the connected device supports an infrared remote control, you can control it almost as much as the remote control provided with the device.
The actual operation is not performed directly from the one button remote control, but is performed by transmitting infrared rays to the AV equipment from the transmitter installed on the front panel of the main unit of the amplifier. This eliminates the need for wiring for the remote control that connects the amplifier and each equipment. The transmitter is designed for wide directivity so that it operates without any problem even when the main unit of the amplifier and each equipment are not facing each other.
You can also operate the remote control signals of other manufacturers' devices by making the amplifier learn the remote control signals.

The menu screen can be displayed in Japanese (katakana) to make the operation easier to understand, and the contents of the operation screen have been improved so that the frequently used screen such as volume adjustment can be called up immediately.

Equipped with an illuminated flasher that displays remote control operations on the display even when the TV is turned off for operating software that does not involve images such as CD playback.

There are six sound field programs, including Dolby Prologic Surround sound. The ケオーダー mode is the signal output from the Dolby Prologic d theater plus the sound of the theater. You can enjoy Dolby stereo-type film software in the atmosphere of a movie theater.

The amplifier section is equipped with 5 channel power amplifiers for front L/R, center and rear L/R.

Equipped with an FM/AM tuner that supports AM stereo.
