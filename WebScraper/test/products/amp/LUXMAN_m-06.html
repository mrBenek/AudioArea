Commentary

This is a stereo power amplifier that uses an authentic pure class A system and a very strong power supply.

It uses an authentic pure class A system that supplies a constant bias current up to the maximum output.

The power supply section is equipped with a large toroidal power transformer and a separate transformer is used for digital display to strictly eliminate interference with the signal system.
In addition, it is compatible with low impedance loads from 2 Ω and has acquired stable speaker driving capability.

Equipped with a digital display type peak power meter.
The display hold time of this meter can be changed in 3 steps of 1/3/5 seconds, peak hold is also possible, and the display impedance can be changed in 4/6/8.
The meter brightness can be adjusted in three steps with the dimmer switch, or the whole display system can be turned off to be completely disconnected from the signal system.

It uses PC-OCC wiring, nitrogen gas sealed contact relay, bus bar to reduce power source impedance, and high-precision attenuator using fixed resistance.

A heat pipe system is adopted to achieve a high heat radiation effect.

The input terminal is equipped with a common coaxial type as well as a canon type.
The Canon type also supports direct balance input without using an inverting amplifier.

It can be used not only as a stereo power amplifier but also as a high-output mono power amplifier.
