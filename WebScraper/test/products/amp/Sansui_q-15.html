  • SANSUI Q-15

  • ¥ 14,400 (around the 1950 s?)

Commentary

Tube type power amplifier with monaural configuration.

The input side is equipped with volume control for level adjustment. Even if an excessive signal is applied to the input terminal, it can be easily adjusted with a screwdriver.

The input voltage can be switched to 3 types by replacing the plug, and it can be used overseas.

With its robust mechanism and durability in mind, it can be used not only for home use but also for business use and long-term regeneration.
