Commentary

Kit model of quartz synthesizer FM stereo tuner equipped with timer function.

Equipped with a timer function, you can program up to three stations you want to listen to, and the tuner automatically selects and plays back the station at the set time. You can also record the program by connecting to the deck.
This function uses a microcomputer to store and control three types of programs, two everyday programs and one once program.

Quartz synthesizer is used for the front end part.
Preset memories can be set up to 8 stations. In addition, automatic / manual switching for a comfortable tuning feeling is provided, and scanning function for listening by flowing the memory station every 5 seconds is also included.

The front end section employs a dual-gate MOS-FET for high-frequency amplification, and with a double-tune connection using four twin-type varicaps, cross-modulation, intermodulation and image interference are completely eliminated.
A newly developed ultra-low distortion factor IC is also used for detection and stereo demodulation, giving importance to sound quality in the design.

An accurate digital quartz clock can be displayed with one touch operation.
This function is displayed for approximately 5 seconds.

It is also equipped with power failure compensation function using batteries.

It includes a test tone circuit, front output level control, 3-color signal indicators and station indicators.
