  • Technics ST-S4T

  • ¥ 52,800 (around 1981)

Commentary

Microcomputer / quartz synthesizer FM/AM tuner with enhanced timer function.

As with the ST-S4, the RF and IF stages are designed with an emphasis on waveform transmission.

The IF stage is equipped with a super narrow IF with 20 dB selectivity at 200 kHz detuning, and excellent adjacent station interference rejection capability is obtained.

It is equipped with a swirl-in-counter type 25 kHz reference frequency audio synthesizer.

The double-switching PLL MPX realizes high-precision stereo demodulation.

FM/AM Each 8-station preset memory is available.

Equipped with manual / auto-scan tuning function.

The ST-S4T is equipped with a microcomputer programmable timer.
This timer can be used to control the FM/AM receiving station and power on/off time in addition to the time management display function of 10 seconds per month. There are two types of timer operation (everyday) and one type of timer operation (once), and it can be set up to three times per day.

Rear panel with up to 680W AC outlet.

Equipped with memory backup function using 3 AA dry batteries.
The power failure compensation period of this function is about 2 months.
