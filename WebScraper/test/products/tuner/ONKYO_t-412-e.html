  • ONKYO T-412

  • ¥ 29,800 (around 1995)

Commentary

Quartz synthesizer type FM/AM stereo tuner for AM stereo reception.

It is equipped with an APR system that automatically switches between sensitivity (DX/LOCAL) and stereo mode according to the radio wave condition of FM broadcasting.
As a result, the optimum receiving condition can be automatically set without troublesome operation.

An easy-to-operate rotary knob is used for the tuning dial, which can be used for both frequency adjustment and preset station selection.

A total of 30 stations can be preset, with 10 stations for each group of A, B and C.

It is equipped with an auto-preset function that presets all the receivable broadcasting stations with one touch operation.

Equipped with a signal level indicator that displays in five levels according to the strength of radio waves.

You can set a character that can display up to 8 digits of alphabet or number in the stored frequency.

If you know the frequency of a broadcasting station beforehand, you can directly select the station with 10 keys.

Equipped with F-type connector to which antenna coaxial cable can be connected.

It is equipped with RI remote control terminal. It can be operated with the remote control attached to the amplifier by combining it with an Onkyo amplifier with RI mark.

Power cord with polarity display is used.
