Commentary

An FM/AM tuner with a compact body of the same size as the pre-main amplifier TA-P7F of the PRECISE component and circuit technology such as a crystal lock digital frequency synthesizer receiving system.

With the Digital Frequency Synthesizer (frequency rigidity) method, which is one step further than the crystal lock method which suppresses tuning deviation due to aging or temperature change, the performance of each tuner block can always be fully demonstrated.
Step reception is available at 100 kHz for FM and 9 kHz for AM.

Both FM and AM are equipped with an auto-tuning function that automatically searches for a tuning frequency and a manual tuning function that allows you to select an arbitrary station.
The manual and auto operation buttons are independent of each other. In manual tuning, you can switch between step and quick tune.

With the 4-chip microprocessor MNOS memory IC, up to 8 FM/AM stations can be preset at random. Also, with the built-in last station memory function, it is possible to store the selected station just before the power switch is turned off and before switching FM/AM.

Built-in memory -in memory scan circuit that can be automatically recalled every approximately 3.5 seconds in the order of preset memories.

The built-in canceling function of the pilot signal realizes the MPX section with high separation and low distortion factor.

The RF amplification stage by MOS FET and the FM pure electronic tuning front end by 4-stage varicap enable stable reception even under strong field input.

We have improved both selectivity and distortion factor by using carefully selected uniphase filters and IF ICs for the IF amplifier.

A muting circuit using a reed relay allows for comfortable tuning by cutting off inter-station noise without losing sound quality.

The AM receiver is equipped with an IC and a pure electronic tuner using IF carefully selected for auto-tuning.

Equipped with a 5-step signal indicator.
