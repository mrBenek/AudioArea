Commentary

This FM tuner incorporates various technologies such as the crystal lock system.

It adopts the X' tal Locked method which always fixes the IF frequency at 10.7 MHz using a crystal oscillator.
This eliminates the need for troublesome tuning meters and allows easy tuning to the minimum strain point. In addition, since the lock accuracy is high, the maximum strain point can be maintained for a long time.

A newly developed 1-block, 6-pole linear phase type IF filter is used in 3 blocks for a total of 18 poles to achieve high selectivity.

It uses an input signal intensity meter with a dB scale, and can directly read the intensity of 20 dB ~ 80 dB.

The front end uses a new type precision five series varicon and dual gate MOS FET to obtain high sensitivity and interference signal rejection capability.
The dial scale is a 200 kHz regular interval scale with a total travel distance of 250 mm.

The muting is a 2-stage switching type of Local and Distant, and a reed relay is also used.

A double-balanced demodulator is used in the MPX circuit to achieve good separation characteristics up to high frequencies.

3-stage direct connection output circuit of 2-power supply system is adopted.
