Commentary

FM/AM tuner equipped with super narrow IF position and active sensor.

The IF stage is equipped with a super-narrow IF position with an effective selectivity of 20 dB at 200 kHz detuning in addition to an IF normal position of 60 dB at 400 kHz detuning. In this position, excellent selectivity is achieved and excellent adjacent station interference rejection capability is obtained.

It uses an active sensor system where all tuning information is concentrated on the dial indicator.
The indicator consists of two arrow-shaped LEDs and a vertically thin red LED. When the FM station is not being received, only the red LED lights up. Near the tuning station, the arrow-shaped LED lights up to indicate the direction in which the dial should be turned. When tuning is performed, only the red LED lights up again.

Equipped with a 5-point LED signal strength indicator to indicate electrolytic strength.

Low distortion and high separation are achieved by double differential switching MPX.

It uses a large flywheel with large inertial mass and smooth rotation.

Uses a dial scale that rises with edge illumination.
