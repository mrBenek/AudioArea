  • SONY ST-V7

  • ¥ 52,000 (around 1983)

Commentary

FM/AM tuner compatible with system remote control.

Remote control tuning is possible with the attached remote commander.
When combined with a pre-main amplifier TA-V7 and a cassette deck TC-V7, these operations can also be controlled by the remote control.

Eight FM/AM stations can be preset at random.
In addition, a multi-process memory is adopted, and settings such as muting and FM mode can be memorized for each station.

Uses a flat panel design for seat key operation.

It is equipped with a central display by FL tube which shows band, receiving frequency, muting, preset channel and stereo indicator all together.

It is equipped with a memory scanning function that allows you to call up preset stations one after another at approximately 4-second intervals and check the program contents.

Equipped with a 3-point LED signal indicator.
