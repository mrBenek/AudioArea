Commentary

FM/AM tuner that uses Sony's original direct comparator system.

By adopting a high-speed PLL-IC developed by Sony, we have adopted a direct comparator method in which the comparison frequency of the synthesizer tuner is set to 100 kHz, which is the same as that of the receiving step. This greatly reduces noise and beat and improves the SN ratio.

A total of 10 FM/AM stations can be preset at random. This memory uses a multi-process memory that stores settings such as muting ON/OFF.

It is equipped with a program memory function that makes it possible to record messages from several stations.
By connecting an optional program timer, you can call up to four specified preset stations in order.

It is equipped with a manual tune function that makes it easy to find weak electric field stations. In addition, it is equipped with three tuning methods : a sweep tune that searches for stations in the band one after another and receives radio waves for about four seconds and moves to the next, and an auto tune that automatically finds a station.

It is equipped with a memory scan function that allows you to recall preset stations one after another and check programs.

It is equipped with a centralized display by FL display tube.

A dual-gate MOS FET is used for the front end.

Oxygen-free copper wire with polarity indication is used for the power cord.

Uses carefully selected audio-only parts.
