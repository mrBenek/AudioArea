  • Marantz ST-55

  • ¥ 39,800 (around 1981)

Commentary

FM/AM tuner that uses a quartz-lock digital synthesizer system.

The front end uses a fully electronically tuned quartz synthesizer to receive FM every 0.1 MHz and AM every 9 kHz. This prevents distortion due to tuning deviation, which is a problem in varicon tuning.

Seven FM/AM stations can be stored in memory, and the preset & memory function is equipped to enable one touch channel selection at any time.
Once preset by the backup power supply, the stored station does not disappear, and the memory function is maintained even when power failure occurs.

Equipped with a last station memory function that stores the station before the power is turned off and before switching the FM/AM switch.
