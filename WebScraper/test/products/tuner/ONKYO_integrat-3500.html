  • ONKYO Integra T-3500

  • ¥ 28,000 (around 1975)

Commentary

Integra GT series FM/AM tuners.

The front end uses a low-noise FET and a 3-row varicon.

The IF stage uses a low-distortion linear phase type ceramic filter.

PLL-IC is used for the MPX circuit.
