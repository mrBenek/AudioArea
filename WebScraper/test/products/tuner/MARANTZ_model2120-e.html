  • Marantz Model2120

  • ¥ 69,800 (around 1977)

Commentary

An FM/AM tuner that uses a dual gate MOS FET in the front end and a linear phase IF circuit with improved delay characteristics.

The built-in 400 Hz reference signal transmitter of the original Record Level Tone mechanism makes it easy to set the recording level. Furthermore, if it is used in combination with the original parametric muting, a 400 Hz signal is generated at the time of tuning, so that channel selection and level setting can be done simultaneously.

Wooden cabinet is sold separately.
