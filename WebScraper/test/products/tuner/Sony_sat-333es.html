Commentary

The BS tuner was developed using digital technology accumulated over the years in order to beautifully catch the B-mode high-fidelity sound of satellite broadcasting.

It is equipped with an 8-by-oversampling 16-bit digital filter before D/A conversion. By making the data to be converted 8 times finer, the information density is increased and the sound quality is improved. In addition, the analog filter after D/A conversion can be used with a type with a gentle curve, which minimizes the deterioration of phase characteristics and group delay characteristics associated with a steep filter.
In addition, high performance is ensured by generating data with 293 order computing power.

The D/A converter section uses a dual staggered 4D/A converter. In this method, two D/A converters operating at 4 fs per channel are used, and the output is synthesized with the operation timing shifted by half a period. The two D/A converters are fed alternately with 8 fs data, so both D/A converters are always operating overlapping each other. The resulting output is the average of the two converters, so noise and distortion are averaged and signal quality is improved.

It is equipped with two digital audio outputs, EIAJ standard coaxial and optical, so that you can record digital signals as they are.
The output can be selected from coaxial + optical / coaxial / optical with the switch on the front panel.

In order to reproduce satellite broadcast images clearly, we have adopted a PLL-based FM detector and a PLL frequency synthesizer to obtain stable images.

Information such as the selected channel, voice mode, stereo / mono, main / sub, etc. can be displayed overlapped on the TV screen. In addition, Sony's unique antenna input level display function with peak hold can be used as a guide when setting the antenna direction.

Equipped with a 3-step program memory function, linked with a timer (sold separately), etc., you can call up the set channel and audio main / sub when the power is turned on.
You can store up to three memories. Each time the power is turned on, the next memory is called up. When this function is not used, the last channel memory function is installed to store the channel you were watching before turning off the power.

Two video and audio outputs are installed, and a gold-plated plug with low contact resistance is used.

It is equipped with an RF output terminal that converts to UHF 13 ch and outputs so that it can be connected to a TV without video / audio input terminal.

It is equipped with a noise filter to improve the graininess of image quality when input is weak.

Equipped with a GOOD RECEPTION display function to clearly indicate good radio wave conditions.

Digital audio output can be turned ON/OFF.

It is equipped with a concentrated display by 2-color fluorescent display tube.

Equipped with an indicator button that can turn off the display while leaving only the channel number.

Bit stream (data) output and detect (detection) output terminals are mounted.

Uses OFC power cord.

Equipped with a headphone port with volume.

It comes with a wireless remote control that can perform most operations including power ON/OFF and channel selection.
With this remote control, you can operate not only the main body but also the basic operation of TV with the mark made by Sony and TV tuner.

As with the built-in VHF/UHF tuner, the EDV 7000 EDBeta VCR has a BS control function that activates the SAT 333 es via the control S jack and also selects the channel.
