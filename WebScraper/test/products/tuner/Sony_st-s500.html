Audio Database

  • SONY ST-S500

  • ¥ 24,800 (released in 1992)

Commentary

FM/AM stereo tuner that can also receive AM stereo broadcast transmission.

It is equipped with a random 30-station memory and an FM auto-preset memory that receives only one station after another and stores that station.

Station name display function is installed.
