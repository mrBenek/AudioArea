  • Technics ST-S44

  • ¥ 49,800 (around 1981)

Commentary

Quartz synthesizer FM/AM tuner with 4-station preset FM/AM for one touch channel selection.

Equipped with the last one memory that can be enjoyed by the last receiving station at the next power ON.

It is equipped with an auto-scan function that allows you to preset desired stations.

Compensation for power failure by newly developed gold capacitor It is equipped with memory backup for about one week.

Equipped with digital / analog 2-way frequency display function.

Equipped with FM muting / mode switch.
