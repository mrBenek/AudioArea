Commentary

A compact size FM/AM tuner that uses a quartz synthesizer system with soft touch sensor buttons for accurate tuning operation.

Up to 8 FM/AM stations can be preset.
In addition, it is equipped with functions such as Last One Memory, which can receive signals when the power is turned off, and Memory Lock, which protects the memory from errors.

A 4-pole MOS FET and back-to-back varicap are used in the front end to obtain high sensitivity and excellent IM interference removal capability.

A surface acoustic wave (SAW) filter is used in the IF stage to achieve both high selectivity and low distortion.

Die-casting cabinet is used to cut off interfering signals.
