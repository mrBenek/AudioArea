  • ONKYO T-424

  • ¥ 29,000 (around 1997)

Commentary

FM/AM stereo tuner with quartz synthesizer system.

It uses a highly reliable digital quartz synthesizer system to enable high-quality reception.
In addition, the preset memory system is a random 30-station preset that can be stored in any order by both FM and AM stations, and music can be directly selected using 10 keys on the front panel.

It is equipped with an APR system (Automatic Precision Reception) that automatically switches between sensitivity (RF : DX/LOCAL) and stereo mode (AUTO/MONO) according to the radio wave condition of FM broadcasting. You can always use the optimum mode without worrying about the reception condition every time you select music.
In addition, it is equipped with a signal meter that can check the strength of radio waves.

The tuning method employs auto-search tuning that automatically searches for and stops sensitive stations in both FM and AM.

Compatible with AM stereo broadcast transmission.

If it is connected with the attached RI cord, it can be operated with the A-924 remote control.

Uses a power cord with polarity display.

8-digit character input is possible.

Antenna adapter is used.
